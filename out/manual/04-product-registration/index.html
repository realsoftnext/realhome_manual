4. 매물 등록하기

← 목차로 돌아가기

매물을 등록하는 상세한 방법을 단계별로 설명합니다.

4.1 매물 등록 시작하기

등록 페이지 접속

  1. 좌측 메뉴에서 📋 매물관리 클릭
  2. 우측 상단 + 매물 등록 버튼 클릭

또는 대시보드에서 빠른 실행 > + 매물 등록 클릭

4.2 1단계: 매물 종류 선택

가장 먼저 매물의 종류를 선택합니다. 이에 따라 입력 항목이 자동으로 달라집니다.

매물 종류 선택 *
┌─────────────────────────────────┐
│ [매물 종류를 선택하세요 ▼]      │
└─────────────────────────────────┘

선택 옵션:
┌─────────────────────────────────┐
│ 주거용                           │
│  ├─ 아파트                      │
│  ├─ 빌라/연립                   │
│  ├─ 단독/다가구                 │
│  └─ 원룸/오피스텔               │
│                                  │
│ 상업용                           │
│  ├─ 상가                        │
│  ├─ 사무실                      │
│  └─ 건물                        │
│                                  │
│ 기타                             │
│  └─ 토지                        │
└─────────────────────────────────┘

💡 중요: 매물 종류를 잘못 선택하면 나중에 변경할 수 없으므로 신중하게 선택하세요.

4.3 2단계: 기본 정보 입력

매물 제목 *

고객에게 보일 매물의 제목을 입력합니다.

매물 제목 *
┌─────────────────────────────────┐
│                                  │
└─────────────────────────────────┘

좋은 제목 예시:

  • ✅ "강남역 5분거리 신축 아파트 급매"
  • ✅ "역세권 풀옵션 원룸 즉시입주"
  • ✅ "1층 카페가능 상가 권리금 없음"

피해야 할 제목:

  • ❌ "아파트" (너무 간단)
  • ❌ "급매매매매매" (과도한 강조)

주소 입력 *

주소 검색

  1. 주소 검색 버튼 클릭
주소 *
┌─────────────────────────────────┐
│ [주소 검색] 🔍                   │
└─────────────────────────────────┘
  1. 주소 검색 팝업창이 나타남
┌─────────────────────────────────┐
│ 주소 검색                        │
│                                  │
│ [도로명/지번 주소 입력...]       │
│ [검색하기]                      │
│                                  │
│ 검색 결과:                      │
│ ○ 서울 강남구 테헤란로 123      │
│ ○ 서울 강남구 역삼동 789-1      │
│ ○ 서울 강남구 테헤란로 125      │
│                                  │
│ [선택]                          │
└─────────────────────────────────┘
  1. 원하는 주소 선택
  2. 선택한 주소가 자동으로 입력됨

상세 주소 입력

선택된 주소:
서울특별시 강남구 테헤란로 123

상세 주소
┌─────────────────────────────────┐
│ 101동 1205호                    │
└─────────────────────────────────┘

상세 주소 예시:

  • 아파트: "101동 1205호"
  • 빌라: "2층 201호"
  • 상가: "1층 102호"
  • 건물: "3층"

💡 자동 지도 표시: 주소를 선택하면 지도 좌표가 자동으로 설정됩니다.

면적 입력

제곱미터(㎡) 또는 평 중 하나만 입력하면 자동으로 변환됩니다.

면적
┌───────────────┬───────────────┐
│ [_____] ㎡    │ [_____] 평    │
│ (제곱미터)     │               │
└───────────────┴───────────────┘

변환 공식:

  • 1평 = 3.3058㎡
  • 1㎡ = 0.3025평

예시:

  • 84㎡ 입력 → 자동으로 25.41평 표시
  • 32평 입력 → 자동으로 105.79㎡ 표시

4.6 5단계: 상세 설명 작성

매물의 특징, 장점, 주변 환경 등을 자유롭게 작성합니다.

상세 설명
┌────────────────────────────────────┐
│                                     │
│                                     │
│                                     │
│                                     │
│                                     │
│                                     │
└────────────────────────────────────┘

[B] [I] [U] 📎 🖼️ (에디터 도구)

작성 팁:

  1. 위치 정보

    • 지하철역과의 거리
    • 버스 정류장
    • 주요 시설 (학교, 병원, 마트 등)

  2. 매물 특징

    • 옵션 (가전제품, 가구 등)
    • 최근 수리/리모델링 이력
    • 일조량, 조망

  3. 입주 조건

    • 즉시 입주 가능 여부
    • 반려동물 가능 여부
    • 주차 가능 대수

예시:

🏠 매물 특징
• 강남역 도보 5분 거리의 역세권 아파트
• 2023년 신축으로 깨끗한 상태
• 남향으로 하루 종일 햇볕이 잘 듭니다

🎯 옵션
• 풀옵션 (에어컨, 냉장고, 세탁기, 침대 등)
• 빌트인 주방가전
• 현관 중문 설치

🚇 교통
• 강남역 도보 5분
• 버스 정류장 바로 앞

🏢 주변 시설
• 롯데백화점 도보 3분
• 강남구청 도보 10분
• 초중고등학교 학군 우수

✅ 입주 조건
• 즉시 입주 가능
• 주차 1대 가능
• 반려동물 협의 가능

  • 사진을 클릭하면 대표로 설정

순서 변경

  • 사진을 드래그하여 순서 조정

사진 촬영 팁

💡 좋은 매물 사진:

  • ✅ 밝은 시간대에 촬영 (낮 10시~14시)
  • ✅ 가로 방향으로 촬영
  • ✅ 방 전체가 보이도록 촬영
  • ✅ 깨끗하게 정리 후 촬영

추천 사진 구성:

  1. 외관/현관 (1장)
  2. 거실 (2장)
  3. 방 (2-3장)
  4. 주방 (1장)
  5. 욕실 (1장)
  6. 전망/발코니 (1장)

4.9 8단계: 등록 완료

최종 확인

모든 정보를 입력했으면 하단의 버튼을 클릭합니다.

┌────────────────────────────────────┐
│ [임시저장]  [미리보기]  [등록하기] │
└────────────────────────────────────┘

버튼 설명:

  • 임시저장: 나중에 이어서 작성
  • 미리보기: 고객이 볼 화면 미리보기
  • 등록하기: 매물 등록 완료
