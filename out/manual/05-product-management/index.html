5.4 매물 수정하기

수정 방법

  1. 매물 목록 또는 상세 화면에서 수정 버튼 클릭
  2. 수정할 내용 변경
  3. 하단의 저장하기 버튼 클릭

수정 가능 항목

✅ 수정 가능:

  • 제목
  • 상세 주소
  • 면적
  • 가격
  • 상세 정보 (층수, 방향 등)
  • 상세 설명
  • 사진 (추가/삭제/순서 변경)
  • 계약 상태
  • 연락처
  • 테마/라벨

❌ 수정 불가:

  • 매물 종류 (삭제 후 재등록 필요)
  • 기본 주소 (삭제 후 재등록 필요)
  • 매물 번호 (자동 생성)

빠른 수정

[취소]  [삭제]
  1. 삭제 버튼 클릭

⚠️ 주의: 삭제된 매물은 복구할 수 없습니다!

대신 이렇게 하세요

매물을 완전히 삭제하기 전에 다음을 고려하세요:

계약 완료된 매물:

  • 상태를 계약완료로 변경
  • 나중에 통계 자료로 활용

일시적으로 숨기기:

  • 상태를 거래중지로 변경
  • 필요 시 다시 진행중으로 변경

일괄 삭제

여러 매물을 한 번에 삭제할 수 있습니다.

  1. 삭제할 매물들의 체크박스 선택
  2. 상단 일괄 작업 버튼 클릭
  3. 선택 항목 삭제 선택
  4. 확인 메시지에서 삭제 클릭

5.7 매물 복사하기

비슷한 매물을 등록할 때 기존 매물을 복사하여 시간을 절약할 수 있습니다.

복사 방법

  1. 목록 또는 상세 화면에서 복사 버튼 클릭
  2. 자동으로 복사본이 생성됨
  3. 다른 정보만 수정하여 저장

복사되는 항목:

  • ✅ 매물 종류
  • ✅ 기본 주소
  • ✅ 면적
  • ✅ 상세 정보 (층수, 방향 등)
  • ✅ 상세 설명
  • ❌ 사진 (복사 안 됨)
  • ❌ 가격 (새로 입력 필요)

복사 후 제목:

  • 원본: "강남역 5분거리 신축 아파트"
  • 복사본: "강남역 5분거리 신축 아파트 (복사본)"

5.8 엑셀로 내보내기/가져오기

엑셀 내보내기 (다운로드)

현재 보이는 매물 목록을 엑셀 파일로 다운로드합니다.

방법:

  1. 원하는 필터/검색 조건 설정
  2. 우측 상단 내보내기 버튼 클릭
  3. 엑셀 파일 자동 다운로드

내보내기 옵션:

┌────────────────────────────────────┐
│ 엑셀 내보내기                      │
├────────────────────────────────────┤
│ 내보낼 항목:                       │
│ ☑ 기본 정보 (제목, 주소, 면적)    │
│ ☑ 가격 정보                        │
│ ☑ 상세 정보                        │
│ ☑ 연락처                           │
│ ☐ 상세 설명                        │
├────────────────────────────────────┤
│ 파일명: 매물목록_20251112.xlsx    │
├────────────────────────────────────┤
│      [취소]        [다운로드]     │
└────────────────────────────────────┘

활용 예시:

  • 외부 공유용 자료 작성
  • 백업
  • 통계 분석

엑셀 가져오기 (업로드)

엑셀 파일로 여러 매물을 한 번에 등록합니다.

방법:

1단계: 샘플 양식 다운로드

[가져오기 ▼]
  ├─ 샘플 양식 다운로드
  └─ 엑셀 파일 업로드
  1. 샘플 양식 다운로드 클릭
  2. 매물등록_양식.xlsx 파일 다운로드

2단계: 양식 작성

다운로드한 엑셀 파일을 엽니다:

| 매물종류* | 제목* | 주소* | 상세주소 | 면적(㎡) | 거래종류* | 매매가 | 전세가 | 보증금 | 월세 | ... | |-----------|-------|-------|----------|----------|-----------|--------|--------|--------|------|-----| | 아파트 | 강남 아파트 | 서울 강남구... | 101동 | 84 | 매매 | 500000000 | | | | ... | | 원룸 | 역세권 원룸 | 서울 강남구... | 2층 | 20 | 월세 | | | 10000000 | 500000 | ... |

작성 규칙:

  • * 표시는 필수 항목
  • 매물종류는 정확히 입력 (예: "아파트", "원룸")
  • 가격은 숫자만 입력 (쉼표 없이)
  • 날짜는 YYYY-MM-DD 형식

3단계: 파일 업로드

  1. 엑셀 파일 업로드 클릭
  2. 작성한 엑셀 파일 선택
  3. 업로드 버튼 클릭

  1. 등록 버튼 클릭
