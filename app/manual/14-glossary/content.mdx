## 부동산 용어

### 거래 관련

| 용어 | 설명 |
|------|------|
| **매물** | 판매하거나 임대할 부동산 |
| **매도** | 부동산을 파는 것 |
| **매수** | 부동산을 사는 것 |
| **매도인** | 부동산을 파는 사람 (판매자) |
| **매수인** | 부동산을 사는 사람 (구매자) |
| **임대인** | 부동산을 빌려주는 사람 (집주인) |
| **임차인** | 부동산을 빌리는 사람 (세입자) |

### 거래 유형

| 용어 | 설명 |
|------|------|
| **매매** | 부동산을 사고 파는 거래 |
| **전세** | 보증금만 내고 일정 기간 거주하는 임대 방식 |
| **월세** | 보증금 + 매달 월세를 내는 임대 방식 |
| **반전세** | 전세와 월세의 중간 형태 (큰 보증금 + 적은 월세) |
| **분양** | 새로 지은 아파트 등을 최초로 파는 것 |
| **분양권** | 분양받을 수 있는 권리 (입주 전 매매 가능) |

### 가격 관련

| 용어 | 설명 |
|------|------|
| **매매가** | 매매 시 거래되는 가격 |
| **전세가** | 전세 계약의 보증금 |
| **보증금** | 월세 계약 시 미리 내는 금액 (계약 종료 시 반환) |
| **월세** | 매월 내는 임대료 |
| **권리금** | 상가 영업권을 사는 비용 (법적 보호 없음) |
| **프리미엄** | 분양권 거래 시 분양가 대비 추가 금액 |
| **관리비** | 건물 관리를 위해 매월 내는 비용 |

### 계약 관련

| 용어 | 설명 |
|------|------|
| **중개** | 부동산 거래를 중간에서 도와주는 것 |
| **중개수수료** | 중개업소에 지불하는 수수료 (거래 금액의 일정 비율) |
| **계약금** | 계약 시 먼저 지불하는 금액 (보통 총액의 10%) |
| **중도금** | 계약금과 잔금 사이에 지불하는 금액 |
| **잔금** | 최종적으로 지불하는 나머지 금액 |
| **특약사항** | 계약서에 추가로 합의한 조건 |

## 건물/시설 용어

### 주거 유형

| 용어 | 설명 |
|------|------|
| **아파트** | 5층 이상의 공동주택 |
| **빌라** | 4층 이하의 공동주택 |
| **연립주택** | 연립으로 지어진 주택 (빌라와 유사) |
| **다세대주택** | 여러 세대가 사는 주택 |
| **단독주택** | 한 가구만 사는 집 |
| **다가구주택** | 여러 가구가 사는 단독주택 형태 |
| **원룸** | 방 하나짜리 소형 주택 |
| **투룸** | 방 두 개짜리 주택 |
| **오피스텔** | 주거와 업무를 겸할 수 있는 건물 |

### 상업용

| 용어 | 설명 |
|------|------|
| **상가** | 장사하는 가게 |
| **근린상가** | 주거지역 근처의 상가 |
| **사무실** | 업무용 공간 |
| **빌딩** | 상업용 건물 |
| **지식산업센터** | 공장과 사무실 기능을 합친 건물 |

### 면적 단위

| 용어 | 설명 |
|------|------|
| **평** | 한국 전통 면적 단위 (1평 = 3.3058㎡) |
| **㎡ (제곱미터)** | 국제 표준 면적 단위 |
| **평방피트** | 영미권 면적 단위 (1평방피트 = 0.0929㎡) |
| **전용면적** | 실제 사용할 수 있는 면적 (내부만) |
| **공급면적** | 전용면적 + 계단, 복도 등 공용 면적 |

### 층수 관련

| 용어 | 설명 |
|------|------|
| **저층** | 낮은 층 (보통 1~3층) |
| **중층** | 중간 층 |
| **고층** | 높은 층 |
| **탑층** | 최상층 |
| **반지하** | 절반이 땅 아래인 층 |
| **복층** | 한 세대가 2개 층을 사용 |
| **로프트** | 천장이 높은 구조 (다락방 형태) |

## 건축/구조 용어

### 구조

| 용어 | 설명 |
|------|------|
| **구조** | 건물의 뼈대 (철근콘크리트, 철골조 등) |
| **철근콘크리트** | 철근과 콘크리트로 만든 구조 (튼튼함) |
| **철골조** | 철골로 만든 구조 (지진에 강함) |
| **조적식** | 벽돌을 쌓아 올린 구조 |

### 시설

| 용어 | 설명 |
|------|------|
| **엘리베이터** | 승강기 |
| **주차장** | 차를 세우는 공간 |
| **경비실** | 경비원이 근무하는 곳 |
| **관리사무소** | 건물 관리를 담당하는 사무소 |
| **근린생활시설** | 편의점, 슈퍼 등 생활편의시설 |

## 방향 용어

| 용어 | 설명 |
|------|------|
| **남향** | 남쪽을 향함 (햇빛 잘 듦, 선호도 높음) |
| **동향** | 동쪽을 향함 (아침 햇빛) |
| **서향** | 서쪽을 향함 (오후 햇빛, 여름 더움) |
| **북향** | 북쪽을 향함 (햇빛 적음) |
| **남동향** | 남동쪽을 향함 |
| **남서향** | 남서쪽을 향함 |
| **양지** | 햇빛이 잘 드는 곳 |
| **음지** | 햇빛이 잘 안 드는 곳 |

## 법률 용어

### 토지 관련

| 용어 | 설명 |
|------|------|
| **지목** | 토지의 용도 (대지, 전, 답, 임야 등) |
| **용도지역** | 토지의 이용 목적에 따른 구분 (주거, 상업, 공업 등) |
| **건폐율** | 대지 면적 대비 건물이 차지하는 비율 |
| **용적률** | 대지 면적 대비 건물 연면적 비율 |
| **도로 접면** | 토지가 도로와 닿아있는 것 |

### 권리 관련

| 용어 | 설명 |
|------|------|
| **소유권** | 부동산을 소유할 수 있는 권리 |
| **등기부등본** | 부동산의 권리 관계를 기록한 서류 |
| **근저당** | 대출을 위해 설정한 담보 |
| **전입신고** | 이사한 후 주소를 옮기는 신고 |
| **확정일자** | 전세 계약서에 도장을 찍는 것 (보호 받음) |

## 시스템 용어

### 매물 관리

| 용어 | 설명 |
|------|------|
| **매물 종류** | 매물을 분류하는 카테고리 (아파트, 빌라, 상가 등) |
| **동적 필드** | 매물 종류에 따라 다른 입력 항목 |
| **프리셋** | 미리 만들어둔 설정 세트 |
| **테마** | 매물 그룹 (급매물, 추천매물 등) |
| **라벨** | 매물 태그 (역세권, 신축 등) |

### 사용자 관리

| 용어 | 설명 |
|------|------|
| **계정** | 시스템 사용을 위한 ID |
| **권한** | 사용할 수 있는 기능의 범위 |
| **슈퍼관리자** | 모든 권한을 가진 최고 관리자 |
| **관리자** | 전체 관리 권한을 가진 사용자 |
| **직원** | 제한된 권한을 가진 일반 사용자 |

### 시스템 기능

| 용어 | 설명 |
|------|------|
| **대시보드** | 주요 정보를 한눈에 보는 첫 화면 |
| **필터** | 조건에 맞는 항목만 골라내는 기능 |
| **정렬** | 순서를 바꾸는 기능 (최신순, 가격순 등) |
| **페이징** | 내용을 여러 페이지로 나누는 것 |
| **엑셀 내보내기** | 데이터를 엑셀 파일로 저장 |
| **엑셀 가져오기** | 엑셀 파일의 데이터를 시스템에 입력 |

## 숫자 읽기

| 단위 | 읽기 | 숫자 |
|------|------|------|
| **만** | 만 | 10,000 |
| **십만** | 십만 | 100,000 |
| **백만** | 백만 | 1,000,000 |
| **천만** | 천만 | 10,000,000 |
| **억** | 억 | 100,000,000 |
| **십억** | 십억 | 1,000,000,000 |

**예시**:
- 5천만원 = 50,000,000원
- 3억원 = 300,000,000원
- 5억 5천만원 = 550,000,000원
