## 6.1 연락처 관리 개요

고객 연락처를 통합 관리하여 매물 등록, 문의 처리, 계약 작성 시 쉽게 재사용할 수 있습니다.

**연락처 활용 예시**:
- 매물 등록 시 매도인/관리인 정보
- 문의 접수 시 고객 정보
- 계약 작성 시 당사자 정보

---

│ 📧 hong@email.com         🏷️ 매도인              │

│                               [수정] [삭제] [보기]│

│ 김철수                    010-9876-5432            │
│ 📧 -                      🏷️ 매수인              │

## 6.3 연락처 타입

### 타입 종류

| 타입 | 설명 | 사용 예시 |
|------|------|-----------|
| 🏷️ **매도인** | 매물을 파는 사람 | 집주인, 건물주 |
| 🏷️ **매수인** | 매물을 사는 사람 | 구매 희망자 |
| 🏷️ **관리인** | 건물 관리인 | 관리사무소, 경비실 |
| 🏷️ **소유주** | 건물 소유주 | 임대인 |
| 🏷️ **세입자** | 현재 거주자 | 전세/월세 입주자 |

### 타입별 필터

상단 드롭다운에서 타입을 선택하여 필터링할 수 있습니다.

```
[타입 필터 ▼]
  ├─ 전체 (83)
  ├─ 매도인 (34)
  ├─ 매수인 (29)
  ├─ 관리인 (12)
  ├─ 소유주 (5)
  └─ 세입자 (3)
```

## 6.4 새 연락처 추가

### 추가 방법

1. 우측 상단 **+ 새 연락처** 버튼 클릭
2. 정보 입력
3. **저장** 버튼 클릭

### 입력 폼

```
새 연락처 추가
┌────────────────────────────────────┐
│ 이름 *                             │
│ [________________]                 │
├────────────────────────────────────┤
│ 전화번호 *                         │
│ [________________]                 │
│ 예: 010-1234-5678                 │
├────────────────────────────────────┤
│ 이메일                             │
│ [________________]                 │
│ 예: hong@email.com                │
├────────────────────────────────────┤
│ 타입 *                             │
│ ○ 매도인  ○ 매수인                │
│ ○ 관리인  ○ 소유주  ○ 세입자     │
│                                     │
│ 매도인: 매물을 파는 사람           │
│ 매수인: 매물을 사는 사람           │
│ 관리인: 건물 관리인                │
│ 소유주: 건물 소유주                │
│ 세입자: 현재 거주자                │
├────────────────────────────────────┤
│ 메모 (선택사항)                    │
│ [_______________________________]  │
│ [_______________________________]  │
│ [_______________________________]  │
│                                     │
│ 예: 강남 아파트 소유주             │
│     연락 가능 시간: 평일 오후      │
├────────────────────────────────────┤
│      [취소]        [저장]         │
└────────────────────────────────────┘
```

### 입력 항목 설명

| 항목 | 필수 | 설명 |
|------|------|------|
| **이름** | ✅ | 고객 이름 (2자 이상) |
| **전화번호** | ✅ | 연락 가능한 전화번호 |
| **이메일** | ❌ | 이메일 주소 (선택사항) |
| **타입** | ✅ | 고객 유형 선택 |
| **메모** | ❌ | 추가 정보 기록 |

### 전화번호 형식

다음 형식으로 입력 가능:
- `010-1234-5678` (하이픈 포함)
- `01012345678` (하이픈 없이)
- `02-1234-5678` (지역번호)

> 💡 **자동 변환**: 하이픈 없이 입력해도 자동으로 형식이 맞춰집니다.

## 6.5 연락처 검색

### 통합 검색

검색창에서 다음 항목으로 검색 가능:

```
[검색] [홍길동] 🔍
```

**검색 가능 항목**:
- ✅ 이름
- ✅ 전화번호 (숫자만)
- ✅ 이메일
- ✅ 메모

### 검색 팁

**이름으로 검색**:
- "홍" → 홍길동, 홍영희 등
- 최소 2글자 이상 입력

**전화번호로 검색**:
- "1234" → 010-1234-5678
- 숫자만 입력 (하이픈 불필요)

**빠른 검색**:
- 실시간으로 검색 결과 표시
- 2글자 이상 입력 시 자동 검색

### 수정 방법

1. 연락처 목록에서 **수정** 버튼 클릭
2. 정보 변경
3. **저장** 버튼 클릭

```
연락처 수정 - 홍길동
┌────────────────────────────────────┐
│ 이름 *                             │
│ [홍길동]                           │
├────────────────────────────────────┤
│ 전화번호 *                         │
│ [010-1234-5678]                   │
├────────────────────────────────────┤
│ 이메일                             │
│ [hong@email.com]                  │
├────────────────────────────────────┤
│ 타입 *                             │
│ ● 매도인  ○ 매수인                │
├────────────────────────────────────┤
│ 메모                               │
│ [강남 아파트 소유주]               │
│ [연락 가능: 평일 오후]             │
├────────────────────────────────────┤
│ 생성일: 2025-10-15                │
│ 수정일: 2025-11-10                │
├────────────────────────────────────┤
│      [취소]        [저장]         │
└────────────────────────────────────┘
```

## 6.7 연락처 삭제

### 삭제 방법

1. 연락처 목록에서 **삭제** 버튼 클릭
2. 확인 메시지 확인

```
⚠️ 연락처를 삭제하시겠습니까?

이름: 홍길동
전화번호: 010-1234-5678

삭제된 연락처는 복구할 수 없습니다.

[취소]  [삭제]
```

3. **삭제** 버튼 클릭

### 소프트 삭제

실제로는 완전히 삭제되지 않고 숨겨집니다 (데이터 보호).

**관리자만** 완전 삭제 가능:
- 설정 > 삭제된 연락처
- 완전 삭제 버튼 클릭

## 6.8 연락처 상세 보기

### 상세 정보 확인

연락처 이름을 클릭하거나 **보기** 버튼을 클릭합니다.

```
연락처 상세 - 홍길동
┌────────────────────────────────────────────────────┐
│ [< 목록]                        [수정] [삭제]     │
├────────────────────────────────────────────────────┤
│ 👤 기본 정보                                       │
│ • 이름: 홍길동                                     │
│ • 전화번호: 010-1234-5678                          │
│ • 이메일: hong@email.com                           │
│ • 타입: 매도인                                     │
│ • 메모: 강남 아파트 소유주                         │
├────────────────────────────────────────────────────┤
│ 📋 연결된 매물 (2건)                               │
│ • #123 강남역 5분거리 아파트 (매도인)              │
│ • #115 역세권 빌라 (매도인)                        │
├────────────────────────────────────────────────────┤
│ 📞 문의 이력 (1건)                                 │
│ • 2025-11-05 - 아파트 매도 문의                   │
├────────────────────────────────────────────────────┤
│ 📄 계약 이력 (1건)                                 │
│ • C2025-001 - 강남 아파트 매매 (매도인)           │
│   상태: 서명완료                                   │
├────────────────────────────────────────────────────┤
│ 📅 생성 정보                                       │
│ • 등록일: 2025-10-15 14:30                        │
│ • 수정일: 2025-11-10 09:15                        │
│ • 등록자: 박대리                                   │
└────────────────────────────────────────────────────┘
```

### 활용 정보

연락처 상세에서 다음을 확인할 수 있습니다:

- **연결된 매물**: 이 연락처가 등록된 모든 매물
- **문의 이력**: 과거 문의 내용
- **계약 이력**: 체결한 계약들
- **통화 기록**: 상담 내용 (메모 작성 시)

💡 **고객 관리**: 이 정보로 고객과의 거래 이력을 한눈에 파악할 수 있습니다.

## 6.9 매물 등록 시 연락처 사용

매물을 등록할 때 연락처를 쉽게 추가할 수 있습니다.

### ContactSelector 사용법

매물 등록 화면의 연락처 입력란:

```
매도인 연락처
┌────────────────────────────────────┐
│ 이름 또는 전화번호 검색...         │
│ [__________________] 🔍             │
└────────────────────────────────────┘
```

#### 1단계: 검색

이름 또는 전화번호를 입력하면 실시간으로 검색됩니다.

```
입력: "홍길"

검색 결과:
┌────────────────────────────────────┐
│ ✓ 홍길동                           │
│   매도인 | 010-1234-5678           │
│   hong@email.com                   │
│   강남 아파트 소유주                │
│   [선택하기]                       │
├────────────────────────────────────┤
│ ✓ 홍길순                           │
│   매수인 | 010-2222-3333           │
│   [선택하기]                       │
└────────────────────────────────────┘
```

#### 2단계: 선택 또는 추가

**기존 연락처 선택**:
- **선택하기** 버튼 클릭

**새 연락처 추가**:
- 검색 결과 없음
- **+ 새 연락처** 버튼 표시
- 클릭하여 즉시 추가

```
검색 결과 없음

입력한 내용으로 새 연락처를 추가하시겠습니까?

이름: 홍길동
전화번호: (입력 필요)

[+ 새 연락처 추가]
```

#### 3단계: 선택 확인

선택된 연락처가 표시됩니다.

```
선택된 연락처:
┌────────────────────────────────────┐
│ 👤 홍길동 (매도인)                 │
│ 📞 010-1234-5678                   │
│ ✉️ hong@email.com                  │
│ 📝 강남 아파트 소유주               │
│                                     │
│ [변경] [제거]                      │
└────────────────────────────────────┘
```

### 자동 타입 필터

매물 등록 시 자동으로 적절한 타입만 표시:

- **매도인 선택 시**: 매도인, 소유주 타입만
- **관리인 선택 시**: 관리인 타입만

## 6.10 중복 방지

### 중복 확인

새 연락처 추가 시 전화번호로 중복을 자동 확인합니다.

```
⚠️ 이미 등록된 연락처입니다

전화번호: 010-1234-5678
기존 연락처: 홍길동 (매도인)

[기존 연락처 사용]  [그래도 추가]
```

**권장**:
- **기존 연락처 사용**: 기존 정보 활용 (권장)
- **그래도 추가**: 동명이인인 경우

### 중복 연락처 정리

중복된 연락처를 찾아서 병합할 수 있습니다.

1. 연락처 관리 > **중복 찾기** 버튼
2. 중복 가능성 있는 연락처 목록 표시
3. 병합할 연락처 선택
4. **병합** 버튼 클릭

```
중복 가능성 있는 연락처

┌────────────────────────────────────┐
│ ☑ 홍길동 (010-1234-5678)          │
│ ☑ 홍 길동 (010-1234-5678)         │
│   → 같은 전화번호                  │
├────────────────────────────────────┤
│ ☑ 김철수 (010-9999-8888)          │
│ ☑ 김철수 (010-9999-8888)          │
│   → 같은 이름, 전화번호            │
│                                     │
│      [취소]        [병합]         │
└────────────────────────────────────┘
```

## 6.10 회원 관리

### 회원 관리 개요

웹사이트에 가입한 회원들을 관리하고, 마케팅 수신동의 현황을 확인하여 효과적으로 소통할 수 있습니다.

**주요 기능**:
- 👥 회원 검색 및 조회
- 📊 마케팅 수신동의 현황
- 📧 이메일 목록 다운로드
- 📱 연락처 목록 다운로드

### 회원 관리 화면 접속

좌측 메뉴에서 **👥 고객관리** > **회원 관리**를 클릭합니다.

```
회원 관리 (/admin/members)
┌────────────────────────────────────────────────────┐
│ 📊 통계                                            │
│ 전체 회원: 1,234명 | 마케팅 동의: 856명 (69%)    │
├────────────────────────────────────────────────────┤
│ [검색] [__________________] 🔍                     │
│ 검색 대상: [전체 ▼]  (이름, 이메일, 전화번호)     │
├────────────────────────────────────────────────────┤
│ 이름         이메일              마케팅동의        │
│ 홍길동       hong@email.com     ✅ 동의            │
│ 김철수       kim@email.com      ❌ 미동의          │
│ 이영희       lee@email.com      ✅ 동의            │
└────────────────────────────────────────────────────┘
```

### 회원 검색하기

#### 검색 방법

1. **검색어 입력**: 이름, 이메일, 전화번호로 검색
2. **검색 대상 선택**: 
   - 전체
   - 이름
   - 이메일
   - 전화번호
3. **🔍 검색 버튼 클릭**

#### 검색 예시

- **이름 검색**: "홍길동" 입력
- **이메일 검색**: "hong@email.com" 입력
- **전화번호 검색**: "010-1234-5678" 입력

### 마케팅 수신동의 현황 확인

#### 통계 보기

회원 관리 화면 상단에서 실시간 통계를 확인할 수 있습니다:

```
📊 통계
┌────────────────────────────────────┐
│ 전체 회원:      1,234명           │
│ 마케팅 동의:      856명 (69%)     │
│ 마케팅 미동의:    378명 (31%)     │
└────────────────────────────────────┘
```

#### 개별 회원 동의 상태

회원 목록에서 각 회원의 마케팅 수신동의 상태를 한눈에 확인:
- ✅ **동의**: 마케팅 메일/문자 수신 동의
- ❌ **미동의**: 마케팅 수신 거부

### 회원 목록 다운로드

#### 전체 회원 이메일 목록 다운로드

1. 회원 목록 상단 **[전체 이메일 다운로드]** 버튼 클릭
2. `members_email_전체_YYYYMMDD.csv` 파일 저장
3. 엑셀이나 메일 발송 프로그램에서 활용

**다운로드 파일 형식**:
```
이름,이메일,가입일,마케팅동의
홍길동,hong@email.com,2024-01-15,예
김철수,kim@email.com,2024-01-16,아니오
```

#### 마케팅 동의자만 다운로드

1. **[마케팅 동의자 이메일 다운로드]** 버튼 클릭
2. 마케팅 수신에 동의한 회원만 포함된 파일 저장
3. 프로모션, 이벤트 안내 발송에 활용

> 💡 **개인정보 보호**: 다운로드한 목록은 마케팅 목적으로만 사용하고, 제3자에게 제공하지 마세요.

#### 연락처 목록 다운로드

1. **[연락처 다운로드]** 버튼 클릭
2. 이메일과 전화번호가 포함된 파일 저장
3. SMS 발송 시 활용

**다운로드 파일 형식**:
```
이름,이메일,전화번호,마케팅동의
홍길동,hong@email.com,010-1234-5678,예
```

### 회원 정보 활용

#### 마케팅 활용

마케팅 동의자에게만 다음과 같은 정보를 발송하세요:

- 📧 **신규 매물 안내**: 조건에 맞는 매물 등록 시 알림
- 🎉 **프로모션 안내**: 이벤트, 할인 정보
- 📰 **뉴스레터**: 부동산 시장 동향, 투자 팁

#### 페이지네이션

많은 회원이 있을 경우 페이지 단위로 표시됩니다:

```
[◀ 이전]  1  2  3  4  5  [다음 ▶]
```

- 페이지당 20명씩 표시
- 화살표 버튼으로 페이지 이동

### API 활용 (개발자용)

회원 관리 기능은 다음 API를 통해 제공됩니다:

#### 회원 검색 및 조회
```
GET /api/admin/members
```
- **파라미터**: 
  - `search`: 검색어
  - `page`: 페이지 번호
  - `limit`: 페이지당 항목 수
- **응답**: 회원 목록 (페이지네이션)

#### 회원 통계
```
GET /api/admin/members/stats
```
- **응답**: 
  - 전체 회원 수
  - 마케팅 동의 회원 수
  - 비율

### 💡 활용 팁

1. **정기 검토**: 월 1회 마케팅 동의 현황을 확인하세요
2. **개인화**: 회원의 관심 매물 유형에 맞는 정보를 발송하세요
3. **타이밍**: 신규 매물 등록 후 24시간 이내 발송이 효과적입니다
4. **수신 거부**: 마케팅 메일에 수신 거부 링크를 반드시 포함하세요
5. **통계 활용**: 오픈율, 클릭률을 추적하여 개선하세요

### ⚠️ 주의사항

1. **개인정보 보호**: 다운로드한 정보는 안전하게 보관하세요
2. **동의 범위**: 마케팅 동의자에게만 프로모션을 발송하세요
3. **법률 준수**: 정보통신망법, 개인정보보호법을 준수하세요
4. **수신 거부**: 고객이 수신 거부를 요청하면 즉시 처리하세요
