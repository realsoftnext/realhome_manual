## 10.1 직원 관리 개요

회사 직원을 초대하고, 역할과 권한을 설정하는 방법을 설명합니다.

**관리 가능 항목**:
- 👥 직원 초대
- 🎭 역할 할당
- 🔐 권한 설정
- 📊 활동 모니터링
- 🚫 계정 비활성화

> 💡 **관리자 권한 필요**: 이 기능은 관리자만 사용할 수 있습니다.

## 10.7 직원 비활성화

### 비활성화 vs 삭제

| 작업 | 결과 | 데이터 |
|------|------|--------|
| **비활성화** | 로그인 불가 | 유지 |
| **삭제** | 계정 제거 | 삭제 |

> 💡 **권장**: 퇴사자는 비활성화하고, 삭제는 하지 마세요. 데이터 추적을 위해 필요합니다.

### 비활성화 방법

1. 직원 목록에서 **비활성화** 버튼 클릭

```
직원 비활성화 확인
┌────────────────────────────────────┐
│ "박대리"를 비활성화하시겠습니까?   │
│                                     │
│ • 로그인 불가능                    │
│ • 기존 데이터는 유지               │
│ • 나중에 다시 활성화 가능          │
│                                     │
│      [취소]        [비활성화]      │
└────────────────────────────────────┘
```

2. **비활성화** 버튼 클릭

### 재활성화

비활성화된 직원을 다시 활성화할 수 있습니다.

1. 비활성 직원의 **수정** 버튼 클릭
2. 상태를 **활성**으로 변경
3. **변경사항 저장** 버튼 클릭

## 10.9 실전 예제

### 예제 1: 신입 사원 초대

**상황**: 신입 사원이 입사했습니다. 매물 등록은 가능하지만 삭제는 불가능하게 설정하고 싶습니다.

**해결**:
1. **+ 직원 초대** 클릭
2. 이메일: `new@company.com`
3. 역할: **직원** 선택 (본인이 등록한 매물만 수정 가능, 삭제 불가)
4. **초대 보내기**

### 예제 2: 마케팅 담당자 역할

**상황**: 마케팅 담당자는 블로그와 FAQ만 관리하면 됩니다. 매물이나 계약은 볼 필요가 없습니다.

**해결**:
1. **역할 관리** > **+ 새 역할 만들기**
2. 역할 이름: `마케팅 담당자`
3. 권한 설정:
   - 콘텐츠 관리 ✅
   - 나머지 모두 ❌
4. **역할 생성**
5. 직원 초대 시 이 역할 선택

### 예제 3: 퇴사자 처리

**상황**: 이과장이 퇴사했습니다. 하지만 이과장이 처리한 계약 정보는 남겨야 합니다.

**해결**:
1. 이과장의 **비활성화** 버튼 클릭
2. **비활성화** 확인
3. 결과:
   - 로그인 불가 ❌
   - 기존 매물/계약 정보 유지 ✅
   - "담당자: 이과장" 표시 유지 ✅

### 💡 유용한 팁

1. **역할 미리 계획**: 회사 조직에 맞는 역할을 미리 정의하세요.
2. **최소 권한 원칙**: 필요한 최소한의 권한만 부여하세요.
3. **정기 점검**: 분기마다 직원 목록과 권한을 점검하세요.
4. **활동 모니터링**: 이상 활동이 있는지 주기적으로 확인하세요.

### ⚠️ 주의사항

1. **관리자 계정**: 관리자 계정을 여러 명이 공유하지 마세요.
2. **삭제 주의**: 직원 삭제 시 해당 직원의 활동 기록도 사라집니다.
3. **권한 변경**: 권한 변경 시 즉시 적용되므로 주의하세요.
4. **초대 링크**: 초대 링크는 7일간 유효하니 기간 내에 가입하도록 안내하세요.
