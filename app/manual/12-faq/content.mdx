## 로그인 / 계정

### Q1. 비밀번호를 잊어버렸어요

**A**: 로그인 화면에서 "비밀번호를 잊으셨나요?" 링크를 클릭하고 가입한 이메일 주소를 입력하세요. 비밀번호 재설정 링크가 메일로 발송됩니다.

자세한 내용은 [2. 로그인 및 계정 관리](02-login-account.md#22-비밀번호-찾기) 참조

### Q2. 로그인이 안돼요

**가능한 원인**:
- 비밀번호가 틀렸습니다
- 계정이 비활성화되었습니다
- Caps Lock이 켜져있습니다
- 인터넷 연결 문제

**해결 방법**:
1. 대소문자를 정확히 확인
2. 비밀번호 찾기 이용
3. 관리자에게 문의

### Q3. 여러 기기에서 동시에 사용할 수 있나요?

**A**: 네, 가능합니다. 동일한 계정으로 PC, 스마트폰, 태블릿에서 동시에 로그인할 수 있습니다.

## 매물 관리

### Q4. 매물 사진이 업로드되지 않아요

**원인 확인**:
- 파일 형식: JPG, PNG만 가능
- 파일 크기: 10MB 이하
- 인터넷 연결 상태

**해결 방법**:
1. 사진 파일 크기 줄이기
2. 다른 형식으로 저장 (JPG 권장)
3. 브라우저 새로고침

### Q5. 매물을 삭제했는데 복구할 수 있나요?

**A**: **복구할 수 없습니다**. 삭제된 매물은 완전히 제거됩니다.

**대신 이렇게 하세요**:
- 매물을 숨기고 싶다면 **비활성화** 기능 사용
- 계약이 완료되면 **계약완료** 상태로 변경

### Q6. 매물 종류를 잘못 선택했어요

**A**: 매물 종류는 등록 후 변경할 수 없습니다.

**해결 방법**:
1. 해당 매물 삭제
2. 올바른 종류로 다시 등록
3. 사진과 내용을 복사하여 재입력

### Q7. 매물을 대량으로 등록하고 싶어요

**A**: 엑셀 가져오기 기능을 사용하세요.

**단계**:
1. 매물관리 > **가져오기** 클릭
2. 샘플 양식 다운로드
3. 양식에 맞춰 정보 입력
4. 파일 업로드

자세한 내용은 [5. 매물 관리하기](05-product-management.md#56-엑셀로-내보내기가져오기) 참조

### Q8. 여러 명이 동시에 같은 매물을 수정하면 어떻게 되나요?

**A**: 나중에 저장한 사람의 내용으로 덮어씌워집니다.

**주의**:
- 동료와 소통하여 동시 수정 피하기
- 중요한 수정 전에는 먼저 알리기

## 고객 / 문의

### Q9. 연락처를 추가했는데 검색이 안 돼요

**확인 사항**:
- 이름 또는 전화번호를 정확히 입력했는지
- 최소 2글자 이상 입력했는지
- 검색 필터 설정을 확인

**해결 방법**:
- 페이지 새로고침
- 전체 보기로 확인

### Q10. 문의 메일 알림이 안 와요

**확인 사항**:
1. 알림 설정이 켜져있는지 확인
   - 내 정보 > 알림 설정
2. 스팸 메일함 확인
3. 이메일 주소가 정확한지 확인

**설정 방법**:
내 정보 > 알림 설정 > "신규 문의 알림" 체크

## 계약 관리

### Q11. 계약서는 어떻게 인쇄하나요?

**A**: 
1. 계약 상세 화면 접속
2. 우측 상단 **인쇄** 버튼 클릭
3. 인쇄용 양식이 나타남
4. 프린터로 출력

### Q12. 계약 번호는 어떻게 생성되나요?

**A**: 계약 번호는 자동으로 생성됩니다.

**형식**: C2025-001, C2025-002...
- C: Contract (계약)
- 2025: 연도
- 001: 일련번호

### Q13. 결제 일정을 수정할 수 있나요?

**A**: 네, 가능합니다.

**방법**:
1. 계약 상세 > **수정** 버튼
2. 결제 일정 탭
3. 날짜 또는 금액 수정
4. **저장하기**

## 시스템 / 설정

### Q14. 데이터 백업은 어떻게 하나요?

**A**: 
1. **자동 백업**: 시스템에서 매일 자동으로 백업
2. **수동 백업**: 엑셀 내보내기로 개별 데이터 저장

**수동 백업 방법**:
- 매물관리 > **내보내기** > 엑셀 파일 저장

### Q15. 시스템이 느려요

**원인**:
- 인터넷 속도
- 브라우저 캐시
- 동시 접속자 과다

**해결 방법**:
1. 브라우저 캐시 삭제
   - Chrome: Ctrl + Shift + Delete
2. 브라우저 업데이트
3. 불필요한 탭 닫기
4. 인터넷 속도 확인

### Q16. 권한이 없다는 메시지가 나와요

**A**: 해당 기능을 사용할 권한이 없습니다.

**해결 방법**:
- 관리자에게 권한 요청
- 다른 계정으로 로그인 확인

**권한 구분**:
- 직원: 매물 관리, 문의 처리
- 관리자: 전체 관리
- 슈퍼관리자: 모든 권한

## 모바일 사용

### Q17. 모바일에서 사진 업로드가 안 돼요

**해결 방법**:
1. 카메라/저장소 권한 허용
2. 브라우저 설정 > 권한 확인
3. 다른 브라우저 사용 (Chrome 권장)

### Q18. 모바일에서 화면이 잘려 보여요

**해결 방법**:
1. 브라우저 확대/축소 리셋
   - 주소창 옆 확대/축소 100%로 설정
2. 가로 모드로 회전
3. 최신 브라우저 사용

## 오류 / 문제

### Q19. "세션이 만료되었습니다" 메시지가 나와요

**원인**: 30분간 활동이 없어서 자동 로그아웃

**해결 방법**:
- 다시 로그인
- 작업 중에는 가끔 페이지 새로고침

💡 **팁**: 임시저장 기능을 자주 사용하세요.

### Q20. 페이지에 오류가 발생했어요

**기본 해결 방법**:
1. **새로고침**: F5 또는 Ctrl + R
2. **캐시 삭제**: Ctrl + Shift + Delete
3. **다른 브라우저**: Chrome, Edge 시도
4. **재로그인**: 로그아웃 후 다시 로그인

**그래도 안 되면**:
- 오류 메시지 캡처
- 관리자에게 문의

## 고객 응대

### Q21. 고객에게 매물 정보를 어떻게 공유하나요?

**A**: 
1. 매물 상세 페이지 접속
2. **공유하기** 버튼 클릭
3. 링크 복사 또는 카카오톡 공유

**공유 옵션**:
- 링크 복사 (문자, 메일로 전송)
- 카카오톡 공유
- QR 코드 생성

### Q22. 고객이 직접 매물을 올릴 수 있나요?

**A**: 아니요. 매물 등록은 관리자/직원만 가능합니다.

**대신**:
- 홈페이지 "매물 의뢰" 기능 이용
- 문의하기로 요청

## 기타

### Q23. 시스템 사용 교육은 어디서 받나요?

**A**: 
- 이 매뉴얼 정독
- 관리자에게 1:1 교육 요청
- 기술 지원팀 문의 (support@realsoftnext.com)

### Q24. 새로운 기능 요청은 어떻게 하나요?

**A**:
- 이메일: support@realsoftnext.com
- 제목: [기능 요청] 내용
- 상세한 설명과 필요성 기재

### Q25. 시스템 업데이트는 언제 하나요?

**A**: 
- 정기 업데이트: 매월 첫째 주 일요일 새벽 2시
- 긴급 업데이트: 사전 공지

**업데이트 중**:
- 5-30분간 접속 불가
- 사전 알림 메일 발송
