## 5.1 매물 목록 보기

### 목록 화면 접속

좌측 메뉴에서 **📋 매물관리**를 클릭하면 등록된 모든 매물이 표시됩니다.

```
매물 관리 (총 150건)
┌────────────────────────────────────────────────────┐
│ [검색] [____________] 🔍                           │
│ [매물종류 ▼] [거래종류 ▼] [상태 ▼] [+ 매물등록] │
├────────────────────────────────────────────────────┤
│ #123 ⭐                                            │
│ 강남역 5분거리 신축 아파트                         │
│ 📍 서울 강남구 테헤란로 123                        │
│ 🏠 아파트 | 💰 매매 5억원 | 📏 84㎡(25평)         │
│ 👁️ 조회 156회 | 📅 2025-11-10 등록                │
│                               [수정] [삭제] [복사]│
├────────────────────────────────────────────────────┤
│ #122                                               │
│ 역세권 원룸 급매                                   │
│ 📍 서울 강남구 역삼동 789-1                        │
│ 🏠 원룸 | 💰 월세 500/50 | 📏 20㎡(6평)           │
│ 👁️ 조회 89회 | 📅 2025-11-09 등록                 │
│                               [수정] [삭제] [복사]│
└────────────────────────────────────────────────────┘

[◀ 이전] 1 2 3 4 5 [다음 ▶]
```

### 목록 정보 설명

각 매물 카드에는 다음 정보가 표시됩니다:

- **#123**: 매물 번호
- **⭐**: 추천매물 표시 (테마/라벨)
- **제목**: 매물 제목
- **📍 주소**: 매물 위치
- **🏠 종류**: 매물 종류
- **💰 가격**: 거래 유형 및 가격
- **📏 면적**: 제곱미터 및 평수
- **👁️ 조회수**: 홈페이지에서 조회된 횟수
- **📅 등록일**: 매물 등록 날짜

## 5.2 매물 검색 및 필터링

### 키워드 검색

검색창에 원하는 키워드를 입력합니다.

**검색 가능 항목**:
- 매물 제목
- 주소
- 상세 설명
- 매물 번호

**예시**:
- "강남" → 강남 지역 매물 검색
- "신축" → 신축 매물 검색
- "#123" → 특정 매물 번호로 검색

### 필터 사용

#### 매물 종류 필터

```
[매물종류 ▼]
  ├─ 전체
  ├─ 아파트 (45)
  ├─ 빌라/연립 (23)
  ├─ 원룸/오피스텔 (34)
  ├─ 상가 (18)
  └─ 사무실 (12)
```

#### 거래 종류 필터

```
[거래종류 ▼]
  ├─ 전체
  ├─ 매매 (67)
  ├─ 전세 (43)
  ├─ 월세 (38)
  └─ 분양 (2)
```

#### 계약 상태 필터

```
[상태 ▼]
  ├─ 전체
  ├─ 진행중 (120)
  ├─ 예약 (15)
  ├─ 계약완료 (10)
  └─ 거래중지 (5)
```

### 고급 필터

**더보기 필터** 버튼을 클릭하면 추가 옵션이 나타납니다:

```
┌────────────────────────────────────┐
│ 고급 필터                          │
├────────────────────────────────────┤
│ 가격 범위                          │
│ [________] ~ [________] 원         │
├────────────────────────────────────┤
│ 면적 범위                          │
│ [____] ~ [____] ㎡                │
├────────────────────────────────────┤
│ 등록일                             │
│ ○ 전체                            │
│ ○ 최근 1주일                      │
│ ○ 최근 1개월                      │
│ ○ 최근 3개월                      │
│ ○ 직접 선택                       │
│   [2025-10-01] ~ [2025-11-12]    │
├────────────────────────────────────┤
│ 지역                               │
│ [시/도 ▼] [구/군 ▼] [동 ▼]       │
├────────────────────────────────────┤
│      [초기화]        [적용]       │
└────────────────────────────────────┘
```

### 정렬 옵션

```
[정렬 ▼]
  ├─ 최신 등록순 (기본)
  ├─ 오래된 순
  ├─ 가격 높은 순
  ├─ 가격 낮은 순
  ├─ 조회수 높은 순
  └─ 매물번호 순
```

## 5.4 매물 수정하기

### 수정 방법

1. 매물 목록 또는 상세 화면에서 **수정** 버튼 클릭
2. 수정할 내용 변경
3. 하단의 **저장하기** 버튼 클릭

### 수정 가능 항목

✅ **수정 가능**:
- 제목
- 상세 주소
- 면적
- 가격
- 상세 정보 (층수, 방향 등)
- 상세 설명
- 사진 (추가/삭제/순서 변경)
- 계약 상태
- 연락처
- 테마/라벨

❌ **수정 불가**:
- 매물 종류 (삭제 후 재등록 필요)
- 기본 주소 (삭제 후 재등록 필요)
- 매물 번호 (자동 생성)

### 빠른 수정

목록에서 주요 정보만 빠르게 수정할 수 있습니다.

```
┌────────────────────────────────────┐
│ 빠른 수정 - #123                   │
├────────────────────────────────────┤
│ 가격                               │
│ 매매: [500000000] 원              │
├────────────────────────────────────┤
│ 계약 상태                          │
│ [진행중 ▼]                        │
├────────────────────────────────────┤
│      [취소]        [저장]         │
└────────────────────────────────────┘
```

## 5.5 매물 삭제하기

### 개별 삭제

1. 매물 목록 또는 상세 화면에서 **삭제** 버튼 클릭
2. 확인 메시지 확인

```
⚠️ 매물을 삭제하시겠습니까?

매물번호: #123
제목: 강남역 5분거리 신축 아파트

삭제된 매물은 복구할 수 없습니다.

[취소]  [삭제]
```

3. **삭제** 버튼 클릭

> ⚠️ **주의**: 삭제된 매물은 복구할 수 없습니다!

### 대신 이렇게 하세요

매물을 완전히 삭제하기 전에 다음을 고려하세요:

**계약 완료된 매물**:
- 상태를 **계약완료**로 변경
- 나중에 통계 자료로 활용

**일시적으로 숨기기**:
- 상태를 **거래중지**로 변경
- 필요 시 다시 **진행중**으로 변경

### 일괄 삭제

여러 매물을 한 번에 삭제할 수 있습니다.

1. 삭제할 매물들의 체크박스 선택
2. 상단 **일괄 작업** 버튼 클릭
3. **선택 항목 삭제** 선택
4. 확인 메시지에서 **삭제** 클릭

```
☑ #123 강남역 5분거리 신축 아파트
☑ #122 역세권 원룸 급매
☐ #121 상가 권리금 없음

[일괄 작업 ▼]
  ├─ 상태 변경
  ├─ 선택 항목 삭제
  └─ 엑셀 내보내기
```

## 5.6 매물 상태 변경

### 상태 종류

| 상태 | 의미 | 사용 시점 |
|------|------|-----------|
| **진행중** | 매물 공개 중 | 기본 상태 |
| **예약** | 계약 진행 중 | 계약금 받은 후 |
| **계약완료** | 거래 완료 | 잔금까지 완료 |
| **거래중지** | 일시 중단 | 소유주 요청 시 |

### 개별 상태 변경

1. 매물 상세 화면 또는 목록에서 상태 클릭
2. 새 상태 선택
3. 자동 저장

### 일괄 상태 변경

여러 매물의 상태를 한 번에 변경:

1. 변경할 매물들 체크박스 선택
2. **일괄 작업** > **상태 변경** 클릭
3. 새 상태 선택

```
선택된 3개 매물의 상태 변경

새 상태: [예약 ▼]

[취소]  [변경]
```

## 5.7 매물 복사하기

비슷한 매물을 등록할 때 기존 매물을 복사하여 시간을 절약할 수 있습니다.

### 복사 방법

1. 목록 또는 상세 화면에서 **복사** 버튼 클릭
2. 자동으로 복사본이 생성됨
3. 다른 정보만 수정하여 저장

**복사되는 항목**:
- ✅ 매물 종류
- ✅ 기본 주소
- ✅ 면적
- ✅ 상세 정보 (층수, 방향 등)
- ✅ 상세 설명
- ❌ 사진 (복사 안 됨)
- ❌ 가격 (새로 입력 필요)

**복사 후 제목**:
- 원본: "강남역 5분거리 신축 아파트"
- 복사본: "강남역 5분거리 신축 아파트 (복사본)"

## 5.8 엑셀로 내보내기/가져오기

### 엑셀 내보내기 (다운로드)

현재 보이는 매물 목록을 엑셀 파일로 다운로드합니다.

**방법**:
1. 원하는 필터/검색 조건 설정
2. 우측 상단 **내보내기** 버튼 클릭
3. 엑셀 파일 자동 다운로드

**내보내기 옵션**:
```
┌────────────────────────────────────┐
│ 엑셀 내보내기                      │
├────────────────────────────────────┤
│ 내보낼 항목:                       │
│ ☑ 기본 정보 (제목, 주소, 면적)    │
│ ☑ 가격 정보                        │
│ ☑ 상세 정보                        │
│ ☑ 연락처                           │
│ ☐ 상세 설명                        │
├────────────────────────────────────┤
│ 파일명: 매물목록_20251112.xlsx    │
├────────────────────────────────────┤
│      [취소]        [다운로드]     │
└────────────────────────────────────┘
```

**활용 예시**:
- 외부 공유용 자료 작성
- 백업
- 통계 분석

### 엑셀 가져오기 (업로드)

엑셀 파일로 여러 매물을 한 번에 등록합니다.

**방법**:

#### 1단계: 샘플 양식 다운로드

```
[가져오기 ▼]
  ├─ 샘플 양식 다운로드
  └─ 엑셀 파일 업로드
```

1. **샘플 양식 다운로드** 클릭
2. `매물등록_양식.xlsx` 파일 다운로드

#### 2단계: 양식 작성

다운로드한 엑셀 파일을 엽니다:

| 매물종류* | 제목* | 주소* | 상세주소 | 면적(㎡) | 거래종류* | 매매가 | 전세가 | 보증금 | 월세 | ... |
|-----------|-------|-------|----------|----------|-----------|--------|--------|--------|------|-----|
| 아파트 | 강남 아파트 | 서울 강남구... | 101동 | 84 | 매매 | 500000000 | | | | ... |
| 원룸 | 역세권 원룸 | 서울 강남구... | 2층 | 20 | 월세 | | | 10000000 | 500000 | ... |

**작성 규칙**:
- `*` 표시는 필수 항목
- 매물종류는 정확히 입력 (예: "아파트", "원룸")
- 가격은 숫자만 입력 (쉼표 없이)
- 날짜는 `YYYY-MM-DD` 형식

#### 3단계: 파일 업로드

1. **엑셀 파일 업로드** 클릭
2. 작성한 엑셀 파일 선택
3. **업로드** 버튼 클릭

```
┌────────────────────────────────────┐
│ 엑셀 파일 업로드                   │
├────────────────────────────────────┤
│ [파일 선택] 매물등록_양식.xlsx     │
├────────────────────────────────────┤
│ 검증 중...                         │
│ ✅ 총 10개 행                      │
│ ✅ 필수 항목 확인 완료             │
│ ⚠️ 경고 1개                        │
│    • 2행: 면적 누락 (기본값 적용) │
├────────────────────────────────────┤
│      [취소]        [등록]         │
└────────────────────────────────────┘
```

4. **등록** 버튼 클릭
5. 완료 메시지 확인

**업로드 결과**:
```
✅ 엑셀 업로드 완료

• 성공: 9건
• 실패: 1건
  - 3행: 주소 형식 오류

[실패 내역 다운로드]  [확인]
```

### 주의사항

⚠️ **엑셀 작업 시 주의**:
- 양식을 변경하지 마세요 (열 순서, 이름)
- 한 번에 100개 이하 권장
- 큰 파일은 여러 번에 나누어 업로드
- 업로드 전에 백업 권장
