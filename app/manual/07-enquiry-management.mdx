## 7.1 문의 관리 개요

홈페이지를 통해 들어온 고객 문의를 효율적으로 관리하고 처리하는 방법을 설명합니다.

**문의 유형**:
- 💰 매수 문의: 매물을 사고 싶은 고객
- 🏠 매도 문의: 매물을 팔고 싶은 고객
- ❓ 일반 문의: 기타 문의사항

## 7.3 문의 상태

### 상태 종류

| 상태 | 아이콘 | 의미 | 할 일 |
|------|--------|------|-------|
| **신규** | 🆕 | 새로 들어온 문의 | 빠른 응대 필요 |
| **처리중** | ⏳ | 응대 진행 중 | 계속 진행 |
| **완료** | ✅ | 처리 완료 | 만족도 확인 |
| **취소** | ❌ | 고객 요청 취소 | - |

### 상태별 필터

```
[상태 필터 ▼]
  ├─ 전체 (24)
  ├─ 신규 (5) 🔴
  ├─ 처리중 (12)
  ├─ 완료 (6)
  └─ 취소 (1)
```

## 7.5 문의 처리하기

### 처리 단계

```
1. 신규 문의 확인
   ↓
2. 상태를 "처리중"으로 변경
   ↓
3. 고객에게 전화 연락
   ↓
4. 요구사항 파악 및 메모 작성
   ↓
5. 조건에 맞는 매물 검색
   ↓
6. 매물 추천 (링크 전송)
   ↓
7. 현장 방문 일정 조율
   ↓
8. 상태를 "완료"로 변경
```

### 1단계: 상태 변경

문의 상세 화면에서 상태를 변경합니다.

```
상태: [신규 ▼]
      ↓ 클릭
┌────────────────┐
│ 신규           │
│ 처리중    ← 선택│
│ 완료           │
│ 취소           │
└────────────────┘
```

### 2단계: 담당자 지정

자신을 담당자로 지정하거나 다른 직원에게 배정합니다.

```
담당자: [미지정 ▼]
        ↓
┌────────────────┐
│ 박대리    ← 선택│
│ 이과장         │
│ 김사원         │
└────────────────┘
```

### 3단계: 고객 연락

전화번호를 클릭하면 자동으로 전화 걸기 (모바일).

```
📞 010-9876-5432 ← 클릭하여 전화
```

### 4단계: 메모 작성

통화 내용과 요구사항을 메모로 남깁니다.

```
처리 메모
┌────────────────────────────────────┐
│ [새 메모 추가...]                  │
│                                     │
│ 예시:                              │
│ • 통화 완료                        │
│ • 예산: 5억원                      │
│ • 선호 지역: 강남역 도보 10분 이내 │
│ • 방 개수: 3룸 이상                │
│ • 입주 희망: 2개월 이내            │
│ • 다음 액션: 이번 주말 현장 안내   │
│                                     │
│ [저장]                             │
└────────────────────────────────────┘
```

💡 **메모 작성 팁**:
- 통화 날짜/시간 기록
- 고객 요구사항 상세히 기록
- 다음 할 일 명시
- 다른 직원도 볼 수 있으므로 명확하게 작성

## 7.6 매물 연결하기

고객이 원하는 조건에 맞는 매물을 연결합니다.

### 매물 검색 및 연결

1. **매물 연결하기** 버튼 클릭

```
매물 연결
┌────────────────────────────────────┐
│ 조건에 맞는 매물 검색              │
│                                     │
│ 매물 종류: [아파트 ▼]             │
│ 거래 종류: [매매 ▼]               │
│ 가격 범위: [4억] ~ [6억] 원       │
│ 지역: [강남구 ▼]                  │
│                                     │
│ [검색하기]                         │
└────────────────────────────────────┘
```

2. 검색 결과에서 매물 선택

```
검색 결과 (3건)
┌────────────────────────────────────┐
│ ☐ #123 강남역 5분거리 아파트       │
│    매매 5억원 | 84㎡              │
│                                     │
│ ☐ #119 역세권 신축 아파트          │
│    매매 5.5억원 | 92㎡            │
│                                     │
│ ☐ #115 대치동 아파트               │
│    매매 4.8억원 | 79㎡            │
│                                     │
│      [취소]        [연결]         │
└────────────────────────────────────┘
```

3. **연결** 버튼 클릭

### 연결된 매물 확인

```
관련 매물 (2건)
┌────────────────────────────────────┐
│ #123 강남역 5분거리 아파트         │
│ 매매 5억원 | 84㎡                 │
│ [매물 보기] [공유하기] [연결 해제]│
├────────────────────────────────────┤
│ #119 역세권 신축 아파트            │
│ 매매 5.5억원 | 92㎡               │
│ [매물 보기] [공유하기] [연결 해제]│
└────────────────────────────────────┘
```

### 고객에게 공유

**공유하기** 버튼을 클릭하면:

```
매물 공유
┌────────────────────────────────────┐
│ 공유 방법 선택:                    │
│                                     │
│ [링크 복사]                        │
│ 문자나 메일로 전송                 │
│                                     │
│ [카카오톡]                         │
│ 카카오톡으로 바로 공유             │
│                                     │
│ [이메일]                           │
│ 이메일로 전송                      │
└────────────────────────────────────┘
```

## 7.7 문의 완료 처리

모든 처리가 끝나면 상태를 **완료**로 변경합니다.

### 완료 처리 방법

1. 상태를 **완료**로 변경

```
상태: [처리중 ▼] → [완료 ▼]
```

2. 완료 메모 작성 (선택사항)

```
완료 메모
┌────────────────────────────────────┐
│ 처리 결과를 기록하세요:            │
│                                     │
│ • 현장 방문 완료                   │
│ • #123 매물에 관심 있음            │
│ • 다음 주 계약 예정                │
│                                     │
│ [저장]                             │
└────────────────────────────────────┘
```

3. 고객 만족도 확인 (선택사항)

```
고객 만족도
┌────────────────────────────────────┐
│ 서비스에 만족하셨나요?             │
│                                     │
│ ⭐⭐⭐⭐⭐                         │
│                                     │
│ [나중에] [만족도 저장]            │
└────────────────────────────────────┘
```

## 7.8 계약으로 전환

문의가 계약으로 진행되면 바로 계약서를 작성할 수 있습니다.

### 전환 방법

1. 문의 상세 화면 우측 상단 **계약으로 전환** 버튼 클릭

```
┌────────────────────────────────────┐
│ 계약으로 전환하시겠습니까?         │
│                                     │
│ 문의 정보가 계약서에 자동으로       │
│ 입력됩니다.                        │
│                                     │
│ [취소]  [계약 생성]                │
└────────────────────────────────────┘
```

2. **계약 생성** 버튼 클릭

3. 계약 작성 화면으로 자동 이동

**자동 입력되는 정보**:
- ✅ 고객 정보 (매수인/매도인)
- ✅ 관련 매물
- ✅ 담당자
- ❌ 계약 금액 (수동 입력 필요)
- ❌ 결제 일정 (수동 입력 필요)

자세한 내용은 [8. 계약 관리](08-contract-management.md) 참조

## 7.11 알림 설정

### 신규 문의 알림

새 문의가 들어오면 자동으로 알림을 받을 수 있습니다.

**설정 방법**:
1. 내 정보 > 알림 설정
2. **신규 문의 알림** 체크
3. 알림 방법 선택

```
알림 설정
┌────────────────────────────────────┐
│ 신규 문의 알림                     │
│ ☑ 브라우저 알림                    │
│ ☑ 이메일 알림                      │
│ ☐ 문자 알림 (추가 요금)            │
├────────────────────────────────────┤
│ 알림 시간                          │
│ ○ 항상                            │
│ ● 업무시간만 (09:00-18:00)        │
│ ○ 커스텀                          │
└────────────────────────────────────┘
```

## 7.12 빠른 응답 템플릿 (선택사항)

자주 사용하는 답변을 템플릿으로 저장할 수 있습니다.

### 템플릿 생성

1. 설정 > 응답 템플릿
2. **+ 새 템플릿** 클릭

```
응답 템플릿 추가
┌────────────────────────────────────┐
│ 템플릿 이름                        │
│ [첫 연락 인사]                     │
├────────────────────────────────────┤
│ 내용                               │
│ ┌────────────────────────────────┐│
│ │ 안녕하세요. ○○부동산입니다.   ││
│ │ 문의 주셔서 감사합니다.        ││
│ │                                ││
│ │ 자세한 상담을 위해 전화        ││
│ │ 드리겠습니다.                  ││
│ └────────────────────────────────┘│
├────────────────────────────────────┤
│      [취소]        [저장]         │
└────────────────────────────────────┘
```

### 템플릿 사용

메모 작성 시 템플릿을 불러옵니다.

```
[새 메모 추가...]

[템플릿 불러오기 ▼]
  ├─ 첫 연락 인사
  ├─ 매물 추천 안내
  └─ 현장 방문 안내
```
