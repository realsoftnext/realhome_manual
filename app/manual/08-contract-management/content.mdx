## 8.1 계약 관리 개요

매매, 전세, 월세 등의 부동산 계약을 체계적으로 관리하는 방법을 설명합니다.

**계약 관리 기능**:
- 📄 계약서 작성
- 💰 결제 일정 관리
- 📎 문서 업로드
- 📊 진행 상황 추적
- 📝 활동 로그 자동 기록

## 8.3 계약 상태

### 상태 흐름

```
작성중 → 서명대기 → 서명완료 → 보관
  ✏️      ⏳        ✅       📦
           ↓
         해지 ❌
```

### 상태 설명

| 상태 | 아이콘 | 의미 | 할 일 |
|------|--------|------|-------|
| **작성중** | ✏️ | 계약서 작성 진행 중 | 계속 작성 |
| **서명대기** | ⏳ | 계약서 완성, 서명 필요 | 서명 받기 |
| **서명완료** | ✅ | 계약 체결 완료 | 결제 관리 |
| **해지** | ❌ | 계약 해지 | - |
| **보관** | 📦 | 완료된 계약 보관 | - |

│ 선택된 매물:                                       │

## 8.6 결제 관리

### 결제 완료 처리

결제가 입금되면 완료 처리합니다.

1. 해당 결제 항목의 **완료 처리** 버튼 클릭

```
결제 완료 처리
┌────────────────────────────────────┐
│ 중도금 입금 확인                   │
│                                     │
│ 금액: 100,000,000원                │
│ 예정일: 2025-12-12                │
│                                     │
│ 실제 입금일                        │
│ [2025-12-12 ▼]                    │
│                                     │
│ 입금 방법                          │
│ ○ 현금                            │
│ ● 계좌이체                        │
│ ○ 수표                            │
│                                     │
│ 메모                               │
│ [__________________________]       │
│                                     │
│      [취소]        [완료]         │
└────────────────────────────────────┘
```

2. **완료** 버튼 클릭

### 결제 진행률

계약 상세 화면에서 전체 진행률을 확인할 수 있습니다.

```
진행률: ━━━━━━━━━━░░░░░░░░░░ 50%

✅ 계약금 (10%) - 완료
✅ 중도금 (20%) - 완료
⏳ 잔금 (70%) - 대기중
```

## 8.8 계약 수정

### 수정 방법

1. 계약 상세 화면에서 **수정** 버튼 클릭
2. 변경할 내용 수정
3. **저장하기** 버튼 클릭

### 수정 가능 항목

✅ **수정 가능**:
- 결제 일정 (날짜, 금액)
- 당사자 연락처
- 담당자
- 특이사항
- 문서 추가/삭제

❌ **수정 불가**:
- 계약번호 (자동 생성)
- 매물 (계약 재생성 필요)
- 계약일 (중요 정보)

> ⚠️ **주의**: 이미 서명된 계약은 수정 시 법적 문제가 발생할 수 있습니다.
