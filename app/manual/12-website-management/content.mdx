# 12. 홈페이지 관리

[← 목차로 돌아가기](/)

## 12.1 홈페이지 관리 개요

홈페이지 회원과 약관을 관리하여 고객과의 소통을 효과적으로 진행할 수 있습니다.

**관리 가능 항목**:
- 👥 회원 관리 및 검색
- 📊 마케팅 수신동의 현황
- 📧 이메일 목록 다운로드
- 📜 약관 관리 및 버전 관리

> 💡 **관리자 권한 필요**: 이 기능은 관리자만 사용할 수 있습니다.

---

## 12.2 회원 관리

### 회원 관리 개요

웹사이트에 가입한 회원들을 관리하고, 마케팅 수신동의 현황을 확인하여 효과적으로 소통할 수 있습니다.

**주요 기능**:
- 👥 회원 검색 및 조회
- 📊 마케팅 수신동의 현황
- 📧 이메일 목록 다운로드
- 📱 연락처 목록 다운로드

### 회원 관리 화면 접속

좌측 메뉴에서 **🌐 홈페이지관리** > **회원 관리**를 클릭합니다.

```
회원 관리 (/admin/members)
┌────────────────────────────────────────────────────┐
│ 📊 통계                                            │
│ 전체 회원: 1,234명 | 마케팅 동의: 856명 (69%)    │
├────────────────────────────────────────────────────┤
│ [검색] [__________________] 🔍                     │
│ 검색 대상: [전체 ▼]  (이름, 이메일, 전화번호)     │
├────────────────────────────────────────────────────┤
│ 이름         이메일              마케팅동의        │
│ 홍길동       hong@email.com     ✅ 동의            │
│ 김철수       kim@email.com      ❌ 미동의          │
│ 이영희       lee@email.com      ✅ 동의            │
└────────────────────────────────────────────────────┘
```

### 회원 검색하기

#### 검색 방법

1. **검색어 입력**: 이름, 이메일, 전화번호로 검색
2. **검색 대상 선택**: 
   - 전체
   - 이름
   - 이메일
   - 전화번호
3. **🔍 검색 버튼 클릭**

#### 검색 예시

- **이름 검색**: "홍길동" 입력
- **이메일 검색**: "hong@email.com" 입력
- **전화번호 검색**: "010-1234-5678" 입력

### 마케팅 수신동의 현황 확인

#### 통계 보기

회원 관리 화면 상단에서 실시간 통계를 확인할 수 있습니다:

```
📊 통계
┌────────────────────────────────────┐
│ 전체 회원:      1,234명           │
│ 마케팅 동의:      856명 (69%)     │
│ 마케팅 미동의:    378명 (31%)     │
└────────────────────────────────────┘
```

#### 개별 회원 동의 상태

회원 목록에서 각 회원의 마케팅 수신동의 상태를 한눈에 확인:
- ✅ **동의**: 마케팅 메일/문자 수신 동의
- ❌ **미동의**: 마케팅 수신 거부

### 회원 목록 다운로드

#### 전체 회원 이메일 목록 다운로드

1. 회원 목록 상단 **[전체 이메일 다운로드]** 버튼 클릭
2. `members_email_전체_YYYYMMDD.csv` 파일 저장
3. 엑셀이나 메일 발송 프로그램에서 활용

**다운로드 파일 형식**:
```
이름,이메일,가입일,마케팅동의
홍길동,hong@email.com,2024-01-15,예
김철수,kim@email.com,2024-01-16,아니오
```

#### 마케팅 동의자만 다운로드

1. **[마케팅 동의자 이메일 다운로드]** 버튼 클릭
2. 마케팅 수신에 동의한 회원만 포함된 파일 저장
3. 프로모션, 이벤트 안내 발송에 활용

> 💡 **개인정보 보호**: 다운로드한 목록은 마케팅 목적으로만 사용하고, 제3자에게 제공하지 마세요.

#### 연락처 목록 다운로드

1. **[연락처 다운로드]** 버튼 클릭
2. 이메일과 전화번호가 포함된 파일 저장
3. SMS 발송 시 활용

**다운로드 파일 형식**:
```
이름,이메일,전화번호,마케팅동의
홍길동,hong@email.com,010-1234-5678,예
```

### 회원 정보 활용

#### 마케팅 활용

마케팅 동의자에게만 다음과 같은 정보를 발송하세요:

- 📧 **신규 매물 안내**: 조건에 맞는 매물 등록 시 알림
- 🎉 **프로모션 안내**: 이벤트, 할인 정보
- 📰 **뉴스레터**: 부동산 시장 동향, 투자 팁

#### 페이지네이션

많은 회원이 있을 경우 페이지 단위로 표시됩니다:

```
[◀ 이전]  1  2  3  4  5  [다음 ▶]
```

- 페이지당 20명씩 표시
- 화살표 버튼으로 페이지 이동

### API 활용 (개발자용)

회원 관리 기능은 다음 API를 통해 제공됩니다:

#### 회원 검색 및 조회
```
GET /api/admin/members
```
- **파라미터**: 
  - `search`: 검색어
  - `page`: 페이지 번호
  - `limit`: 페이지당 항목 수
- **응답**: 회원 목록 (페이지네이션)

#### 회원 통계
```
GET /api/admin/members/stats
```
- **응답**: 
  - 전체 회원 수
  - 마케팅 동의 회원 수
  - 비율

### 💡 활용 팁

1. **정기 검토**: 월 1회 마케팅 동의 현황을 확인하세요
2. **개인화**: 회원의 관심 매물 유형에 맞는 정보를 발송하세요
3. **타이밍**: 신규 매물 등록 후 24시간 이내 발송이 효과적입니다
4. **수신 거부**: 마케팅 메일에 수신 거부 링크를 반드시 포함하세요
5. **통계 활용**: 오픈율, 클릭률을 추적하여 개선하세요

### ⚠️ 주의사항

1. **개인정보 보호**: 다운로드한 정보는 안전하게 보관하세요
2. **동의 범위**: 마케팅 동의자에게만 프로모션을 발송하세요
3. **법률 준수**: 정보통신망법, 개인정보보호법을 준수하세요
4. **수신 거부**: 고객이 수신 거부를 요청하면 즉시 처리하세요

## 12.3 약관 관리

### 📌 약관이란?

약관은 회사와 고객 간의 거래 규칙을 정한 문서입니다. 
매물 등록, 고객 정보 처리 등 서비스 이용 시 지켜야 할 사항들이 적혀있습니다.

### 약관 관리 화면 접속

1. 좌측 메뉴에서 **🌐 홈페이지관리** > **약관 관리**를 클릭

### 🔑 관리 가능한 약관 4가지

#### 1️⃣ 회원 이용약관
- **적용 시점**: 회원가입할 때
- **내용**: 서비스 이용 규칙, 책임 사항
- **확인 방법**: 로그인 후 설정에서 언제든 확인 가능

#### 2️⃣ 개인정보 처리방침
- **적용 시점**: 회원가입할 때
- **내용**: 고객 정보를 어떻게 보호하고 관리하는지
- **확인 방법**: 로그인 후 설정에서 언제든 확인 가능

#### 3️⃣ 제3자정보제공약관
- **적용 시점**: 회원가입할 때
- **내용**: 고객 정보를 다른 회사와 공유하는 경우
- **확인 방법**: 로그인 후 설정에서 언제든 확인 가능

#### 4️⃣ 광고/마케팅 수신동의
- **적용 시점**: 회원가입할 때
- **내용**: 새로운 매물, 프로모션 정보를 받을지 말지
- **확인 방법**: 로그인 후 설정에서 언제든 변경 가능

### 약관 수정하기

1. **약관 선택**: 수정할 약관 클릭
2. **내용 편집**: 에디터에서 약관 내용 수정
3. **저장**: [저장하기] 버튼 클릭

> ⚠️ **중요**: 약관을 수정하면 모든 사용자가 다시 동의해야 합니다.

### 약관 버전 관리

- 약관 수정 시 자동으로 새 버전이 생성됩니다
- 이전 버전 약관도 보관되어 언제든 확인 가능합니다
- 각 사용자가 어떤 버전에 동의했는지 추적됩니다

### 약관 동의 이력 확인

직원의 약관 동의 이력은 **직원 관리 > 직원 상세보기**에서 확인할 수 있습니다:
- 동의한 약관 목록
- 동의 시간
- 접속 IP 주소

### 💡 활용 팁

1. **정기 검토**: 분기별로 약관 내용을 검토하세요
2. **명확한 표현**: 고객이 이해하기 쉬운 용어를 사용하세요
3. **변경 공지**: 약관 변경 시 충분한 공지 기간을 주세요
4. **법률 자문**: 중요한 변경 사항은 법률 전문가와 상담하세요
