## 1.1 시스템 접속 방법

### 웹브라우저로 접속하기

1. 웹브라우저(크롬, 엣지, 사파리 등)를 실행합니다
2. 주소창에 회사 홈페이지 주소를 입력합니다
3. 주소 뒤에 `/admin`을 붙여 관리자 페이지로 이동합니다
4. 홈페이지 하단에 오른쪽에 "관리자"링크가 있습니다. 그 링크를 클릭하셔도 됩니다.

**예시**:

```
https://yourcompany.com/admin
```

### 첫 접속 시

처음 접속하면 로그인 화면이 나타납니다. 관리자가 발급한 계정 정보를 입력하세요.

## 1.2 권장 환경

### PC 환경

| 항목         | 권장 사양                      | 최소 사양        |
| ---------- | -------------------------- | ------------ |
| **운영체제**   | Windows 10/11, Mac OS      | Windows 8 이상 |
| **웹브라우저**  | Chrome, Edge, Safari 최신 버전 | -            |
| **인터넷 속도** | 10Mbps 이상                  | 5Mbps        |
| **화면 해상도** | 1920×1080                  | 1366×768     |
| **메모리**    | 8GB 이상                     | 4GB          |

### 권장 브라우저

#### ✅ 추천

* **Google Chrome** (최신 버전)
* **Microsoft Edge** (최신 버전)
* **Safari** (최신 버전, Mac 사용자)

#### ⚠️ 비권장

* Internet Explorer (지원 종료)
* 오래된 버전의 브라우저

### 브라우저 업데이트 확인

#### Chrome

1. 우측 상단 ⋮ (점 3개) 클릭
2. **도움말** > **Chrome 정보**
3. 자동으로 업데이트 확인

#### Edge

1. 우측 상단 ⋯ (점 3개) 클릭
2. **도움말 및 피드백** > **Microsoft Edge 정보**
3. 자동으로 업데이트 확인

## 1.3 모바일 사용

### 모바일 접속

스마트폰이나 태블릿에서도 동일한 주소로 접속할 수 있습니다.

```
https://yourcompany.com/admin
```

### 모바일 권장 환경

| 항목        | 권장                             |
| --------- | ------------------------------ |
| **운영체제**  | iOS 14 이상, Android 10 이상       |
| **브라우저**  | Safari (iOS), Chrome (Android) |
| **화면 크기** | 5인치 이상                         |

### 모바일에서 할 수 있는 일

✅ **가능한 작업**

* 매물 목록 조회
* 매물 상세 정보 확인
* 문의 확인 및 응답
* 간단한 정보 수정
* 사진 촬영 후 업로드

❌ **PC에서 하는 것이 좋은 작업**

* 새 매물 등록 (많은 정보 입력)
* 엑셀 파일 업로드/다운로드
* 복잡한 설정 변경
* 대량 작업

### 모바일 활용 팁

💡 **외근 중에도 활용하세요**

* 현장에서 찍은 사진을 바로 업로드
* 고객 문의에 즉시 응답
* 매물 정보를 실시간 확인

## 1.5 접속 문제 해결

### 페이지가 열리지 않아요

1. **인터넷 연결 확인**
   * Wi-Fi 또는 유선 연결 상태 확인
   * 다른 웹사이트가 열리는지 확인
2. **주소 확인**
   * 주소를 정확히 입력했는지 확인
   * `/admin`을 빠뜨리지 않았는지 확인
3. **브라우저 캐시 삭제**
   * Chrome: Ctrl + Shift + Delete
   * 인터넷 사용 기록 삭제 선택

### 화면이 이상하게 보여요

1. **브라우저 확대/축소 확인**
   * Ctrl + 0 (숫자 0) 눌러서 100%로 리셋
2. **브라우저 업데이트**
   * 최신 버전으로 업데이트
3. **다른 브라우저로 시도**
   * Chrome, Edge 등 다른 브라우저 사용

## 1.6 보안 주의사항

### ⚠️ 계정 보안

* 비밀번호를 다른 사람과 공유하지 마세요
* 정기적으로 비밀번호를 변경하세요 (3개월마다 권장)
* 강력한 비밀번호를 사용하세요 (영문+숫자+특수문자)

### ⚠️ 공용 PC 사용 시

* 사용 후 반드시 **로그아웃**하세요
* 자동 로그인을 설정하지 마세요
* 비밀번호를 브라우저에 저장하지 마세요

### ⚠️ 개인정보 보호

* 고객 정보를 무단으로 외부에 유출하지 마세요
* 업무용으로만 사용하세요
