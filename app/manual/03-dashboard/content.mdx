# 3. 대시보드 살펴보기

## 3.1 대시보드란?

대시보드는 로그인 후 가장 먼저 보이는 화면으로, 중요한 정보를 한눈에 확인할 수 있는 홈 화면입니다.

---

## 3.2 대시보드 화면 구성

```
┌────────────────────────────────────────────────────┐
│ 🏠 로고     [검색]           👤 박대리님 🔔 알림   │
├────────────────────────────────────────────────────┤
│                                                     │
│ 📊 주요 지표                                       │
│ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│ │📋 총매물 │ │📞 신규  │ │📄 진행  │ │✅ 계약  │ │
│ │   150   │ │ 문의 12 │ │ 계약 5  │ │ 완료 8  │ │
│ └─────────┘ └─────────┘ └─────────┘ └─────────┘ │
│                                                     │
│ 📈 이번 달 통계                                    │
│ • 매물 등록: 23건 (전월 대비 +15%)                │
│ • 계약 체결: 8건 (전월 대비 +2%)                  │
│ • 홈페이지 방문: 1,234명 (전월 대비 +45%)         │
│                                                     │
│ 🔔 최근 활동                                       │
│ ┌────────────────────────────────────────┐        │
│ │ • 10분 전: 새 문의가 등록되었습니다    │        │
│ │   김철수님 (강남 아파트 문의)          │        │
│ │ • 1시간 전: 강남 아파트 조회 156회     │        │
│ │ • 2시간 전: 계약 C2025-001 계약금 입금 │        │
│ └────────────────────────────────────────┘        │
│                                                     │
│ 📅 오늘의 일정                                     │
│ • 14:00 - 김철수님 매물 안내                      │
│ • 16:00 - 강남 아파트 계약금 수령                 │
└────────────────────────────────────────────────────┘
```

## 3.3 주요 메뉴 설명

### 좌측 메뉴 (사이드바)

```
┌─────────────────┐
│ 🏠 홈            │ ← 대시보드
│ 📋 매물관리      │ ← 매물 등록/수정/조회
│ 👥 고객관리      │ ← 연락처 관리
│ 📞 문의관리      │ ← 홈페이지 문의
│ 📄 계약관리      │ ← 계약서 작성/관리
│ 📝 콘텐츠        │ ← 블로그, FAQ
│ ⚙️ 설정          │ ← 시스템 설정
└─────────────────┘
```

| 메뉴 | 기능 | 자주 사용도 |
|------|------|-------------|
| 🏠 **홈** | 대시보드 화면, 주요 지표 확인 | ⭐⭐⭐ |
| 📋 **매물관리** | 매물 등록, 수정, 삭제, 조회 | ⭐⭐⭐⭐⭐ |
| 👥 **고객관리** | 고객 연락처 등록 및 관리 | ⭐⭐⭐⭐ |
| 📞 **문의관리** | 홈페이지 문의 확인 및 처리 | ⭐⭐⭐⭐ |
| 📄 **계약관리** | 계약서 작성, 결제 관리 | ⭐⭐⭐ |
| 📝 **콘텐츠** | 블로그, FAQ 작성 | ⭐⭐ |
| ⚙️ **설정** | 매물 종류, 직원, 사이트 설정 | ⭐ |

## 3.4 주요 지표 카드

### 총 매물

현재 등록된 전체 매물 수를 보여줍니다.

```
┌─────────────┐
│ 📋 총 매물   │
│    150     │
│            │
│ [상세보기] │
└─────────────┘
```

- **클릭 시**: 매물 목록 페이지로 이동
- **활용**: 전체 매물 현황 파악

### 신규 문의

아직 처리하지 않은 새 문의 수를 표시합니다.

```
┌─────────────┐
│ 📞 신규문의  │
│     12     │
│    🆕      │
│ [확인하기] │
└─────────────┘
```

- **클릭 시**: 문의 관리 페이지로 이동
- **활용**: 빠른 고객 응대
- **알림**: 새 문의가 있으면 숫자가 빨간색으로 표시

### 진행 중인 계약

현재 진행 중인 계약 건수를 보여줍니다.

```
┌─────────────┐
│ 📄 진행계약  │
│      5     │
│    ⏳      │
│ [확인하기] │
└─────────────┘
```

- **클릭 시**: 계약 관리 페이지로 이동
- **활용**: 계약 진행 상황 모니터링

### 계약 완료

이번 달 완료된 계약 수를 표시합니다.

```
┌─────────────┐
│ ✅ 계약완료  │
│      8     │
│    📈      │
│ [상세보기] │
└─────────────┘
```

- **클릭 시**: 완료된 계약 목록 확인
- **활용**: 월별 성과 확인

## 3.5 이번 달 통계

월별 주요 지표를 전월과 비교하여 보여줍니다.

```
📈 이번 달 통계

┌────────────────────────────────────┐
│ 매물 등록                          │
│ ━━━━━━━━━━━━━━━━ 23건            │
│ 전월 대비 +15% ↗                  │
├────────────────────────────────────┤
│ 계약 체결                          │
│ ━━━━━━━━ 8건                      │
│ 전월 대비 +2% ↗                   │
├────────────────────────────────────┤
│ 홈페이지 방문                      │
│ ━━━━━━━━━━━━━━━━━━ 1,234명       │
│ 전월 대비 +45% ↗                  │
└────────────────────────────────────┘
```

### 통계 항목 설명

| 항목 | 의미 | 중요도 |
|------|------|--------|
| **매물 등록** | 이번 달 새로 등록한 매물 수 | ⭐⭐⭐ |
| **계약 체결** | 이번 달 완료된 계약 수 | ⭐⭐⭐⭐⭐ |
| **홈페이지 방문** | 홈페이지 방문자 수 | ⭐⭐ |

💡 **활용 팁**: 전월 대비 지표를 보고 영업 전략을 수립하세요.

## 3.6 최근 활동

시스템에서 최근 발생한 중요한 활동을 시간순으로 보여줍니다.

```
🔔 최근 활동

┌────────────────────────────────────────┐
│ 🆕 10분 전                              │
│ 새 문의가 등록되었습니다               │
│ 김철수님 - 강남 아파트 매수 문의       │
│ [문의 확인하기]                        │
├────────────────────────────────────────┤
│ 👀 1시간 전                             │
│ 강남역 5분거리 아파트 조회 156회       │
│ [매물 보기]                            │
├────────────────────────────────────────┤
│ 💰 2시간 전                             │
│ 계약 C2025-001 계약금 입금 완료        │
│ [계약 확인하기]                        │
├────────────────────────────────────────┤
│ 📋 3시간 전                             │
│ 박대리님이 새 매물을 등록했습니다      │
│ [매물 보기]                            │
└────────────────────────────────────────┘
```

### 활동 유형

| 아이콘 | 활동 유형 | 설명 |
|--------|-----------|------|
| 🆕 | 신규 문의 | 홈페이지에서 새 문의 접수 |
| 👀 | 매물 조회 | 매물 상세 페이지 조회 |
| 💰 | 결제 완료 | 계약금/중도금/잔금 입금 |
| 📋 | 매물 등록 | 새 매물 등록 |
| ✏️ | 정보 수정 | 매물/고객 정보 수정 |

💡 **팁**: 최근 활동을 통해 놓친 일이 없는지 확인하세요.

## 3.7 오늘의 일정

오늘 예정된 일정을 확인할 수 있습니다.

```

┌────────────────────────────────────────┐
│ 14:00  👤 고객 미팅                    │
│        김철수님 매물 안내               │
│        📍 강남역 5분거리 아파트         │
│        [일정 상세]                     │
├────────────────────────────────────────┤
│ 16:00  💰 계약금 수령                   │
│        강남 아파트 계약 (C2025-001)    │
│        금액: 50,000,000원              │
│        [계약 보기]                     │
├────────────────────────────────────────┤
│ 18:00  🏠 현장 방문                     │

│        [매물 보기]                     │

💡 **팁**: 일정은 계약 관리에서 자동으로 추가되거나 직접 등록할 수 있습니다.

## 3.8 상단 바 (헤더)

### 검색 기능

```
┌────────────────────────────────────────┐
│ [🔍 매물, 고객, 문의 검색...]          │
└────────────────────────────────────────┘
```

- 매물 제목, 주소 검색
- 고객 이름, 전화번호 검색
- 문의 내용 검색
- **단축키**: Ctrl + K

### 알림 아이콘 🔔

```
    🔔 (3)
    ↓
┌────────────────────────────────┐
│ 알림                           │
├────────────────────────────────┤
│ 🆕 새 문의가 등록되었습니다    │
│    5분 전                      │
├────────────────────────────────┤
│ 💰 계약금이 입금되었습니다     │

├────────────────────────────────┤

## 3.9 대시보드 커스터마이징

### 위젯 순서 변경 (관리자)

관리자는 대시보드에 표시할 위젯과 순서를 변경할 수 있습니다.

1. 우측 상단 **⚙️ (설정)** 아이콘 클릭
2. **대시보드 설정** 선택
3. 위젯을 드래그하여 순서 변경
4. 불필요한 위젯 숨기기
