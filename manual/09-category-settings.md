## 9.1 매물 종류 설정 개요

매물 종류(카테고리)를 추가하고, 각 종류별로 입력 항목을 맞춤 설정하는 방법을 설명합니다.

**설정 가능 항목**:
- 🏢 매물 종류 추가/삭제
- 📝 입력 항목 구성
- 🎨 아이콘 및 색상
- ⚙️ 필드 프리셋 관리
- 💰 수익률 유형 설정

> 💡 **관리자 권한 필요**: 이 기능은 관리자만 사용할 수 있습니다.

---

## 9.2 설정 화면 접속

1. 좌측 메뉴에서 **⚙️ 설정** 클릭
2. **매물 설정** 선택

```
설정
┌────────────────────────────────────────────────────┐
│ 📋 일반 설정                                       │
│ 👥 사용자 설정                                     │
│ ► 🏢 매물 설정                                     │
│   ├─ 매물 종류                                     │
│   ├─ 필드 프리셋                                   │
│   └─ 수익률 유형                                   │
│ 📄 계약 설정                                       │
└────────────────────────────────────────────────────┘
```

| 타입 | 설명 | 예시 |
|------|------|------|
| **텍스트** | 한 줄 입력 | 매물 제목, 단지명 |

| **파일** | 파일 업로드 | 도면, 사진 |

## 9.5 필드 프리셋 관리

### 프리셋이란?

자주 사용하는 입력 항목 세트를 저장하여 재사용할 수 있습니다.

### 프리셋 화면

설정 > 매물 설정 > **필드 프리셋** 선택

```
필드 프리셋 관리
┌────────────────────────────────────────────────────┐
│ [+ 새 프리셋 만들기]                               │
├────────────────────────────────────────────────────┤
│ 📦 기본 프리셋                            [수정]  │
│    • 주소, 제목, 가격, 면적, 설명                 │
│    • 사용 중: 0개 카테고리                        │
├────────────────────────────────────────────────────┤
│ 🏠 주거용 프리셋                          [수정]  │
│    • 주소, 제목, 가격, 면적, 방, 욕실, 층수       │
│    • 사용 중: 4개 카테고리                        │
│    • 적용된 카테고리: 아파트, 빌라, 단독, 오피스텔 │
├────────────────────────────────────────────────────┤
│ 🏪 상업용 프리셋                          [수정]  │
│    • 주소, 제목, 가격, 면적, 용도, 주차, 권리금   │
│    • 사용 중: 1개 카테고리                        │
│    • 적용된 카테고리: 상가/사무실                 │
├────────────────────────────────────────────────────┤
│ 🌾 토지용 프리셋                          [수정]  │
│    • 주소, 제목, 가격, 면적, 지목, 용도지역       │
│    • 사용 중: 0개 카테고리                        │
└────────────────────────────────────────────────────┘
```

│ │ ✅ 제목                              │          │
│ │ ✅ 가격                              │          │
│ │ ✅ 면적                              │          │

## 9.6 수익률 유형 설정

### 수익률이란?

임대 수익을 계산하는 방식을 정의합니다.

### 수익률 유형 화면

설정 > 매물 설정 > **수익률 유형** 선택

```
수익률 유형 관리

│    • 상태: 활성화 ✅                               │
├────────────────────────────────────────────────────┤
│ 📊 월 수익률                              [수정]  │
│    • 계산식: 월세 ÷ 매매가 × 100                  │
│    • 단위: %                                       │
│    • 상태: 활성화 ✅                               │

│    • 계산식: [(월세 × 12) - 세금] ÷ 매매가 × 100  │

## 9.7 실전 예제

### 예제 1: "농지" 카테고리 추가

**요구사항**:
- 농업용 토지 매물 관리
- 필요 항목: 지목, 용도지역, 접도, 경작 가능 여부

**단계**:
1. **+ 새 카테고리 추가**
2. 이름: `농지`
3. 아이콘: `🌾`
4. 색상: 초록색 (`#10B981`)
5. 프리셋: `토지용 프리셋` 선택
6. 추가 항목 설정:
   - 지목 (선택: 전, 답, 과수원...)
   - 용도지역 (선택: 농림지역, 보전녹지...)
   - 접도 (텍스트: 예 - 4m 도로 접함)
   - 경작 가능 (선택: 가능, 불가능)

### 예제 2: "펜션" 카테고리 추가

**요구사항**:
- 펜션/민박 매물 관리
- 필요 항목: 객실 수, 부대시설, 주차 대수

**단계**:
1. **+ 새 카테고리 추가**
2. 이름: `펜션`
3. 아이콘: `🏡`
4. 색상: 주황색 (`#F59E0B`)
5. 프리셋: `상업용 프리셋` 선택
6. 추가 항목 설정:
   - 객실 수 (숫자)
   - 부대시설 (복수 선택: 수영장, BBQ장, 개별 화장실...)
   - 주차 대수 (숫자)
   - 전망 (선택: 바다, 산, 계곡...)

## 9.8 카테고리 삭제

### 삭제 방법

1. 삭제할 카테고리의 **수정** 버튼 클릭
2. 우측 하단 **카테고리 삭제** 버튼 클릭
3. 확인 메시지 읽기
4. **삭제** 버튼 클릭

```
카테고리 삭제 확인
┌────────────────────────────────────┐
│ ⚠️ 경고                            │
│                                     │
│ "농지" 카테고리를 삭제하시겠습니까? │
│                                     │
│ • 이 카테고리의 매물: 5개           │
│ • 매물은 삭제되지 않습니다          │
│ • 매물은 "기타"로 이동됩니다        │
│                                     │
│      [취소]        [삭제]          │
└────────────────────────────────────┘
```

> ⚠️ **주의**: 카테고리를 삭제해도 매물은 삭제되지 않습니다. 매물은 "기타" 카테고리로 이동됩니다.

┌────────────────────────────────────┐
│ ≡ 주소                [수정]      │  ← 드래그
│ ≡ 제목                [수정]      │
│ ≡ 가격                [수정]      │
│ ≡ 전용면적            [수정]      │
│ ≡ 방 개수             [수정]      │
│ ≡ 욕실 개수           [수정]      │
└────────────────────────────────────┘
```

### 순서 번호로 변경

또는 각 항목의 **수정** 버튼을 클릭하여 순서 번호를 직접 입력할 수 있습니다.

항목 수정
┌────────────────────────────────────┐
│ 항목 이름: 방 개수                 │
│                                     │
│ 표시 순서                          │
│ [5] ▲▼                             │
│                                     │
│      [취소]        [저장]          │
└────────────────────────────────────┘
```

│    숨겨지지만 데이터는 유지됩니다. │
│                                     │
│      [취소]        [저장]          │

```

## 9.11 팁 & 주의사항

### 💡 유용한 팁

1. **프리셋 활용**: 비슷한 카테고리는 프리셋을 만들어 재사용하세요.
2. **필수 항목 최소화**: 필수 항목이 너무 많으면 등록이 어려워집니다.
3. **도움말 추가**: 복잡한 항목은 도움말 텍스트를 추가하세요.
4. **순서 최적화**: 자주 입력하는 항목을 위쪽에 배치하세요.

### ⚠️ 주의사항

1. **삭제 전 확인**: 카테고리 삭제 전 매물 개수를 확인하세요.
2. **항목 삭제**: 항목을 삭제하면 해당 데이터가 모두 삭제됩니다.
3. **타입 변경**: 항목 타입 변경 시 기존 데이터가 손실될 수 있습니다.
4. **필수 항목**: 이미 등록된 매물이 있으면 필수 항목 추가에 주의하세요.
