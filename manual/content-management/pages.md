## 페이지 관리란?

**페이지 관리**는 홈페이지에 자유로운 형식의 페이지를 만드는 기능입니다.

### 활용 예시
- 🏢 **회사 소개**: 회사 연혁, 비전, 조직도
- 👥 **팀 소개**: 직원 소개, 전문 분야
- 📞 **오시는 길**: 지도, 주차 안내, 대중교통
- 🎯 **서비스 안내**: 제공 서비스, 가격표
- 📰 **언론 보도**: 뉴스 기사, 수상 내역

### 주요 특징
- 🎨 **섹션 기반 구성**: Hero, Text, Image, Features 등 다양한 섹션
- 🤖 **AI 자동 생성**: 제목만 입력하면 AI가 페이지 구조 생성
- 🔗 **메뉴 연동**: 생성한 페이지를 메뉴에 쉽게 연결
- 📱 **반응형 디자인**: 모바일/태블릿에서도 최적화
- 🔍 **SEO 설정**: 검색 엔진 최적화 메타데이터 설정

|:---|:---|

## 새 페이지 만들기

{/* pages_create.png */}

### 수동 작성 방법

#### 1단계: 기본 정보 입력
```
제목: 회사 소개
슬러그: company-info (URL 주소로 사용)
부제: 고객과 함께 성장하는 부동산 전문 기업
```

#### 2단계: 커버 이미지 설정 (선택)
- 페이지 상단에 표시될 대표 이미지
- 권장 크기: 1920 x 600px

#### 3단계: 섹션 추가
**섹션 추가** 버튼을 클릭하여 원하는 섹션 타입 선택:
- **Hero**: 대형 타이틀 + 부제 + 배경
- **Text**: 일반 텍스트 블록
- **Image**: 이미지 + 캡션
- **Features**: 3단 특징 소개
- **Stats**: 통계 수치 강조

#### 4단계: SEO 설정 (선택)
```
SEO 제목: 회사 소개 | 부동산 전문 기업
SEO 설명: 20년 경력의 부동산 전문가들이 함께하는...
```

#### 5단계: 공개 설정
- **공개**: 체크하면 홈페이지에 즉시 표시
- **비공개**: 작성 중이거나 임시 저장

#### 6단계: 저장
- **"페이지 저장"** 버튼 클릭

{/* pages_create_success.png */}

## AI로 페이지 생성하기

{/* pages_ai_generate.png */}

### AI 생성 방법

#### 1단계: AI 생성 버튼 클릭
페이지 목록 상단의 **"AI로 페이지 생성"** 버튼 클릭

#### 2단계: 정보 입력
```
페이지 제목: 회사 소개
설명: 부동산 중개업 전문 회사, 20년 경력, 
      강남 지역 특화, 고객 만족도 98%
키워드: 부동산, 중개, 강남, 전문가
```

#### 3단계: AI 생성 실행
- **"AI로 생성하기"** 버튼 클릭
- AI가 자동으로 페이지 구조와 내용 생성

#### 4단계: 내용 수정
- AI가 생성한 초안을 확인
- 필요한 부분 수정
- 이미지 추가

#### 5단계: 저장
- 만족하면 **"페이지 저장"** 버튼 클릭

### AI 생성 예시

**입력:**
```
제목: 회사 소개
설명: 20년 경력의 부동산 전문 기업
```

**AI가 생성한 구조:**
```
┌─────────────────────────────────┐
│ [Hero 섹션]                     │
│ 회사 소개                       │
│ 20년 경력의 부동산 전문 기업     │
├─────────────────────────────────┤
│ [Text 섹션]                     │
│ 회사 연혁과 비전 소개            │
├─────────────────────────────────┤
│ [Features 섹션]                 │
│ ✓ 전문성   ✓ 신뢰   ✓ 만족      │
├─────────────────────────────────┤
│ [Stats 섹션]                    │
│ 20년 | 1,000+ | 98%             │
└─────────────────────────────────┘
```

> **💡 TIP**: AI 생성 후에도 자유롭게 섹션을 추가/삭제하고 내용을 수정할 수 있습니다.

## 페이지 수정하기

{/* pages_edit.png */}

### 수정 방법

1. 목록에서 수정할 페이지 찾기
2. 해당 행의 **"수정"** 버튼 클릭
3. 내용 변경:
   - 기본 정보 수정
   - 섹션 추가/삭제
   - 섹션 순서 변경 (드래그 앤 드롭)
4. **"페이지 저장"** 버튼 클릭

### 섹션 관리
- **추가**: "섹션 추가" 버튼
- **삭제**: 각 섹션의 "삭제" 버튼
- **순서 변경**: 섹션을 드래그하여 이동

---

## 페이지 삭제하기

{/* pages_delete.png */}

### 삭제 방법

1. 목록에서 삭제할 페이지 찾기
2. **메뉴 연결** 개수 확인
   - 0개: 삭제 가능
   - 1개 이상: 메뉴 연결 먼저 해제 필요
3. 해당 행의 **"삭제"** 버튼 클릭
4. 확인 메시지에서 **"확인"** 클릭

> **⚠️ 주의**: 
> - 삭제된 페이지는 복구할 수 없습니다
> - 메뉴에 연결된 페이지는 삭제할 수 없습니다

## 섹션 타입 안내

{/* pages_sections.png */}

### 1. Hero 섹션
페이지 상단에 크게 표시되는 대표 영역

**구성 요소:**
- 제목 (대형 폰트)
- 부제
- 배경 이미지 (선택)

**활용:**
- 페이지 첫인상
- 핵심 메시지 전달

---

### 2. Text 섹션
일반 텍스트 콘텐츠 영역

**구성 요소:**
- 제목 (선택)
- 본문 텍스트

**활용:**
- 상세 설명
- 회사 소개
- 서비스 설명

---

### 3. Image 섹션
이미지 + 설명 영역

**구성 요소:**
- 이미지
- 제목 (선택)
- 설명 (선택)

**활용:**
- 사무실 사진
- 팀 단체 사진
- 인증서

---

### 4. Gallery 섹션
여러 이미지를 그리드로 표시

**구성 요소:**
- 제목
- 이미지 목록 (최대 6개 권장)

**활용:**
- 프로젝트 포트폴리오
- 사무실 갤러리
- 수상 내역

---

### 5. Features 섹션
3단 구조로 특징 강조

**구성 요소:**
- 전체 제목
- 항목 1, 2, 3 (제목 + 설명)

**활용:**
- 서비스 특징
- 차별점 소개
- 핵심 가치

---

### 6. Stats 섹션
숫자로 성과 강조

**구성 요소:**
- 통계 1, 2, 3 (수치 + 설명)

**활용:**
- 회사 실적
- 고객 만족도
- 경력 연수

**예시:**
```
┌──────────┬──────────┬──────────┐
│   20+    │  1,000+  │   98%    │
│ 경력 년수 │ 계약 건수 │ 만족도   │
└──────────┴──────────┴──────────┘
```

---

### 7. CTA (Call-to-Action) 섹션
행동 유도 버튼 영역

**구성 요소:**
- 제목
- 부제
- 버튼 텍스트
- 버튼 링크

**활용:**
- 상담 신청
- 문의하기
- 매물 보기

## 메뉴에 페이지 연결하기

{/* pages_menu_link.png */}

### 연결 방법

1. **메뉴 관리** 페이지로 이동
2. **"+ 메뉴 추가"** 클릭
3. **메뉴 타입**에서 **"페이지"** 선택
4. 드롭다운에서 연결할 페이지 선택
5. 메뉴 제목 입력 (페이지 제목과 다르게 설정 가능)
6. 저장

### 메뉴 구조 예시
```
메인 메뉴
├─ 회사 소개 → /pages/company-info
├─ 서비스 안내 → /pages/services
├─ 팀 소개 → /pages/team
└─ 오시는 길 → /pages/location
```

> **💡 TIP**: 하나의 페이지를 여러 메뉴에 연결할 수 있습니다.

## FAQ

### Q1. 페이지와 공지사항의 차이는 무엇인가요?
**A.** 
- **페이지**: 회사 소개, 서비스 안내 등 고정 콘텐츠
- **공지사항**: 시간순으로 나열되는 업데이트 소식

### Q2. 슬러그(Slug)는 무엇인가요?
**A.** URL 주소로 사용되는 고유 식별자입니다.
```
페이지 제목: 회사 소개
슬러그: company-info
URL: https://your-site.com/pages/company-info
```
- 영문 소문자, 숫자, 하이픈(-)만 사용
- 한글 제목은 자동으로 영문으로 변환됨

### Q3. AI 생성 기능은 어떻게 작동하나요?
**A.** OpenAI GPT를 사용하여:
1. 입력한 정보 분석
2. 적절한 섹션 구조 제안
3. 초안 콘텐츠 생성
4. SEO 메타데이터 자동 생성

### Q4. 섹션 개수에 제한이 있나요?
**A.** 이론적으로는 제한이 없지만, 페이지 로딩 속도를 고려하여 10~15개 이내를 권장합니다.

### Q5. 비공개 페이지는 어떻게 활용하나요?
**A.** 
- 작성 중인 페이지 임시 저장
- 특정 기간에만 공개할 콘텐츠 준비
- 내부 검토용 페이지

### Q6. 페이지를 복사할 수 있나요?
**A.** 현재는 복사 기능이 없습니다. 유사한 페이지는 기존 페이지를 수정하거나 새로 작성해야 합니다.

### Q7. 페이지에 동영상을 넣을 수 있나요?
**A.** 직접 업로드는 안 되지만, YouTube나 Vimeo 링크를 텍스트 섹션에 삽입할 수 있습니다.

### Q8. 메뉴에 연결된 페이지를 수정하면 어떻게 되나요?
**A.** 페이지 내용이 즉시 반영됩니다. 슬러그를 변경하지 않는 한 메뉴 링크는 정상 작동합니다.

### Q9. 페이지 조회수는 언제 증가하나요?
**A.** 방문자가 페이지를 열 때마다 자동으로 증가합니다. 관리자가 미리보기로 보는 것은 제외됩니다.

### Q10. 페이지 템플릿이 있나요?
**A.** AI 생성 기능이 템플릿 역할을 합니다. "회사 소개", "서비스 안내" 등 일반적인 제목을 입력하면 적절한 구조를 제안합니다.

### Q11. SEO 설정은 필수인가요?
**A.** 선택사항이지만 강력히 권장합니다. 검색 엔진에 노출되려면 SEO 제목과 설명을 반드시 입력하세요.

### Q12. 여러 명이 동시에 같은 페이지를 수정하면?
**A.** 마지막에 저장한 사람의 내용이 적용됩니다. 가능하면 한 명씩 작업하고, 중요한 페이지는 백업해두세요.

## 💡 활용 팁

### 1. 효과적인 페이지 구조
```
[Hero 섹션] - 첫인상
    ↓
[Text 섹션] - 상세 설명
    ↓
[Features 섹션] - 핵심 특징
    ↓
[Stats 섹션] - 수치로 신뢰성
    ↓
[CTA 섹션] - 행동 유도
```

### 2. 슬러그 명명 규칙
```
✅ 좋은 예:
- about-us
- service-guide
- team-introduction

❌ 나쁜 예:
- page1
- 회사소개 (한글)
- About Us (공백)
```

### 3. 이미지 최적화
- 커버 이미지: 1920 x 600px
- 섹션 이미지: 1200 x 800px
- 용량: 500KB 이하 권장
- 형식: JPG, PNG, WebP

### 4. SEO 최적화
```
SEO 제목: 
- 50자 이내
- 페이지 핵심 키워드 포함
- 사이트명 추가

예: 회사 소개 | 강남 부동산 전문 | 리얼홈

SEO 설명:
- 150자 이내
- 구체적인 정보 제공
- 검색자 의도 반영

예: 20년 경력의 부동산 전문가들이 강남 지역
    아파트, 오피스텔 매매와 임대를 책임집니다.
    고객 만족도 98%, 1,000건 이상의 계약 실적.
```

### 5. 정기 점검 항목
- [ ] 오래된 정보 업데이트
- [ ] 깨진 이미지 확인
- [ ] 조회수가 낮은 페이지 개선
- [ ] 메뉴 연결 상태 확인
