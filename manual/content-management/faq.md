## FAQ란?

**FAQ**(Frequently Asked Questions)는 고객들이 자주 묻는 질문과 답변을 미리 정리해두는 기능입니다.

### 주요 특징
- ❓ **질문/답변 관리**: 고객 문의를 줄이는 셀프 서비스
- 🏷️ **카테고리 분류**: 주제별로 FAQ 구분
- 🔢 **순서 설정**: 중요한 질문을 먼저 배치
- 👁️ **공개 여부 제어**: 작성 중인 FAQ는 비공개로 저장
- 🔍 **검색 기능**: 질문/답변에서 빠른 검색

## 새 FAQ 등록하기

{/* faq_create.png */}

### 등록 단계

#### 1단계: 질문 입력
```
예: 상담은 어떻게 진행되나요?
```

#### 2단계: 답변 작성
```
예: 매물 상세 페이지의 "문의하기" 버튼을 클릭하거나, 
    대표 전화 02-1234-5678로 연락 주시면 
    친절하게 상담해 드립니다.
```

#### 3단계: 카테고리 선택
- 기존 카테고리 선택 또는 새로 입력
- 예: "일반 문의", "매물 문의", "계약 문의"

#### 4단계: 순서 설정
- 숫자가 작을수록 먼저 표시
- 같은 카테고리 내에서 정렬
- 예: 0, 10, 20... (10 단위 권장)

#### 5단계: 공개 설정
- **공개**: 체크하면 홈페이지에 즉시 표시
- **비공개**: 작성 중이거나 임시 저장

#### 6단계: 저장
- **"등록"** 버튼 클릭

## FAQ 수정하기

{/* faq_edit.png */}

### 수정 방법

1. 목록에서 수정할 FAQ 찾기
2. 해당 행의 **"수정"** 버튼 클릭
3. 내용 변경
4. **"수정 완료"** 버튼 클릭

### 수정 가능한 항목
- ✅ 질문
- ✅ 답변
- ✅ 카테고리
- ✅ 순서
- ✅ 공개 여부

> **💡 TIP**: 수정 중에는 화면 상단에 현재 수정 중인 FAQ가 표시됩니다.

## FAQ 삭제하기

{/* faq_delete.png */}

### 삭제 방법

1. 목록에서 삭제할 FAQ 찾기

## 카테고리 관리

{/* faq_category.png */}

### 카테고리란?
FAQ를 주제별로 분류하는 태그입니다.

### 카테고리 생성 방법
FAQ 등록 시 **카테고리** 필드에 새로운 이름을 입력하면 자동으로 생성됩니다.

### 추천 카테고리 예시

| 카테고리 | 설명 |
|:---|:---|
| **일반 문의** | 서비스 이용 방법, 영업시간 등 |
| **매물 문의** | 매물 등록, 수정, 삭제 방법 |
| **계약 문의** | 계약 절차, 필요 서류 등 |
| **결제 문의** | 결제 방법, 환불 정책 등 |
| **기술 지원** | 로그인 문제, 오류 해결 등 |

### 카테고리 수정
- 기존 FAQ의 카테고리를 변경하면 자동으로 적용
- 더 이상 사용하지 않는 카테고리는 자동으로 사라짐

## 순서 변경하기

{/* faq_sorting.png */}

### 순서 설정 방법

#### 방법 1: 등록/수정 시 숫자 입력
```
중요도 높음: 0, 10, 20
중요도 보통: 100, 110, 120
중요도 낮음: 200, 210, 220
```

#### 방법 2: 10 단위로 설정 (권장)
- 나중에 중간에 FAQ를 추가할 때 편리
```
FAQ 1: 순서 10
FAQ 2: 순서 20
FAQ 3: 순서 30

새로 추가 → 순서 15 (FAQ 1과 2 사이에 배치)
```

### 정렬 규칙
1. **같은 카테고리** 내에서 순서 번호대로 정렬
2. 순서 번호가 같으면 **작성일 최신순**으로 표시
3. 순서 번호가 **음수**도 가능 (최상단 배치)

## 실전 활용 예시

### 예시 1: 기본 FAQ 세트 만들기

```markdown
[일반 문의]
Q1 (순서: 10): 영업시간은 어떻게 되나요?
A1: 평일 오전 9시부터 오후 6시까지 영업합니다.

Q2 (순서: 20): 주차 공간이 있나요?
A2: 건물 지하에 무료 주차장이 있습니다.

[매물 문의]
Q3 (순서: 10): 매물 상세 정보는 어떻게 확인하나요?
A3: 매물 목록에서 원하는 매물을 클릭하면 상세 페이지로 이동합니다.

Q4 (순서: 20): 직접 방문 상담이 가능한가요?
A4: 네, 사전 예약 후 방문하시면 됩니다.
```

### 예시 2: 계절별 FAQ 관리

**여름철 추가**
```
Q: 냉방 시설이 있나요?
카테고리: 일반 문의
순서: 5 (최상단 배치)
```

**겨울철 수정**
```
Q: 난방 시설이 있나요?
카테고리: 일반 문의
순서: 5 (최상단 배치)
```

## FAQ 질문

### Q1. 카테고리는 몇 개까지 만들 수 있나요?
**A.** 개수 제한은 없습니다. 하지만 너무 많으면 방문자가 혼란스러울 수 있으므로 5~7개 정도가 적당합니다.

### Q2. 순서 번호는 꼭 연속된 숫자여야 하나요?
**A.** 아니요. 10, 20, 30처럼 건너뛰어도 됩니다. 오히려 나중에 중간에 FAQ를 추가할 때 편리합니다.

### Q3. 비공개 FAQ는 누구에게 보이나요?
**A.** 관리자 페이지에서만 보입니다. 홈페이지 방문자에게는 보이지 않습니다.

### Q4. FAQ를 임시 저장할 수 있나요?
**A.** "공개 여부"를 해제하고 저장하면 비공개 상태로 보관됩니다. 나중에 내용을 완성한 후 공개할 수 있습니다.

### Q5. 여러 개의 FAQ를 한 번에 삭제할 수 있나요?
**A.** 현재는 개별 삭제만 지원합니다. 하나씩 삭제해야 합니다.

### Q6. FAQ에 이미지를 첨부할 수 있나요?
**A.** 현재 버전에서는 텍스트만 지원합니다. 복잡한 설명이 필요한 경우 별도의 안내 페이지를 만들고 FAQ에서 링크로 연결하는 것을 권장합니다.

### Q7. 카테고리 순서를 바꿀 수 있나요?
**A.** 카테고리는 자동으로 알파벳/가나다순으로 정렬됩니다. 순서를 조정하려면 카테고리 이름 앞에 번호를 붙이는 방법을 사용할 수 있습니다.
```
예: 1. 일반 문의, 2. 매물 문의, 3. 계약 문의
```

### Q8. 같은 질문이 여러 카테고리에 해당하면 어떻게 하나요?
**A.** 가장 적합한 카테고리 하나를 선택하거나, 각 카테고리에 동일한 내용을 개별 FAQ로 등록할 수 있습니다.

### Q9. FAQ 작성 권한은 누구에게 있나요?
**A.** 관리자 페이지에 접근할 수 있는 직원 계정은 모두 FAQ를 작성할 수 있습니다. 권한 설정은 "직원 관리" 메뉴에서 조정할 수 있습니다.

### Q10. 고객이 자주 묻는 질문을 자동으로 FAQ에 추가할 수 있나요?
**A.** 현재는 수동으로 추가해야 합니다. "문의 관리" 메뉴에서 반복되는 질문을 파악한 후 FAQ로 등록하는 것을 권장합니다.

## 💡 활용 팁

### 1. 효과적인 질문 작성
```
❌ 가격은?
✅ 매물 가격은 어떻게 확인하나요?

❌ 계약
✅ 계약 시 필요한 서류는 무엇인가요?
```

### 2. 명확한 답변 작성
- **구체적인 정보 제공**: 모호한 표현 피하기
- **단계별 설명**: 복잡한 절차는 번호로 나열
- **연락처 포함**: 추가 문의가 필요한 경우 안내

### 3. 정기적인 업데이트
- 계절에 따라 변하는 내용 갱신
- 정책 변경 시 관련 FAQ 수정
- 더 이상 유효하지 않은 FAQ 삭제

### 4. 카테고리별 우선순위
```
[일반 문의] - 순서 10, 20, 30...
├── 가장 많이 묻는 질문
├── 영업시간, 연락처
└── 기타 일반 정보

[매물 문의] - 순서 10, 20, 30...
├── 매물 검색 방법
├── 매물 정보 확인
└── 문의 방법

[계약 문의] - 순서 10, 20, 30...
├── 계약 절차
├── 필요 서류
└── 결제 방법
```
