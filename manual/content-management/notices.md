## 공지사항이란?

**공지사항**은 홈페이지 방문자에게 중요한 정보를 전달하는 게시판 기능입니다.

### 주요 특징
- ✅ **고정 공지**: 중요한 공지를 최상단에 고정
- ⏰ **공개 예약**: 특정 날짜/시간에 자동 공개
- ⏳ **만료일 설정**: 기한이 지나면 자동으로 비공개
- 📊 **조회수 추적**: 각 공지사항의 조회수 확인
- 🔍 **검색 기능**: 제목/내용으로 빠른 검색

## 새 공지사항 작성하기

{/* notices_create.png */}

### 작성 단계

#### 1단계: 기본 정보 입력
```
제목: 2025년 설 연휴 휴무 안내
내용: 2025년 1월 28일(화)부터 2월 1일(토)까지 
      설 연휴로 인해 휴무입니다.
```

#### 2단계: 공개 설정
- **공개 여부**: 체크하면 즉시 공개, 해제하면 비공개
- **고정 여부**: 체크하면 목록 최상단에 고정 표시

#### 3단계: 공개 일정 설정 (선택)
- **공개 시작일시**: 특정 시간에 자동 공개
- **만료일시**: 기한 지나면 자동으로 비공개

#### 4단계: 저장
- **"등록"** 버튼 클릭

## 공지사항 수정하기

{/* notices_edit.png */}

### 수정 방법

1. 목록에서 수정할 공지사항 찾기
2. 해당 행의 **"수정"** 버튼 클릭
3. 내용 변경
4. **"수정 완료"** 버튼 클릭

> **💡 TIP**: 수정 중에는 화면 상단에 "수정 중" 표시가 나타납니다.

## 공지사항 삭제하기

{/* notices_delete.png */}

### 삭제 방법

1. 목록에서 삭제할 공지사항 찾기
2. 해당 행의 **"삭제"** 버튼 클릭
3. 확인 메시지에서 **"확인"** 클릭

> **⚠️ 주의**: 삭제된 공지사항은 복구할 수 없습니다.

## 고정 공지 설정

{/* notices_pin.png */}

### 고정 공지란?
- 목록 최상단에 항상 표시
- 📌 아이콘으로 구분
- 여러 개 고정 가능 (작성일 최신순 정렬)

### 고정/해제 방법

**방법 1: 작성/수정 시**
- "고정 여부" 체크박스 선택

**방법 2: 목록에서 바로 전환**
- 📌 아이콘 클릭하여 즉시 고정/해제

### 고정 공지 활용 예시
- ✅ 중요한 공지사항 (휴무 안내, 서비스 점검)
- ✅ 장기간 노출이 필요한 내용
- ✅ 신규 서비스 안내

## 공개 예약 설정

{/* notices_schedule.png */}

### 공개 예약이란?
- 미래의 특정 시간에 자동으로 공개
- 이벤트 공지를 미리 작성해두고 D-Day에 공개

### 설정 방법

1. 공지사항 작성 시 **"공개 시작일시"** 입력
   ```
   예: 2025-12-24 09:00
   ```

2. **"공개 여부"** 체크박스 선택

3. 저장하면 자동으로 "공개 예약" 상태로 표시

### 상태 확인
- **공개 예약**: 설정한 시간 전
- **공개 중**: 설정한 시간 이후

---

## 만료일 설정

{/* notices_expire.png */}

### 만료일이란?
- 특정 날짜/시간이 지나면 자동으로 비공개
- 기한이 있는 이벤트 공지에 유용

### 설정 방법

1. 공지사항 작성 시 **"만료일시"** 입력
   ```
   예: 2025-02-28 23:59
   ```

2. 저장하면 만료일까지 공개

### 상태 확인
- **공개 중**: 만료일 전
- **만료됨**: 만료일 이후 (자동 비공개)

## FAQ

### Q1. 고정 공지는 몇 개까지 설정할 수 있나요?
**A.** 개수 제한은 없습니다. 하지만 너무 많으면 방문자가 혼란스러울 수 있으므로 3~5개 정도가 적당합니다.

### Q2. 공개 예약과 만료일을 함께 사용할 수 있나요?
**A.** 네, 가능합니다. 예를 들어:
- 공개 시작: 2025-01-01 00:00
- 만료일: 2025-01-31 23:59
- → 1월 한 달 동안만 자동으로 공개됩니다.

### Q3. 비공개로 저장한 공지는 어떻게 공개하나요?
**A.** 목록에서 해당 공지의 **상태** 버튼을 클릭하거나, 수정 화면에서 "공개 여부"를 체크하고 저장하면 됩니다.

### Q4. 공지사항을 임시 저장할 수 있나요?
**A.** "공개 여부"를 해제하고 저장하면 비공개 상태로 보관됩니다. 나중에 내용을 완성한 후 공개할 수 있습니다.

### Q5. 공지사항 순서를 바꿀 수 있나요?
**A.** 기본적으로 작성일 최신순으로 정렬됩니다. 고정 공지는 항상 상단에 표시되며, 고정 공지 간에도 최신순으로 정렬됩니다.

### Q6. 작성한 공지사항이 홈페이지에 안 보여요
**A.** 다음을 확인하세요:
1. "공개 여부"가 체크되어 있는지
2. 공개 시작일시가 현재보다 미래로 설정되지 않았는지
3. 만료일이 지나지 않았는지

### Q7. 공지사항에 이미지를 첨부할 수 있나요?
**A.** 현재 버전에서는 텍스트만 지원합니다. 이미지가 필요한 경우 별도의 페이지 관리 기능을 이용하거나, 이미지 URL을 텍스트로 안내할 수 있습니다.

### Q8. 공지사항을 카테고리별로 분류할 수 있나요?
**A.** 현재는 카테고리 기능이 없습니다. 제목에 [공지], [이벤트] 같은 태그를 붙여 구분하는 것을 권장합니다.

### Q9. 공지사항 작성 권한은 누구에게 있나요?
**A.** 관리자 페이지에 접근할 수 있는 직원 계정은 모두 공지사항을 작성할 수 있습니다. 권한 설정은 "직원 관리" 메뉴에서 조정할 수 있습니다.

### Q10. 과거 공지사항도 계속 보관되나요?
**A.** 네, 삭제하지 않는 한 모든 공지사항이 보관됩니다. 만료된 공지도 관리자 페이지에서는 계속 확인할 수 있습니다.

## 💡 활용 팁

### 1. 효과적인 제목 작성
```
❌ 안내
✅ [긴급] 12월 25일 시스템 점검 안내

❌ 이벤트
✅ 🎉 신규 가입 고객 특별 할인 이벤트 (12/1~12/31)
```

### 2. 공지사항 유형별 관리
- **긴급 공지**: 고정 설정 + 제목에 [긴급] 표시
- **정기 공지**: 만료일 설정으로 자동 관리
- **이벤트 공지**: 공개 예약 + 만료일 함께 사용

### 3. 정기적인 점검
- 만료된 공지는 정기적으로 삭제
- 고정 공지가 너무 많지 않은지 확인
- 중복되는 내용은 하나로 통합
