## 이 기능은 무엇인가요?

**거래 종류 관리**는 매물을 등록할 때 선택하는 거래 방식(매매, 전세, 월세, 분양 등)을 관리하는 메뉴입니다.

### 이런 것들을 할 수 있습니다
- ✅ 매매, 전세, 월세 등 거래 종류를 추가하거나 삭제
- ✅ 각 거래 종류마다 필요한 가격 항목 설정
- ✅ 매물 등록할 때 나오는 순서 변경
- ✅ 계절별로 사용/중지 전환

### 왜 필요한가요?
- 매물마다 입력하는 가격이 다릅니다
  - 매매: 매매가만 입력
  - 전세: 전세가만 입력
  - 월세: 보증금 + 월세 둘 다 입력
- 이 설정을 미리 해두면 매물 등록이 훨씬 편리해집니다

### 메뉴 찾기
```
로그인 → 관리자 페이지 → 거래 종류 관리
```

## 거래 종류 보기

### 화면 설명

![거래 종류 목록 화면](/images/transaction_type/1.PNG)

화면에서 볼 수 있는 내용:

| 항목 | 설명 |
|------|------|
| **≡ 아이콘** | 마우스로 끌어서 순서 변경 |
| **거래 종류** | 매매, 전세, 월세 등의 이름 |
| **입력 항목** | 어떤 가격을 입력해야 하는지 |
| **사용/중지** | 현재 사용 중인지 여부 |
| **수정/삭제** | 내용 변경 또는 삭제 버튼 |


### 상태 변경
- 활성 상태이면 사용 가능한 상태
- 비활성 상태이면 다른 기능에서는 전혀 사용하지 않음
- 기존에 사용하다가 비활성상태로 전환해도 매물은 그대로 보입니다.
- 단지 새로운 매물을 등록할 때나 검색할 때 나오지 않습니다.


## 새로운 거래 종류 만들기

### 어떻게 시작하나요?
1. 화면 오른쪽 위의 **"거래 종류 추가"** 버튼을 클릭하세요

![거래 종류 추가 화면](/images/transaction_type/2.PNG)

### 방법 1: 빠른 선택 사용하기 (추천)

![노란색 빠른 선택 박스](/images/transaction_type/3.PNG)

가장 많이 사용하는 4가지는 버튼 하나로 만들 수 있습니다:

#### 매매 만들기
- **매매 템플릿** 버튼 클릭
- 자동으로 입력됨: 이름(매매), 입력항목(매매가, 권리금)

#### 전세 만들기
- **전세 템플릿** 버튼 클릭
- 자동으로 입력됨: 이름(전세), 입력항목(전세가)

#### 월세 만들기
- **월세 템플릿** 버튼 클릭
- 자동으로 입력됨: 이름(월세), 입력항목(보증금, 월세, 권리금)

#### 분양 만들기
- **분양 템플릿** 버튼 클릭
- 자동으로 입력됨: 이름(분양), 입력항목(분양가)

### 방법 2: 직접 입력하기

**[이미지 5]** 입력 항목들

#### 입력할 내용

| 항목 | 꼭 입력? | 설명 | 예시 |
|------|---------|------|------|
| **거래 종류 이름** | ✅ 필수 | 매물 등록 시 보이는 이름 | 상가임대, 단기월세 등 |
| **코드** | ✅ 필수 | 영문으로 입력 (자동으로 대문자로 바뀜) | SHOP, SHORTRENT 등 |
| **설명** | 선택 | 어떤 거래인지 간단히 설명 | "상가 임대용입니다" |
| **순서** | 선택 | 비워두면 맨 뒤에 표시됨 | 보통 0, 1, 2, 3... |

#### 가격 항목 선택 (최소 1개 선택)

**[이미지 4]** 가격 항목 체크박스

어떤 가격을 입력받을지 선택하세요:

| 선택 | 언제 필요한가요? |
|------|----------------|
| ☑️ **매매가** | 집이나 건물을 파는 경우 |
| ☑️ **전세가** | 전세로 임대하는 경우 |
| ☑️ **보증금** | 월세의 보증금 |
| ☑️ **월세** | 매달 내는 임대료 |
| ☑️ **분양가** | 신규 아파트 분양 |
| ☑️ **권리금** | 상가나 사무실의 권리금 |
| ☑️ **실입주금** | 신축빌라의 경우에 사용하기도 함 |

### 저장하기
1. 모든 내용을 입력했으면 **"등록하기"** 버튼 클릭
2. "거래 종류가 등록되었습니다" 메시지가 나오면 완료!


### 입력 오류 안내

| 오류 메시지 | 해결 방법 |
|------------|---------|
| "거래 종류 이름을 입력해주세요" | 이름을 입력하지 않았습니다 |
| "코드를 입력해주세요" | 영문 코드를 입력하세요 |
| "이미 존재하는 코드입니다" | 다른 코드를 사용하세요 |
| "최소 하나의 가격 필드를 선택해주세요" | 가격 항목을 1개 이상 선택하세요 |

## 거래 종류 수정하기

### 어떻게 수정하나요?
1. 목록에서 수정하고 싶은 거래 종류의 **"수정"** 버튼 클릭
2. 내용을 고치고 **"수정하기"** 버튼 클릭

### 수정할 수 있는 것
- ✅ 거래 종류 이름 (예: "매매" → "일반매매")
- ✅ 설명 내용
- ✅ 순서 번호
- ✅ 가격 항목 추가/제거

### 수정할 수 없는 것
- ❌ 코드 (한번 정하면 못 바꿉니다)

### 💡 주의할 점
- 이미 등록된 매물은 그대로 유지됩니다
- 가격 항목을 없애도 기존 매물 가격은 남아있습니다

## 순서 바꾸기

### 마우스로 끌어서 순서 변경하기

### 사용 방법
1. 왼쪽의 **≡ 아이콘**에 마우스를 올립니다
2. 마우스가 **손 모양**으로 바뀝니다
3. **클릭한 채로 끌어서** 원하는 위치로 옮깁니다
4. 놓으면 순서가 바로 바뀝니다

### 언제 순서를 바꾸나요?
- 매물 등록할 때 자주 쓰는 거래 종류를 위로
- 예: 매매 → 전세 → 월세 → 분양 순서로

## 사용/중지 설정

### 거래 종류 끄고 켜기

### 어떻게 하나요?
1. 상태 버튼을 클릭합니다
2. 확인 창이 나오면 **"확인"** 클릭
3. 바로 변경됩니다

### 버튼 색깔 의미

| 색깔 | 상태 | 의미 |
|------|------|------|
| 🟢 **초록색** | 사용 | 매물 등록 시 선택할 수 있음 |
| ⚪ **회색** | 중지 | 매물 등록 시 안 보임 |


### 언제 사용하나요?

#### 계절별 관리
- **여름**: 휴가용 단기월세만 사용
- **겨울**: 분양 위주로 사용

#### 지역별 관리
- **서울/수도권**: 매매, 전세, 월세 사용
- **지방**: 전세를 중지하고 월세만 사용

### 💡 안심하세요
- 중지해도 기존 매물은 그대로 있습니다
- 언제든 다시 켤 수 있습니다
- 삭제하는 게 아니라 숨기는 것입니다

## 거래 종류 지우기

### 삭제하는 방법
1. **"삭제"** 버튼 클릭
2. 확인 창이 나오면 내용을 읽고 **"확인"** 클릭


### ⚠️ 삭제 전에 꼭 확인하세요!

#### 삭제 가능한 경우
- ✅ 아직 매물이 하나도 없는 거래 종류
- ✅ 테스트로 만들어본 거래 종류


#### 삭제 안 되는 경우
- ❌ 이미 매물이 등록된 거래 종류
- 예: "이 거래 종류를 사용하는 매물이 5개 있습니다"


### 매물이 있을 때는?

#### 방법 1: 매물 삭제하기
1. 해당 거래 종류로 등록된 매물을 모두 삭제
2. 그 다음 거래 종류 삭제

#### 방법 2: 매물 변경하기
1. 매물을 하나씩 열어서
2. 거래 종류를 다른 것으로 변경
3. 모두 변경한 후 삭제

#### 방법 3: 중지만 하기 (추천!)
- 삭제하지 말고 **"중지"**로만 변경
- 기존 데이터가 모두 보존됨
- 나중에 다시 쓸 수 있음

### 💡 추천 방법

| 상황 | 추천 |
|------|------|
| 잘못 만든 거래 종류 | 삭제 |
| 이번 달만 안 쓰는 거래 종류 | 중지 |
| 매물이 있는 거래 종류 | 중지 (삭제 불가) |

### ⚠️ 주의!
- **삭제하면 복구 안 됩니다**
- 확실할 때만 삭제하세요
- 고민된다면 "중지"를 사용하세요

## 자주 묻는 질문

### Q1. 거래 종류를 왜 만들어야 하나요?
**A.** 매물마다 입력하는 가격이 다르기 때문입니다.
- 매매는 "매매가"만
- 월세는 "보증금"과 "월세" 둘 다
- 이렇게 미리 설정해두면 매물 등록이 편해집니다

### Q2. 코드는 왜 영문으로 입력하나요?
**A.** 시스템 내부에서 사용하는 고유 이름입니다.
- 한글은 중복될 수 있지만 코드는 중복 불가
- 예: "매매"라는 이름이 2개 있어도 코드는 달라야 함

### Q3. 삭제와 중지의 차이는?
**A.** 
- **삭제**: 완전히 없어짐 (복구 불가)
- **중지**: 숨기기만 함 (언제든 다시 켤 수 있음)
- **추천**: 확실하지 않으면 "중지"를 사용하세요

### Q4. 순서를 왜 바꾸나요?
**A.** 매물 등록 시 자주 쓰는 거래 종류가 위에 있으면 편합니다.
- 예: 매매를 많이 하면 매매를 1번으로

### Q5. 권리금은 언제 사용하나요?
**A.** 주로 상가나 사무실 거래에서 사용합니다.
- 주택 매매: 권리금 보통 없음
- 상가 임대: 권리금 있는 경우 많음

### Q6. 거래 종류를 삭제하려는데 안 됩니다
**A.** 이미 매물이 등록되어 있기 때문입니다.
- 해결 방법 1: 매물을 삭제하거나 다른 거래 종류로 변경
- 해결 방법 2: 삭제 대신 "중지"만 하기 (추천)

### Q7. 빠른 선택 템플릿이 뭔가요?
**A.** 자주 쓰는 4가지(매매, 전세, 월세, 분양)를 버튼 하나로 만들어주는 기능입니다.
- 일일이 입력할 필요 없이 바로 만들어집니다

### Q8. 여러 개의 가격을 동시에 입력할 수 있나요?
**A.** 네, 가능합니다.
- 예: "매매+임대" → 매매가, 보증금, 월세 모두 체크
- 매물 등록 시 모든 가격을 입력할 수 있습니다

### Q9. 실수로 삭제했는데 복구할 수 있나요?
**A.** 아니요, 삭제된 거래 종류는 복구가 불가능합니다.
- 그래서 "중지"를 추천드립니다

### Q10. 계절마다 거래 종류를 바꿀 수 있나요?
**A.** 네, "사용/중지" 기능을 활용하세요.
- 여름: 단기월세 사용, 분양 중지
- 겨울: 분양 사용, 단기월세 중지

## 실전 활용 예시

### 예시 1: 처음 시작하기 (기본 설정)

**상황**: 새로 시스템을 도입한 중개사무소

**설정 순서**:
1. **매매 템플릿** 버튼 클릭 → 바로 만들어짐
2. **전세 템플릿** 버튼 클릭 → 바로 만들어짐
3. **월세 템플릿** 버튼 클릭 → 바로 만들어짐
4. 완료! 이제 매물 등록 가능


### 예시 2: 상가 전문 중개사무소

**상황**: 상가만 전문으로 하는 중개사무소

**만들 거래 종류**:
- **상가 매매**: 매매가 + 권리금
- **상가 월세**: 보증금 + 월세 + 권리금

**설정 방법**:
1. "거래 종류 추가" 클릭
2. 이름: "상가 매매", 코드: "SHOP_SALE"
3. 체크: 매매가, 권리금
4. 등록 완료


### 예시 3: 계절별 전환

**상황**: 여름에는 휴가용 단기월세, 겨울에는 일반 거래만

**여름 설정** (6-8월):
1. "단기월세" 거래 종류를 "사용"으로
2. "분양" 거래 종류를 "중지"로

**겨울 설정** (12-2월):
1. "단기월세" 거래 종류를 "중지"로
2. "분양" 거래 종류를 "사용"으로

## 처음 사용자를 위한 간단 가이드

### 1단계: 로그인하기
```
홈페이지 → 로그인 → 관리자 계정
```

### 2단계: 거래 종류 메뉴 찾기
```
관리자 페이지 → 좌측 메뉴 → 거래 종류 관리
```

### 3단계: 기본 거래 종류 만들기
```
① "거래 종류 추가" 버튼 클릭
② "매매 템플릿" 버튼 클릭
③ "등록하기" 버튼 클릭
④ 완료!
```

### 4단계: 나머지도 같은 방법으로
```
전세 템플릿 → 등록
월세 템플릿 → 등록
분양 템플릿 → 등록 (필요시)
```

### 5단계: 매물 등록해보기
```
매물 등록 메뉴로 가면 방금 만든 거래 종류가 나타남!
```

## 도움말

### 💡 이 기능의 장점

#### 타 시스템 대비 우수한 점
1. **드래그로 순서 변경**
   - 다른 시스템: 숫자 입력해서 순서 변경
   - 우리 시스템: 마우스로 끌어서 간편하게
