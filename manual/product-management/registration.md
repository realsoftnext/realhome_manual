# 4. 매물 등록하기

[← 목차로 돌아가기](../README.md)

---

매물을 등록하는 상세한 방법을 단계별로 설명합니다.

---

## 4.1 매물 등록 시작하기

### 등록 페이지 접속

1. 좌측 메뉴에서 **📋 매물관리** 클릭
2. 우측 상단 **+ 매물 등록** 버튼 클릭

또는 대시보드에서 **빠른 실행 > + 매물 등록** 클릭

---

## 4.2 1단계: 매물 종류 선택

가장 먼저 매물의 종류를 선택합니다. 이에 따라 입력 항목이 자동으로 달라집니다.

```
매물 종류 선택 *
┌─────────────────────────────────┐
│ [매물 종류를 선택하세요 ▼]      │
└─────────────────────────────────┘

선택 옵션:
┌─────────────────────────────────┐
│ 주거용                           │
│  ├─ 아파트                      │
│  ├─ 빌라/연립                   │
│  ├─ 단독/다가구                 │
│  └─ 원룸/오피스텔               │
│                                  │
│ 상업용                           │
│  ├─ 상가                        │
│  ├─ 사무실                      │
│  └─ 건물                        │
│                                  │
│ 기타                             │
│  └─ 토지                        │
└─────────────────────────────────┘
```

> 💡 **중요**: 매물 종류를 잘못 선택하면 나중에 변경할 수 없으므로 신중하게 선택하세요.

---

## 4.3 2단계: 기본 정보 입력

### 매물 제목 *

고객에게 보일 매물의 제목을 입력합니다.

```
매물 제목 *
┌─────────────────────────────────┐
│                                  │
└─────────────────────────────────┘
```

**좋은 제목 예시**:
- ✅ "강남역 5분거리 신축 아파트 급매"
- ✅ "역세권 풀옵션 원룸 즉시입주"
- ✅ "1층 카페가능 상가 권리금 없음"

**피해야 할 제목**:
- ❌ "아파트" (너무 간단)
- ❌ "급매매매매매" (과도한 강조)

### 주소 입력 *

#### 주소 검색

1. **주소 검색** 버튼 클릭

```
주소 *
┌─────────────────────────────────┐
│ [주소 검색] 🔍                   │
└─────────────────────────────────┘
```

2. 주소 검색 팝업창이 나타남

```
┌─────────────────────────────────┐
│ 주소 검색                        │
│                                  │
│ [도로명/지번 주소 입력...]       │
│ [검색하기]                      │
│                                  │
│ 검색 결과:                      │
│ ○ 서울 강남구 테헤란로 123      │
│ ○ 서울 강남구 역삼동 789-1      │
│ ○ 서울 강남구 테헤란로 125      │
│                                  │
│ [선택]                          │
└─────────────────────────────────┘
```

3. 원하는 주소 선택
4. 선택한 주소가 자동으로 입력됨

#### 상세 주소 입력

```
선택된 주소:
서울특별시 강남구 테헤란로 123

상세 주소
┌─────────────────────────────────┐
│ 101동 1205호                    │
└─────────────────────────────────┘
```

**상세 주소 예시**:
- 아파트: "101동 1205호"
- 빌라: "2층 201호"
- 상가: "1층 102호"
- 건물: "3층"

> 💡 **자동 지도 표시**: 주소를 선택하면 지도 좌표가 자동으로 설정됩니다.

### 면적 입력

제곱미터(㎡) 또는 평 중 하나만 입력하면 자동으로 변환됩니다.

```
면적
┌───────────────┬───────────────┐
│ [_____] ㎡    │ [_____] 평    │
│ (제곱미터)     │               │
└───────────────┴───────────────┘
```

**변환 공식**:
- 1평 = 3.3058㎡
- 1㎡ = 0.3025평

**예시**:
- 84㎡ 입력 → 자동으로 25.41평 표시
- 32평 입력 → 자동으로 105.79㎡ 표시

## 4.4 3단계: 거래 정보 입력

### 거래 종류 선택 *

하나 이상의 거래 종류를 선택하고 가격을 입력합니다.

#### 매매

```
☑ 매매
┌─────────────────────────────────┐
│ 매매가 (원)                     │
│ [_____________]                 │
│ 예: 500000000 (5억원)           │
└─────────────────────────────────┘
```

#### 전세

```
☑ 전세
┌─────────────────────────────────┐
│ 전세가 (원)                     │
│ [_____________]                 │
│ 예: 300000000 (3억원)           │
└─────────────────────────────────┘
```

#### 월세

```
☑ 월세
┌──────────────┬──────────────────┐
│ 보증금 (원)   │ 월세 (원)        │
│ [__________] │ [__________]     │
│ 예: 10000000 │ 예: 1500000      │
└──────────────┴──────────────────┘
```

#### 분양

```
☑ 분양
┌─────────────────────────────────┐
│ 분양가 (원)                     │
│ [_____________]                 │
└─────────────────────────────────┘
```

> 💡 **중복 선택 가능**: 한 매물에 여러 거래 유형을 설정할 수 있습니다.
> 
> **예시**: 전세 3억원 또는 월세 보증금 1억/월세 150만원

### 권리금 (상가인 경우)

```
☑ 권리금 있음
┌─────────────────────────────────┐
│ 권리금 (원)                     │
│ [_____________]                 │
└─────────────────────────────────┘
```

### 계약 상태

```
계약 상태 *
┌─────────────────────────────────┐
│ [진행중 ▼]                      │
└─────────────────────────────────┘

옵션:
• 진행중 (매물 공개 중)
• 예약 (계약 진행 중)
• 계약완료 (거래 완료)
• 거래중지 (일시 중단)
```

## 4.6 5단계: 상세 설명 작성

매물의 특징, 장점, 주변 환경 등을 자유롭게 작성합니다.

```
상세 설명
┌────────────────────────────────────┐
│                                     │
│                                     │
│                                     │
│                                     │
│                                     │
│                                     │
└────────────────────────────────────┘

[B] [I] [U] 📎 🖼️ (에디터 도구)
```

**작성 팁**:

1. **위치 정보**
   - 지하철역과의 거리
   - 버스 정류장
   - 주요 시설 (학교, 병원, 마트 등)

2. **매물 특징**
   - 옵션 (가전제품, 가구 등)
   - 최근 수리/리모델링 이력
   - 일조량, 조망

3. **입주 조건**
   - 즉시 입주 가능 여부
   - 반려동물 가능 여부
   - 주차 가능 대수

**예시**:
```
🏠 매물 특징
• 강남역 도보 5분 거리의 역세권 아파트
• 2023년 신축으로 깨끗한 상태
• 남향으로 하루 종일 햇볕이 잘 듭니다

🎯 옵션
• 풀옵션 (에어컨, 냉장고, 세탁기, 침대 등)
• 빌트인 주방가전
• 현관 중문 설치

🚇 교통
• 강남역 도보 5분
• 버스 정류장 바로 앞

🏢 주변 시설
• 롯데백화점 도보 3분
• 강남구청 도보 10분
• 초중고등학교 학군 우수

✅ 입주 조건
• 즉시 입주 가능
• 주차 1대 가능
• 반려동물 협의 가능
```

## 4.7 6단계: 사진 업로드

### 사진 올리기

```
매물 사진 *
┌────────────────────────────────────┐
│                                     │
│  📷 여기로 사진을 드래그하거나      │
│     [파일 선택] 버튼을 클릭하세요   │
│                                     │
│  • 최대 20장까지 업로드 가능        │
│  • JPG, PNG 형식                   │
│  • 파일당 최대 10MB                │
│                                     │
│  [파일 선택]                       │
└────────────────────────────────────┘
```

### 업로드된 사진 관리

```
업로드된 사진 (5장)

┌────────┐ ┌────────┐ ┌────────┐
│        │ │        │ │        │
│ 사진 1  │ │ 사진 2  │ │ 사진 3  │
│        │ │        │ │        │
└────────┘ └────────┘ └────────┘
  ⭐대표     [삭제]     [삭제]

┌────────┐ ┌────────┐
│        │ │        │
│ 사진 4  │ │ 사진 5  │
│        │ │        │
└────────┘ └────────┘
  [삭제]     [삭제]
```

### 사진 기능

**⭐ 대표 사진 설정**
- 매물 목록에서 보일 대표 사진
- 사진을 클릭하면 대표로 설정

**순서 변경**
- 사진을 드래그하여 순서 조정
- 왼쪽부터 순서대로 표시

**사진 삭제**
- [X] 버튼 클릭하여 삭제

### 사진 촬영 팁

💡 **좋은 매물 사진**:
- ✅ 밝은 시간대에 촬영 (낮 10시~14시)
- ✅ 가로 방향으로 촬영
- ✅ 방 전체가 보이도록 촬영
- ✅ 깨끗하게 정리 후 촬영

**추천 사진 구성**:
1. 외관/현관 (1장)
2. 거실 (2장)
3. 방 (2-3장)
4. 주방 (1장)
5. 욕실 (1장)
6. 전망/발코니 (1장)

## 4.9 8단계: 등록 완료

### 최종 확인

모든 정보를 입력했으면 하단의 버튼을 클릭합니다.

```
┌────────────────────────────────────┐
│ [임시저장]  [미리보기]  [등록하기] │
└────────────────────────────────────┘
```

**버튼 설명**:

- **임시저장**: 나중에 이어서 작성
- **미리보기**: 고객이 볼 화면 미리보기
- **등록하기**: 매물 등록 완료

### 필수 항목 확인

등록하기를 클릭하면 필수 항목(*)을 체크합니다.

```
⚠️ 필수 항목을 입력해주세요

다음 항목을 입력하지 않았습니다:
• 매물 제목
• 주소
• 거래 정보

[확인]
```

### 등록 완료

```
✅ 매물이 등록되었습니다!

매물번호: #12345
제목: 강남역 5분거리 신축 아파트

[매물 보기]  [계속 등록하기]  [목록으로]
```

## 4.10 등록 후 확인사항

### 홈페이지 확인

1. 홈페이지 매물 목록에서 확인
2. 정보가 정확한지 확인
3. 사진이 잘 보이는지 확인

### 수정이 필요한 경우

등록 후에도 언제든지 수정할 수 있습니다.

매물 관리 > 해당 매물 찾기 > [수정] 버튼
