## 이 기능은 무엇인가요?

**매물 종류 관리**는 아파트, 오피스텔, 빌라, 상가, 토지 등 부동산 유형을 관리하는 메뉴입니다.

### 이런 것들을 할 수 있습니다
- ✅ 아파트, 오피스텔, 빌라 등 매물 종류 추가/삭제
- ✅ **AI로 매물 종류에 맞는 이미지 자동 생성** ⭐
- ✅ 각 매물 종류마다 필요한 입력 항목 설정
- ✅ 3단계 계층 구조 (대분류 > 중분류 > 소분류)
- ✅ 드래그로 순서 변경
- ✅ 프리셋으로 빠른 설정

### 왜 필요한가요?
- 매물마다 필요한 정보가 다릅니다
  - 아파트: 평수, 층수, 방향, 입주년도
  - 상가: 업종 제한, 권리금, 영업시간
  - 토지: 지목, 용도지역, 도로조건
- 이 설정을 미리 해두면 매물 등록이 자동으로 구성됩니다

### 🌟 타 시스템 대비 우수한 점

#### 1. AI 이미지 자동 생성
- **일반 시스템**: 직접 이미지 업로드만 가능
- **우리 시스템**: 
  - "아파트"라고 입력하면 AI가 자동으로 아파트 이미지 생성
  - 프롬프트 자동 생성 기능으로 전문적인 이미지 생성
  - 생성된 이미지 라이브러리에서 재사용 가능

#### 2. 필드 프리셋 시스템
- **일반 시스템**: 일일이 필드 하나씩 추가
- **우리 시스템**:
  - "아파트 프리셋" 버튼 클릭 → 필요한 필드 50개 자동 생성
  - 아파트, 오피스텔, 빌라, 상가, 토지, 공장 등 8종 프리셋 제공

#### 3. 유연한 필드 관리
- **일반 시스템**: 고정된 입력 항목
- **우리 시스템**:
  - 필드 타입 9가지 (텍스트, 숫자, 선택, 라디오, 체크박스 등)
  - 필드별 필수/선택 설정
  - 필드별 도움말, 플레이스홀더 설정
  - 드래그로 순서 변경

#### 4. 3단계 계층 구조
- **일반 시스템**: 평면적인 1단계 분류
- **우리 시스템**:
  - 대분류: 주거, 상업, 토지
  - 중분류: 아파트, 오피스텔, 빌라
  - 소분류: 일반아파트, 재건축아파트, 신축아파트

#### 5. 거래 종류 연결
- **일반 시스템**: 모든 매물이 동일한 거래 종류
- **우리 시스템**:
  - 아파트는 매매+전세+월세
  - 상가는 상가매매+상가임대만
  - 매물 종류별 맞춤 거래 종류 설정

### 메뉴 찾기
```
로그인 → 관리자 페이지 → 매물 종류
```

## 새로운 매물 종류 만들기

### 어떻게 시작하나요?
1. 화면 오른쪽 위의 **"+ 매물 종류 추가"** 버튼을 클릭하세요

![매물 종류 추가 화면](/images/category/3.png)

### 기본 정보 입력


| 항목 | 꼭 입력? | 설명 |
|------|---------|------|
| **매물 종류명** | ✅ 필수 | 아파트, 오피스텔, 빌라 등 |
| **상위 카테고리** | 선택 | 비워두면 대분류, 선택하면 하위 분류 |
| **설명** | 선택 | 매물 종류에 대한 간단한 설명 |
| **사용 여부** | 선택 | 기본값은 "사용" |

#### 계층 구조 만들기

**대분류 만들기**
```
매물 종류명: 주거
상위 카테고리: (선택 안 함)
→ 결과: "대분류"로 생성됨
```

**중분류 만들기**
```
매물 종류명: 아파트
상위 카테고리: 주거 선택
→ 결과: "중분류"로 생성됨 (주거 밑에 표시)
```

**소분류 만들기**
```
매물 종류명: 재건축아파트
상위 카테고리: 아파트 선택
→ 결과: "소분류"로 생성됨 (아파트 밑에 표시)
```

## AI 이미지 자동 생성 ⭐

### 🎨 AI가 자동으로 이미지를 만들어줍니다!

![AI 이미지 생성 섹션](/images/category/5.png)

### 방법 1: 자동 프롬프트 생성 (추천!)

1. **매물 종류명을 입력**합니다 (예: "아파트")
2. **"AI 프롬프트 자동 생성"** 버튼을 클릭합니다
3. AI가 자동으로 프롬프트를 만들어줍니다
   ```
   "현대적인 아파트 단지의 전문적인 부동산 이미지,
   깨끗한 외관, 조경이 잘 된 모습, 밝은 하늘"
   ```
4. **"AI 이미지 생성"** 버튼을 클릭합니다
5. 20~30초 후 이미지가 자동으로 생성됩니다!

![AI가 생성한 아파트 이미지 예시](/images/category/6.png)

### 방법 2: 직접 프롬프트 입력

```
고급스러운 오피스텔 외관, 유리 파사드,
도심 속 건물, 현대적인 디자인, 전문 사진
```

### 방법 3: 이미지 라이브러리에서 선택

1. **"이미지 라이브러리"** 버튼 클릭
2. 이전에 생성한 이미지들이 나타납니다
3. 원하는 이미지를 클릭하여 선택

![이미지 라이브러리 화면](/images/category/7.png)

### 방법 4: 직접 업로드

- **"파일 선택"** 버튼으로 컴퓨터에서 이미지 업로드
- JPG, PNG 파일 지원
- 최대 5MB까지 가능

### 💡 AI 이미지 생성 팁

#### 좋은 프롬프트 예시
```
✅ "현대적인 아파트 단지, 깨끗한 외관, 푸른 하늘"
✅ "도심 속 고급 오피스텔, 유리 외관, 전문 사진"
✅ "한적한 빌라, 주택가 분위기, 따뜻한 느낌"
```

#### 나쁜 프롬프트 예시
```
❌ "아파트" (너무 짧음)
❌ "아주 멋진 최고의 아파트 사진" (추상적)
❌ "사람들이 많은 아파트" (부동산 이미지에 부적합)
```

## 필드 관리하기

### 필드란?
매물을 등록할 때 입력하는 항목들입니다.
- 아파트: 평수, 층수, 방향, 방/욕실 개수
- 상가: 업종, 보증금, 권리금, 영업시간

![필드 관리 화면](/images/category/8.png)

### 방법 1: 프리셋 사용하기 (초간단!) ⭐

![프리셋 선택 화면](/images/category/9.png)

1. **"프리셋 적용"** 버튼 클릭
2. 매물 종류 선택
   - 아파트 (50개 필드)
   - 오피스텔 (45개 필드)
   - 빌라/연립 (40개 필드)
   - 상가 (35개 필드)
   - 토지 (30개 필드)
   - 공장/창고 (25개 필드)
3. **"기존 필드 유지"** 또는 **"기존 필드 삭제"** 선택
4. **"적용"** 버튼 클릭
5. 자동으로 필드가 생성됩니다!

![프리셋 적용 후 모습](/images/category/10.png)

### 방법 2: 직접 추가하기

![필드 직접 추가 화면](/images/category/11.png)

#### 필드 추가 항목

| 항목 | 설명 | 예시 |
|------|------|------|
| **필드명** | 시스템 내부 이름 (영문) | `floor`, `area`, `rooms` |
| **라벨** | 화면에 표시되는 이름 | "층수", "면적", "방 개수" |
| **필드 타입** | 입력 방식 | 텍스트, 숫자, 선택 등 |
| **필수 여부** | 꼭 입력해야 하는지 | 필수 / 선택 |
| **플레이스홀더** | 입력 예시 | "예: 15층" |
| **도움말** | 추가 설명 | "전체 층수 중 해당 호수의 층" |

#### 필드 타입 9가지

![필드 타입 선택 화면](/images/category/12.png)

| 타입 | 사용 예시 |
|------|----------|
| **텍스트** | 주소, 단지명 |
| **여러줄 텍스트** | 상세설명, 특징 |
| **숫자** | 층수, 방 개수 |
| **선택(드롭다운)** | 방향(동향/서향/남향/북향) |
| **라디오 버튼** | 엘리베이터(있음/없음) |
| **체크박스** | 옵션(주차가능, 반려동물가능) |
| **날짜** | 입주가능일 |
| **예/아니오** | 즉시입주가능 |
| **면적** | 전용면적, 공급면적 |

### 필드 순서 변경

![필드 순서 드래그 변경](/images/category/13.png)

1. 필드 왼쪽의 **≡ 아이콘**을 마우스로 클릭
2. 끌어서 원하는 위치로 이동
3. 자동 저장됨

### 필드 수정/삭제

![필드 수정 화면](/images/category/14.png)

- **수정**: 연필 아이콘 클릭
- **삭제**: 휴지통 아이콘 클릭
- **활성화/비활성화**: 토글 스위치

## 거래 종류 연결하기

### 매물 종류별로 가능한 거래 종류를 설정합니다

![거래 종류 선택 화면](/images/category/15.png)

### 설정 방법

#### 예시 1: 아파트
```
✅ 매매
✅ 전세
✅ 월세
✅ 반전세
❌ 상가매매 (필요 없음)
❌ 상가임대 (필요 없음)
```

#### 예시 2: 상가
```
❌ 매매 (일반 매매는 필요 없음)
❌ 전세
❌ 월세
✅ 상가매매
✅ 상가임대
```

#### 예시 3: 분양
```
✅ 분양
❌ 매매
❌ 전세
```

### 기본 거래 종류 설정

**중요!** 여러 개를 선택했을 때, **기본값**을 지정해야 합니다.

![기본 거래 종류 선택](/images/category/16.png)

```
선택된 거래 종류:
○ 매매
● 전세 ← 기본값 (매물 등록 시 자동 선택됨)
○ 월세
```

### 💡 왜 필요한가요?

매물 등록할 때 해당 매물 종류에 맞는 거래 종류만 선택지에 나타납니다!

```
❌ 이전 시스템:
   아파트 등록하는데 "상가매매"도 선택 가능 → 혼란

✅ 우리 시스템:
   아파트 등록하면 "매매, 전세, 월세"만 나타남 → 명확!
```

## 수익률 타입 설정

### 이 매물 종류에서 제공할 수익률 정보를 선택합니다

![수익률 타입 선택 화면](/images/category/17.png)

### 수익률 타입 종류

| 수익률 타입 | 설명 | 적용 매물 |
|------------|------|----------|
| **임대 수익률** | 월세 대비 수익 | 아파트, 오피스텔 |
| **전세가율** | 매매가 대비 전세가 비율 | 아파트, 빌라 |
| **투자 회수 기간** | 원금 회수까지 기간 | 상가, 건물 |
| **순수익률** | 실제 수익 계산 | 상가, 사무실 |

### 설정 예시

#### 아파트
```
✅ 임대 수익률
✅ 전세가율
❌ 투자 회수 기간 (주거용에는 불필요)
```

#### 상가
```
✅ 임대 수익률
❌ 전세가율 (상가에는 없음)
✅ 투자 회수 기간
✅ 순수익률
```

## 테마 연결하기 ⭐

### 매물 종류별로 사용 가능한 테마를 설정합니다

**[이미지 17-1]** 테마 선택 화면

### 💡 테마란?

테마는 매물의 **특징이나 용도**를 나타내는 태그입니다.
- 예: 신혼부부, 투자용, 역세권, 학군, 반려동물, 넓은평수 등
- 한 매물에 여러 개의 테마를 붙일 수 있습니다
- 검색 필터로도 활용됩니다


### 🌟 왜 매물 종류별로 설정하나요?

매물 종류마다 적합한 테마가 다릅니다!

#### 주거용 (아파트, 오피스텔)
```
✅ 신혼부부
```

## 순서 바꾸기

### 드래그로 순서 변경하기


### 사용 방법

#### 일반 드래그 (같은 레벨에서 이동)
1. 왼쪽의 **≡ 아이콘**에 마우스를 올립니다
2. **클릭한 채로 끌어서** 원하는 위치로 옮깁니다
3. 놓으면 순서가 바로 바뀝니다


```
변경 전:
1. 아파트
2. 오피스텔
3. 빌라

변경 후:
1. 오피스텔
2. 아파트
3. 빌라
```

#### Shift + 드래그 (다른 부모로 이동)
1. **Shift 키를 누른 채로**
2. 매물 종류를 드래그합니다
3. 다른 상위 카테고리 위에 놓습니다
4. 해당 상위 카테고리의 하위 항목으로 이동됩니다


```
변경 전:
주거
  - 아파트
상업
  - 상가

Shift + 드래그로 "아파트"를 "상업" 위에 놓기

변경 후:
주거
  (비어있음)
상업
  - 아파트
  - 상가
```

### 💡 주의사항
- **Shift 없이** 드래그: 같은 레벨에서만 순서 변경
- **Shift + 드래그**: 다른 부모로 이동 가능
- 최대 3단계까지만 가능 (대분류 > 중분류 > 소분류)

## 매물 종류 삭제하기

### 삭제하는 방법
1. **"삭제"** 버튼 클릭
2. 확인 창이 나오면 내용을 읽고 **"확인"** 클릭

### ⚠️ 삭제 전에 꼭 확인하세요!

#### 삭제 가능한 경우
- ✅ 아직 매물이 하나도 없는 매물 종류
- ✅ 테스트로 만들어본 매물 종류

#### 삭제 안 되는 경우
- ❌ 이미 매물이 등록된 매물 종류
- 예: "이 카테고리에 5개의 매물이 있습니다"

### 하위 매물 종류가 있을 때는?

하위 매물 종류가 있으면 삭제할 수 없습니다.

```
주거
  - 아파트 ← 이걸 삭제하려면
    - 일반아파트 ← 먼저 이걸 삭제해야 함
```

### 💡 추천 방법

| 상황 | 추천 |
|------|------|
| 잘못 만든 매물 종류 | 삭제 |
| 이번 달만 안 쓰는 매물 종류 | 중지 |
| 매물이 있는 매물 종류 | 중지 (삭제 불가) |

## 자주 묻는 질문

### Q1. AI 이미지 생성이 느려요
**A.** 품질 좋은 이미지를 만들려면 20~30초 정도 걸립니다.
- 생성 중에는 진행바가 표시됩니다
- 기다리는 동안 다른 항목을 입력할 수 있습니다

### Q2. 프리셋을 적용하면 기존 필드는 어떻게 되나요?
**A.** 선택할 수 있습니다.
- **"기존 필드 유지"**: 프리셋 필드가 추가됨
- **"기존 필드 삭제"**: 프리셋 필드로 전부 교체됨

### Q3. 필드를 삭제하면 기존 매물 데이터는?
**A.** 데이터는 남아있지만 화면에 표시되지 않습니다.
- 필드를 다시 추가하면 데이터가 다시 보입니다
- 완전히 지우려면 매물을 직접 수정해야 합니다

### Q4. 최대 몇 단계까지 만들 수 있나요?
**A.** 3단계까지 가능합니다.
- 대분류 > 중분류 > 소분류
- 더 깊은 단계는 만들어지지 않습니다

### Q5. 거래 종류를 나중에 변경할 수 있나요?
**A.** 네, 언제든 가능합니다.
- 수정 페이지에서 체크박스 변경
- 기존 매물은 영향받지 않음

### Q6. AI가 만든 이미지가 마음에 안 들어요
**A.** 다시 생성할 수 있습니다.
- 프롬프트를 수정하고 다시 생성
- 또는 이미지 라이브러리에서 다른 것 선택
- 또는 직접 이미지 업로드

### Q7. 프리셋에 없는 필드를 추가하고 싶어요
**A.** 프리셋 적용 후 개별 추가 가능합니다.
- "필드 추가" 버튼으로 하나씩 추가
- 드래그로 원하는 위치에 배치

### Q8. 다른 매물 종류의 설정을 복사할 수 있나요?
**A.** 현재는 프리셋만 지원합니다.
- 비슷한 매물 종류는 같은 프리셋 사용
- 이후 개별적으로 필드 추가/수정

### Q9. 매물 종류 이미지는 어디에 사용되나요?
**A.** 여러 곳에서 사용됩니다.
- 매물 목록 화면
- 매물 검색 필터
- 카테고리 선택 화면
- 홈페이지 메인 화면

### Q10. Shift + 드래그가 안 돼요
**A.** 다음을 확인하세요.
- Shift 키를 누르고 있는지 확인
- 3단계를 초과하지 않았는지 확인
- 소분류를 대분류로 바로 이동할 수 없음

### Q11. 테마는 어디서 관리하나요?
**A.** 별도의 테마 관리 메뉴에서 관리합니다.
- 관리자 페이지 → 테마 관리
- 여기서는 어떤 테마를 사용할지만 선택

### Q12. 테마를 나중에 추가/삭제할 수 있나요?
**A.** 네, 언제든 가능합니다.
- 수정 페이지에서 체크박스 변경
- 이미 등록된 매물에는 영향 없음

### Q13. 한 매물에 여러 테마를 붙일 수 있나요?
**A.** 네, 가능합니다.
- 매물 등록 시 여러 개 선택 가능
- 예: "역세권" + "학군" + "신혼부부"
