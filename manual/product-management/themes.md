## 테마란 무엇인가요?

**테마**는 매물의 특징이나 용도를 나타내는 **태그**입니다.

### 🏷️ 테마의 역할

테마를 사용하면 매물을 더 효과적으로 분류하고 검색할 수 있습니다.

**예시:**
- **신혼부부**: 신혼부부에게 적합한 아파트
- **역세권**: 지하철역 근처의 매물
- **학군**: 좋은 학군의 매물
- **투자용**: 투자 목적의 매물
- **반려동물 가능**: 반려동물과 함께 살 수 있는 매물
- **코너자리**: 코너에 위치한 상가
- **1층상가**: 1층에 위치한 상가

### 🎯 테마 vs 매물 종류

| 구분 | 매물 종류 | 테마 |
|------|----------|------|
| **의미** | 부동산의 유형 | 매물의 특징/용도 |
| **예시** | 아파트, 오피스텔, 상가 | 신혼부부, 역세권, 투자용 |
| **개수** | 매물당 1개만 선택 | 매물당 여러 개 선택 가능 |
| **용도** | 매물 분류 | 검색 필터, 추천 |

### 💡 테마가 필요한 이유

#### 1. 고객이 원하는 매물을 쉽게 찾을 수 있습니다
```
고객: "신혼부부가 살기 좋고, 역세권이면서 반려동물 키울 수 있는 아파트 찾아주세요"
→ 테마: "신혼부부" + "역세권" + "반려동물 가능" 필터로 검색!
```

#### 2. 홈페이지에서 매물을 효과적으로 노출할 수 있습니다
- "신혼부부 추천 매물" 섹션
- "투자용 매물 모음" 페이지
- "반려동물 가능 매물" 특집

#### 3. 매물의 특징을 한눈에 알 수 있습니다
```
[아파트 매물]
🏷️ 신혼부부 🏷️ 역세권 🏷️ 학군 🏷️ 넓은평수
```

```
- 신혼부부
- 역세권
- 학군
- 투자용
- 반려동물 가능
- 넓은평수
- 코너자리

❌ 나쁜 예시:
- 좋은 아파트 (너무 모호함)
- 추천 (무엇을 추천하는지 불명확)
```

**순서 정하기**
```
순서는 홈페이지와 매물 등록 시 테마가 표시되는 순서입니다.

예시:
1번: 신혼부부 (가장 먼저 표시)
2번: 역세권
3번: 학군
4번: 투자용
...
```

### 3단계: 저장하기

## 테마 이미지 설정

테마에 이미지를 추가하면 홈페이지에서 더 눈에 띄게 표시됩니다.

### 방법 1: 파일 업로드

**[이미지 삽입: 이미지 업로드 영역]**
<!-- 이미지 파일명: /images/themes/image-upload.png -->

1. **"파일 선택"** 버튼 클릭
2. 컴퓨터에서 이미지 선택
3. 자동으로 업로드됩니다

**지원 형식:**
- JPG, PNG, GIF
- 최대 5MB

### 방법 2: 드래그 앤 드롭

1. 이미지 파일을 컴퓨터에서 찾습니다
2. 점선 박스 영역으로 **드래그**합니다
3. 놓으면 자동으로 업로드됩니다

**[이미지 삽입: 드래그 앤 드롭]**
<!-- 이미지 파일명: /images/themes/drag-drop.png -->

### 방법 3: AI 이미지 생성 ⭐

**[이미지 삽입: AI 이미지 생성 버튼]**
<!-- 이미지 파일명: /images/themes/ai-button.png -->

1. **"AI 이미지 생성"** 버튼 클릭
2. 프롬프트 입력 또는 자동 생성
3. **"생성"** 버튼 클릭
4. 20~30초 후 이미지 생성 완료!

**프롬프트 예시:**
```
✅ "신혼부부를 상징하는 따뜻한 이미지, 하트, 행복한 분위기"
✅ "지하철역 아이콘, 역세권을 나타내는 심플한 디자인"
✅ "학교 건물, 학군을 상징하는 깨끗한 이미지"
✅ "돈 아이콘, 투자를 나타내는 전문적인 이미지"
```

### 방법 4: 이미지 라이브러리

**[이미지 삽입: 이미지 라이브러리]**
<!-- 이미지 파일명: /images/themes/library.png -->

1. **"이미지 라이브러리"** 버튼 클릭
2. 이전에 생성한 이미지 목록이 나타납니다
3. 원하는 이미지 클릭하여 선택

### 이미지 미리보기

업로드한 이미지는 바로 미리보기로 표시됩니다.
- 이미지 클릭 시 **확대**하여 볼 수 있습니다
- **"이미지 제거"** 버튼으로 삭제 가능

## 매물 종류에 테마 연결하기

테마를 만든 후, **어떤 매물 종류에서 이 테마를 사용할 수 있는지** 설정해야 합니다.

### 왜 연결이 필요한가요?

매물 종류마다 적합한 테마가 다릅니다!

```
🏢 아파트:
✅ 신혼부부, 역세권, 학군, 반려동물 가능
❌ 코너자리 (상가용 테마)

🏪 상가:
✅ 코너자리, 1층상가, 유동인구 많음
❌ 신혼부부 (주거용 테마)
```

### 연결 방법

**[이미지 삽입: 테마-매물종류 연결]**
<!-- 이미지 파일명: /images/themes/category-link.png -->

1. 테마 추가/수정 폼 하단의 **"매물 종류 연결"** 섹션을 찾습니다
2. 체크박스로 매물 종류를 선택합니다
   - ✅ 체크: 이 매물 종류에서 테마 사용 가능
   - ☐ 미체크: 이 매물 종류에서 테마 사용 불가
3. 여러 개 선택 가능합니다

### 연결 예시

#### 예시 1: "신혼부부" 테마
```
테마 이름: 신혼부부
설명: 신혼부부가 살기 좋은 매물

연결할 매물 종류:
✅ 아파트
✅ 오피스텔
✅ 빌라
☐ 상가
☐ 토지
```

#### 예시 2: "코너자리" 테마
```
테마 이름: 코너자리
설명: 코너에 위치한 상가

연결할 매물 종류:
☐ 아파트
☐ 오피스텔
☐ 빌라
✅ 상가
✅ 사무실
☐ 토지
```

#### 예시 3: "투자용" 테마
```
테마 이름: 투자용
설명: 투자 목적으로 적합한 매물

연결할 매물 종류:
✅ 아파트
✅ 오피스텔
✅ 빌라
✅ 상가
✅ 사무실
✅ 토지 (모든 종류에 해당)
```

### 💡 연결 후 동작

매물 등록 시, 선택한 매물 종류에 따라 **사용 가능한 테마만** 표시됩니다.

**예시:**
```
아파트 매물 등록 시:
→ "신혼부부", "역세권", "학군" 테마 선택 가능
→ "코너자리", "1층상가" 테마는 목록에 없음
```

## 테마 수정하기

### 수정 방법

**[이미지 삽입: 테마 수정 버튼]**
<!-- 이미지 파일명: /images/themes/edit-button.png -->

1. 테마 목록에서 수정할 테마의 **"수정"** 버튼 클릭
2. 정보를 수정합니다
3. **"저장"** 버튼 클릭

### 수정 가능한 항목

- ✅ 테마 이름
- ✅ 설명
- ✅ 순서
- ✅ 활성화 상태
- ✅ 이미지
- ✅ 연결된 매물 종류

### ⚠️ 주의사항

**테마 이름을 변경하면:**
- 이미 등록된 매물에는 영향 없음
- 새로운 이름으로 표시됩니다

**테마를 비활성화하면:**
- 홈페이지에서 숨겨집니다
- 매물 등록 시 선택할 수 없습니다
- 기존 매물의 테마는 유지됩니다

## 테마 삭제하기

### 삭제 방법

**[이미지 삽입: 테마 삭제 버튼]**
<!-- 이미지 파일명: /images/themes/delete-button.png -->

1. 테마 목록에서 삭제할 테마의 **"삭제"** 버튼 클릭
2. 확인 창이 나타나면 내용을 읽습니다
3. **"확인"** 클릭

### ⚠️ 삭제 전 확인사항

#### 삭제 가능한 경우
- ✅ 아직 사용 중인 매물이 없는 테마
- ✅ 테스트로 만들어본 테마

#### 삭제 불가능한 경우
- ❌ 이미 매물에 사용 중인 테마
- 예: "5개 매물이 이 테마를 사용 중입니다"

### 삭제 시 영향

```
테마를 삭제하면:
✅ 테마 목록에서 제거됩니다
✅ 매물 등록 시 선택할 수 없습니다
❌ 이미 등록된 매물의 테마는 제거되지 않습니다 (데이터 보존)
```

### 💡 권장 방법

| 상황 | 권장 조치 |
|------|----------|
| 잘못 만든 테마 | 삭제 |
| 임시로 안 쓰는 테마 | 비활성화 |
| 매물이 있는 테마 | 비활성화 (삭제 불가) |

## 순서 변경하기

테마의 표시 순서를 쉽게 변경할 수 있습니다.

### 방법 1: 드래그 앤 드롭 (추천)

**[이미지 삽입: 테마 드래그 변경]**
<!-- 이미지 파일명: /images/themes/drag-reorder.png -->

1. 테마 왼쪽의 **≡ 아이콘**에 마우스를 올립니다
2. **클릭한 채로 끌어서** 원하는 위치로 이동합니다
3. 놓으면 순서가 자동으로 저장됩니다

```
변경 전:
1. 신혼부부
2. 역세권
3. 학군

드래그로 "학군"을 맨 위로 이동

변경 후:
1. 학군
2. 신혼부부
3. 역세권
```

### 방법 2: 순서 번호 직접 입력

1. 테마 **"수정"** 버튼 클릭
2. **"순서"** 입력 필드의 숫자를 변경
3. **"저장"** 버튼 클릭

```
순서 번호 입력 예시:
- 순서 1: 가장 먼저 표시
- 순서 5: 다섯 번째로 표시
- 순서 10: 열 번째로 표시
```

### 💡 순서가 중요한 이유

테마 순서는 다음 장소에 반영됩니다:
- 📱 홈페이지 테마 필터
- ✏️ 매물 등록 화면
- 🔍 매물 검색 필터

```
추천 순서:
1번: 신혼부부 (인기)
2번: 역세권 (인기)
3번: 학군 (인기)
4번: 투자용
5번: 반려동물 가능
...
```

## 실전 활용 예시

### 예시 1: 주거용 테마 세트 만들기

#### 1) 신혼부부 테마
```
테마 이름: 신혼부부
설명: 신혼부부가 살기 좋은 매물
순서: 1
활성화: ✅

이미지: AI 생성
프롬프트: "신혼부부를 상징하는 하트와 집 아이콘, 따뜻한 분위기"

연결할 매물 종류:
✅ 아파트
✅ 오피스텔
✅ 빌라
```

#### 2) 역세권 테마
```
테마 이름: 역세권
설명: 지하철역 도보 10분 이내
순서: 2
```

## 자주 묻는 질문

### Q1. 테마와 매물 종류의 차이는 무엇인가요?
**A.** 매물 종류는 "아파트, 상가" 같은 부동산 유형이고, 테마는 "신혼부부, 역세권" 같은 매물의 특징입니다.
- 매물 종류: 1개만 선택 가능
- 테마: 여러 개 선택 가능

### Q2. 한 매물에 여러 테마를 붙일 수 있나요?
**A.** 네, 가능합니다.
```
예시: 아파트 매물
테마: 신혼부부 + 역세권 + 학군 + 반려동물 가능
```

### Q3. 테마 순서를 변경하면 어디에 반영되나요?
**A.** 다음 장소에 반영됩니다:
- 홈페이지 테마 필터
- 매물 등록 화면
- 매물 검색 화면

### Q4. AI 이미지 생성이 너무 느려요
**A.** 고품질 이미지를 만드는 데 20~30초 정도 걸립니다.
- 기다리는 동안 다른 항목 입력 가능
- 이미지 라이브러리에서 기존 이미지 재사용 추천

### Q5. 테마를 삭제하면 기존 매물은 어떻게 되나요?
**A.** 이미 등록된 매물의 테마는 유지됩니다.
- 단, 새로운 매물에는 사용할 수 없습니다
- 매물이 있는 테마는 삭제 불가능

### Q6. 테마를 비활성화하면 어떻게 되나요?
**A.** 다음과 같이 동작합니다:
- 홈페이지에서 숨겨집니다
- 매물 등록 시 선택할 수 없습니다
- 기존 매물의 테마는 유지됩니다
- 언제든 다시 활성화 가능

### Q7. 테마 이미지는 필수인가요?
**A.** 선택 사항입니다.
- 이미지가 있으면 홈페이지에서 더 눈에 띕니다
- 이미지가 없으면 테마 이름만 표시됩니다

### Q8. 어떤 테마를 만들어야 할까요?
**A.** 다음을 참고하세요:

**주거용 추천 테마:**
- 신혼부부, 역세권, 학군, 반려동물 가능
- 넓은평수, 투자용, 전망좋음, 신축

**상업용 추천 테마:**
- 코너자리, 1층상가, 유동인구 많음
- 주차가능, 대로변, 권리금없음

**토지용 추천 테마:**
- 개발예정지, 도로접면, 전원주택부지
- 농업용, 상업지역, 전기인입

### Q9. 테마를 매물 종류에 연결하지 않으면?
**A.** 해당 매물 종류에서 테마를 선택할 수 없습니다.
```
예시:
"신혼부부" 테마를 아파트에 연결 안 함
→ 아파트 매물 등록 시 "신혼부부" 선택 불가
```

### Q10. 테마 개수에 제한이 있나요?
**A.** 제한은 없지만, 너무 많으면 관리가 어렵습니다.
- 추천: 주거용 10개, 상업용 10개 정도
- 자주 사용하는 테마 위주로 만드세요

### Q11. 테마 이름을 변경하면 URL도 바뀌나요?
**A.** 네, 자동으로 생성됩니다.
```
테마 이름: 신혼부부
→ URL: /themes/신혼부부
```

### Q12. 드래그로 순서 변경이 안 돼요
**A.** 다음을 확인하세요:
- ≡ 아이콘을 클릭하고 있는지 확인
- 마우스를 너무 빨리 움직이지 않기
- 브라우저 새로고침 후 다시 시도

## 도움말

### 💡 효과적인 테마 관리 팁

#### 1. 테마는 간결하게
```
✅ 좋은 예시: 신혼부부, 역세권, 학군
❌ 나쁜 예시: 신혼부부가 살기 좋은 매물
```

#### 2. 매물 종류별로 테마 분리
```
주거용: 신혼부부, 학군, 반려동물 가능
상업용: 코너자리, 1층상가, 유동인구 많음
토지용: 개발예정지, 도로접면
```

#### 3. 자주 사용하는 테마를 앞쪽에 배치
```
1번: 역세권 (가장 많이 사용)
2번: 투자용
3번: 신혼부부
...
```

#### 4. 계절별/이벤트 테마 활용
```
- "봄맞이 특가"
- "여름 휴가용 펜션"
- "연말정산 투자용"
```

### 🎯 테마 활용 전략

#### 홈페이지에서 활용
- 테마별 매물 모음 페이지 제작
- "신혼부부 추천" 섹션
- "역세권 매물 특집"

#### SEO 최적화
- 테마 이름에 검색 키워드 포함
- 예: "학군좋은", "역세권", "투자용"

#### 마케팅 활용
- 테마별 이벤트 진행
- "신혼부부 테마 매물 할인"
- "투자용 매물 추천 이벤트"
