## 라벨이란 무엇인가요?

**라벨**은 매물에 붙이는 **시각적 태그**입니다. 배지 형태로 표시되어 매물의 상태나 특징을 한눈에 알 수 있습니다.

### 🏷️ 라벨의 역할

라벨은 매물의 **상태나 특별한 정보**를 강조하는 데 사용됩니다.

**예시:**
- 🔥 **급매**: 빠른 거래가 필요한 매물
- ⭐ **추천**: 추천하는 매물
- 🆕 **신규**: 새로 등록된 매물
- 💎 **프리미엄**: 프리미엄 매물
- ✅ **계약완료**: 계약이 완료된 매물
- 🔴 **오늘등록**: 오늘 등록된 매물

### 🎯 라벨 vs 테마

| 구분 | 라벨 | 테마 |
|------|------|------|
| **목적** | 매물의 **상태** 표시 | 매물의 **특징** 표시 |
| **표시 방식** | 배지 (색상 박스) | 태그 |
| **예시** | 급매, 신규, 추천 | 신혼부부, 역세권, 학군 |
| **사용 위치** | 매물 썸네일 위 | 매물 상세 정보 |
| **개수** | 보통 1~2개 | 여러 개 가능 |

### 💡 라벨이 필요한 이유

#### 1. 매물의 상태를 한눈에 알 수 있습니다
```
[매물 카드]
┌─────────────────┐
│ 🔥 급매         │ ← 라벨
│                 │
│  아파트 사진    │
│                 │
│  서울시 강남구  │
│  5억 5천만원    │
└─────────────────┘
```

#### 2. 긴급하거나 특별한 매물을 강조할 수 있습니다
- 🔥 **급매**: 고객의 주목을 끌어 빠른 거래 유도
- ⭐ **추천**: 우수 매물을 부각
- 🆕 **신규**: 새로운 매물임을 알림

#### 3. 매물 관리 상태를 표시합니다
- ✅ **계약완료**: 이미 계약된 매물 표시
- 🔒 **비공개**: 비공개 매물 표시

## 라벨 색상 설정하기

라벨의 색상은 **HEX 코드** 형식으로 입력합니다.

### HEX 코드란?

색상을 표현하는 6자리 코드입니다.

**형식:** `#RRGGBB`
- `#` + 빨강(RR) + 초록(GG) + 파랑(BB)
- 예: `#ef4444` (빨강), `#3b82f6` (파랑)

### 추천 색상 조합

**[이미지 삽입: 색상 조합 예시]**

{/* 이미지 파일명: /images/labels/colors.png */}

#### 1. 긴급/중요 (빨강 계열)
```
🔥 급매
배경색: #ef4444 (빨강)
글자색: #ffffff (흰색)
```

#### 2. 추천/우수 (파랑 계열)
```
⭐ 추천
배경색: #3b82f6 (파랑)
글자색: #ffffff (흰색)
```

#### 3. 신규 (초록 계열)
```
🆕 신규
배경색: #10b981 (초록)
글자색: #ffffff (흰색)
```

#### 4. 프리미엄 (보라 계열)
```
💎 프리미엄
배경색: #8b5cf6 (보라)
글자색: #ffffff (흰색)
```

#### 5. 할인 (주황 계열)
```
💰 할인
배경색: #f97316 (주황)
글자색: #ffffff (흰색)
```

#### 6. 완료/비활성 (회색 계열)
```
✅ 계약완료
배경색: #6b7280 (회색)
글자색: #ffffff (흰색)
```

### 색상 입력 방법

**방법 1: 직접 입력**
```
1. 배경색 입력 필드 클릭
2. HEX 코드 입력 (예: #ef4444)
3. 글자색 입력 필드 클릭
4. HEX 코드 입력 (예: #ffffff)
```

**방법 2: 색상 선택기 사용 (브라우저 지원 시)**
```
1. 입력 필드 옆의 색상 아이콘 클릭
2. 색상 선택기에서 원하는 색상 선택
3. 자동으로 HEX 코드 입력됨
```

### 💡 색상 선택 팁

#### 대비(Contrast)가 중요합니다
```
✅ 좋은 조합:
- 어두운 배경 + 밝은 글자 (#ef4444 + #ffffff)
- 밝은 배경 + 어두운 글자 (#fef3c7 + #000000)

❌ 나쁜 조합:
- 밝은 배경 + 밝은 글자 (#fef3c7 + #ffffff) → 안 보임
- 어두운 배경 + 어두운 글자 (#1f2937 + #000000) → 안 보임
```

#### 용도에 맞는 색상 선택
```
🔴 빨강: 긴급, 급매, 마감임박
🔵 파랑: 추천, 인기, 신뢰
🟢 초록: 신규, 새로운, 완료
🟣 보라: 프리미엄, 고급, 특별
🟠 주황: 할인, 특가, 이벤트
⚫ 회색: 비활성, 완료, 일반
```

## 라벨 수정하기

### 수정 방법

**[이미지 삽입: 라벨 수정 버튼]**

{/* 이미지 파일명: /images/labels/edit-button.png */}

1. 라벨 목록에서 수정할 라벨의 **"수정"** 버튼 클릭
2. 정보를 수정합니다
3. **"저장"** 버튼 클릭

### 수정 가능한 항목

- ✅ 라벨 이름
- ✅ 배경색
- ✅ 글자색
- ✅ 순서
- ✅ 활성화 상태

### ⚠️ 주의사항

**라벨 이름을 변경하면:**
- 이미 등록된 매물에 즉시 반영됩니다
- 모든 매물에서 새 이름으로 표시됩니다

**라벨 색상을 변경하면:**
- 이미 등록된 매물에 즉시 반영됩니다
- 모든 매물에서 새 색상으로 표시됩니다

**라벨을 비활성화하면:**
- 홈페이지에서 숨겨집니다
- 매물 등록 시 선택할 수 없습니다
- 기존 매물의 라벨은 유지되지만 표시되지 않습니다

## 라벨 삭제하기

### 삭제 방법

**[이미지 삽입: 라벨 삭제 버튼]**

{/* 이미지 파일명: /images/labels/delete-button.png */}

1. 라벨 목록에서 삭제할 라벨의 **"삭제"** 버튼 클릭
2. 확인 창이 나타나면 내용을 읽습니다
3. **"확인"** 클릭

### ⚠️ 삭제 전 확인사항

#### 삭제 가능한 경우
- ✅ 아직 사용 중인 매물이 없는 라벨
- ✅ 테스트로 만들어본 라벨

#### 삭제 불가능한 경우
- ❌ 이미 매물에 사용 중인 라벨
- 예: "10개 매물이 이 라벨을 사용 중입니다"

### 삭제 시 영향

```
라벨을 삭제하면:
✅ 라벨 목록에서 제거됩니다
✅ 매물 등록 시 선택할 수 없습니다
⚠️ 이미 등록된 매물의 라벨은 자동으로 제거됩니다
```

### 💡 권장 방법

| 상황 | 권장 조치 |
|------|----------|
| 잘못 만든 라벨 | 삭제 |
| 임시로 안 쓰는 라벨 | 비활성화 |
| 계절 이벤트 라벨 | 비활성화 후 보관 |
| 매물이 많은 라벨 | 수정하여 재사용 |

## 순서 변경하기

라벨의 표시 순서를 설정할 수 있습니다.

### 순서 변경 방법

1. 라벨 **"수정"** 버튼 클릭
2. **"순서"** 입력 필드의 숫자를 변경
3. **"저장"** 버튼 클릭

```
순서 번호 입력 예시:
- 순서 1: 가장 먼저 표시 (가장 중요)
- 순서 2: 두 번째로 표시
- 순서 3: 세 번째로 표시
...
```

### 💡 순서가 중요한 이유

라벨 순서는 다음 장소에 반영됩니다:
- 📱 **홈페이지 매물 카드**: 첫 번째 라벨만 표시
- ✏️ **매물 등록 화면**: 드롭다운에 순서대로 표시
- 🔍 **매물 상세 페이지**: 최대 2개 라벨 표시

```
추천 순서:
1번: 급매 (가장 중요)
2번: 신규
3번: 추천
4번: 프리미엄
5번: 할인
...
```

### 여러 라벨이 있을 때

한 매물에 여러 라벨이 있으면 **순서가 작은 것부터** 표시됩니다.

**예시:**
```
매물에 설정된 라벨:
- 급매 (순서: 1)
- 신규 (순서: 2)
- 추천 (순서: 3)

→ 홈페이지에 표시: [급매] [신규]
(최대 2개까지 표시)
```

```
글자색: #ffffff (흰색)
순서: 1
활성화: ✅

용도: 당일 등록된 매물 강조
```

## 자주 묻는 질문

### Q1. 라벨과 테마의 차이는 무엇인가요?
**A.** 라벨은 매물의 **상태**를 표시하고, 테마는 매물의 **특징**을 표시합니다.
- **라벨**: "급매", "신규", "추천" → 상태/긴급도
- **테마**: "신혼부부", "역세권", "학군" → 특징/용도

### Q2. 한 매물에 여러 라벨을 붙일 수 있나요?
**A.** 네, 가능합니다. 하지만 홈페이지에는 보통 1~2개만 표시됩니다.
```
매물에 설정: 급매 + 신규 + 추천
홈페이지 표시: [급매] [신규] (순서대로 최대 2개)
```

### Q3. 라벨 색상을 어떻게 정해야 하나요?
**A.** 용도에 맞는 색상을 선택하세요.
- 🔴 **빨강**: 긴급, 급매, 마감임박
- 🟢 **초록**: 신규, 새로운
- 🔵 **파랑**: 추천, 인기
- 🟣 **보라**: 프리미엄, 고급
- 🟠 **주황**: 할인, 특가

### Q4. HEX 코드를 모르는데 어떻게 입력하나요?
**A.** 다음 방법을 사용하세요:
1. 위의 추천 색상 조합을 복사해서 사용
2. 브라우저 색상 선택기 사용
3. 검색: "빨강 hex 코드" → #ef4444

### Q5. 라벨을 삭제하면 기존 매물은 어떻게 되나요?
**A.** 이미 등록된 매물의 라벨도 자동으로 제거됩니다.
- 라벨 데이터가 완전히 삭제됩니다
- 삭제 전에 비활성화로 테스트 추천

### Q6. 라벨을 비활성화하면 어떻게 되나요?
**A.** 다음과 같이 동작합니다:
- 홈페이지에서 숨겨집니다
- 매물 등록 시 선택할 수 없습니다
- 기존 매물의 라벨 데이터는 유지됩니다
- 다시 활성화하면 즉시 표시됩니다

### Q7. 라벨 순서를 변경하면 어디에 반영되나요?
**A.** 다음 장소에 반영됩니다:
- 홈페이지 매물 카드 (첫 번째 라벨)
- 매물 상세 페이지 (최대 2개)
- 매물 등록 화면 드롭다운

### Q8. 라벨은 몇 개까지 만들 수 있나요?
**A.** 제한은 없지만, 너무 많으면 관리가 어렵습니다.
- 추천: 5~10개 정도
- 자주 사용하는 라벨 위주로 만드세요

### Q9. 계절별 이벤트 라벨은 어떻게 관리하나요?
**A.** 비활성화 기능을 활용하세요.
```
봄: "봄특가" 활성화
여름: "봄특가" 비활성화 → "여름특가" 활성화
가을: "여름특가" 비활성화 → "가을특가" 활성화
```

### Q10. 라벨이 홈페이지에 안 보여요
**A.** 다음을 확인하세요:
- 라벨이 **활성화** 상태인지 확인
- 매물에 라벨이 **선택**되어 있는지 확인
- 브라우저 **캐시 삭제** 후 새로고침

### Q11. 배경색과 글자색이 너무 비슷해서 안 보여요
**A.** 대비(Contrast)를 높이세요.
```
❌ 나쁜 예: 배경 #fef3c7 + 글자 #ffffff (둘 다 밝음)
✅ 좋은 예: 배경 #fef3c7 + 글자 #000000 (밝음 + 어두움)
```

### Q12. 라벨 이름을 이모지로만 만들 수 있나요?
**A.** 가능하지만 추천하지 않습니다.
```
❌ 나쁜 예: "🔥" (이모지만)
✅ 좋은 예: "🔥 급매" (이모지 + 텍스트)
```
이유: 이모지만 있으면 검색이 어렵고, 의미 전달이 명확하지 않습니다.

## 도움말

### 💡 효과적인 라벨 관리 팁

#### 1. 라벨은 간결하고 명확하게
```
✅ 좋은 예시: 급매, 신규, 추천
❌ 나쁜 예시: 지금 바로 계약 가능한 매물
```

#### 2. 색상은 일관성 있게 사용
```
빨강: 항상 긴급/중요한 것
초록: 항상 새로운 것
파랑: 항상 추천/우수한 것
```

#### 3. 자주 사용하는 라벨을 앞쪽에 배치
```
1번: 급매 (가장 많이 사용)
2번: 신규
3번: 추천
...
```

#### 4. 계절/이벤트 라벨은 비활성화로 관리
```
사용 시: 활성화
종료 후: 비활성화 (삭제 안 함)
재사용: 이름만 변경하여 활성화
```

### 🎯 라벨 활용 전략

#### 긴급도에 따른 라벨 활용
```
상: 급매 (빨강) - 즉시 거래 필요
중: 추천 (파랑) - 일반 추천
하: 일반 (라벨 없음)
```

#### 시기에 따른 라벨 활용
```
등록 시: 신규 (7일간 표시)
추천 시: 추천
이벤트: 특가/할인
완료 시: 계약완료
```

#### SEO 및 마케팅 활용
- "급매" 라벨로 긴급 매물 강조
- "신규" 라벨로 새 매물 부각
- "추천" 라벨로 우수 매물 홍보
- 계절 이벤트 라벨로 프로모션

### 📊 라벨 사용 현황 확인

정기적으로 라벨 사용 현황을 확인하세요:
- 사용하지 않는 라벨 정리
- 자주 사용하는 라벨 순서 조정
- 계절 라벨 활성화/비활성화 관리
