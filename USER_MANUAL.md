## 1. 시작하기

### 1.1 시스템 접속 방법

1. 웹브라우저(크롬, 엣지, 사파리 등)를 실행합니다
2. 주소창에 회사 홈페이지 주소를 입력합니다
3. 주소 뒤에 `/admin`을 붙여 관리자 페이지로 이동합니다
   - 예: `https://yourcompany.com/admin`

### 1.2 권장 환경

- **PC**: Windows 10 이상, Mac OS
- **웹브라우저**: Chrome, Edge, Safari 최신 버전
- **인터넷 속도**: 10Mbps 이상 권장
- **화면 해상도**: 1920x1080 권장

### 1.3 모바일 사용

- 스마트폰이나 태블릿에서도 접속 가능합니다
- 외근 중에도 매물 정보를 확인하거나 간단한 수정이 가능합니다

## 2. 로그인 및 계정 관리

### 2.1 로그인하기

1. 관리자 페이지(`/admin`)에 접속합니다
2. 이메일과 비밀번호를 입력합니다
3. **로그인** 버튼을 클릭합니다

![로그인 화면 예시]
```
┌─────────────────────────────────┐
│   부동산 관리자 로그인           │
│                                  │
│   이메일: [____________]         │
│   비밀번호: [____________]       │
│                                  │
│   [  로그인  ]                   │
│   비밀번호를 잊으셨나요?          │
└─────────────────────────────────┘
```

### 2.2 비밀번호 찾기

1. 로그인 화면에서 **비밀번호를 잊으셨나요?** 클릭
2. 가입한 이메일 주소 입력
3. 메일로 받은 링크를 클릭하여 새 비밀번호 설정

### 2.3 비밀번호 변경하기

1. 우측 상단 프로필 아이콘 클릭
2. **내 정보** 선택
3. **비밀번호 변경** 탭 클릭
4. 현재 비밀번호와 새 비밀번호 입력
5. **변경하기** 버튼 클릭

> 💡 **팁**: 비밀번호는 8자 이상, 영문+숫자 조합을 권장합니다.

## 4. 매물 등록하기

### 4.1 매물 등록 시작하기

1. 좌측 메뉴에서 **매물관리** 클릭
2. 우측 상단 **+ 매물 등록** 버튼 클릭
3. 매물 정보를 단계별로 입력합니다

### 4.2 1단계: 기본 정보 입력

#### 매물 종류 선택
먼저 매물의 종류를 선택합니다.

```
매물 종류 선택
┌─────────────────────────────────┐
│ 주거용 ▼                         │
│   ├─ 아파트                      │
│   ├─ 빌라/연립                   │
│   ├─ 단독/다가구                 │
│   └─ 원룸/오피스텔               │
│                                  │
│ 상업용 ▼                         │
│   ├─ 상가                        │
│   ├─ 사무실                      │
│   └─ 건물                        │
└─────────────────────────────────┘
```

> 💡 **중요**: 매물 종류를 선택하면 그에 맞는 입력 항목이 자동으로 나타납니다.

#### 제목 입력
- 고객이 보게 될 매물 제목을 입력합니다
- 예: "강남역 5분거리 신축 아파트"

#### 주소 입력

1. **주소 검색** 버튼 클릭
2. 도로명 또는 지번 주소 검색
3. 검색 결과에서 선택
4. 상세 주소 입력 (동/호수, 층수 등)

```
주소 입력
┌─────────────────────────────────┐
│ [주소 검색]                      │
│                                  │
│ 선택된 주소:                     │
│ 서울특별시 강남구 테헤란로 123   │
│                                  │
│ 상세주소:                        │
│ [________________]               │
│ (예: 101동 1205호)               │
└─────────────────────────────────┘
```

> 💡 **팁**: 주소를 선택하면 지도 좌표가 자동으로 설정됩니다.

#### 면적 입력

면적은 제곱미터(㎡) 또는 평으로 입력할 수 있습니다.

```
면적
┌─────────────────────────────────┐
│ [_______] ㎡  = [_______] 평     │
│                                  │
│ (자동 변환됨)                    │
└─────────────────────────────────┘
```

- 한쪽에 입력하면 다른 쪽이 자동으로 계산됩니다
- 1평 = 3.3058㎡

### 4.3 2단계: 거래 정보 입력

#### 거래 종류 선택

원하는 거래 종류를 선택하고 가격을 입력합니다.

**매매**
```
☑ 매매
매매가: [____________] 원
```

**전세**
```
☑ 전세
전세가: [____________] 원
```

**월세**
```
☑ 월세
보증금: [____________] 원
월세: [____________] 원
```

> 💡 **여러 거래 가능**: 한 매물에 여러 거래 종류를 동시에 등록할 수 있습니다.
> 예: 전세 3억 또는 월세 1억/150만원

#### 계약 상태 선택

```
계약 상태: [진행중 ▼]
  - 진행중
  - 예약
  - 계약완료
  - 거래중지
```

### 4.4 3단계: 상세 정보 입력

매물 종류에 따라 다른 항목들이 나타납니다.

#### 아파트 예시
```
세대수: [_______] 세대
층수: [___] / [___] 층 (현재층/전체층)
방향: [ ▼ ] (남향, 동남향, 남서향 등)
방/욕실: [___] 개 / [___] 개
엘리베이터: [ ] 있음
주차: [___] 대 가능
```

#### 상가 예시
```
업종: [_________________]
권리금: [____________] 원
월 매출: [____________] 원
순이익: [____________] 원
영업시간: [__:__] ~ [__:__]
```

### 4.5 4단계: 상세 설명 작성

매물의 특징, 장점, 주변 환경 등을 자유롭게 작성합니다.

```
상세 설명
┌────────────────────────────────────┐
│                                     │
│ (예시)                              │
│ • 강남역 도보 5분 거리              │
│ • 풀옵션 완비                       │
│ • 남향으로 채광이 좋습니다          │
│ • 즉시 입주 가능                    │
│                                     │
└────────────────────────────────────┘
```

### 4.6 5단계: 사진 업로드

#### 사진 올리기

1. **사진 업로드** 버튼 클릭 또는 드래그 앤 드롭
2. 여러 장을 한 번에 선택 가능
3. 업로드된 사진들이 나타남

#### 사진 관리

```
업로드된 사진 (5장)
┌────────────────────────────────────┐
│ [사진1]  [사진2]  [사진3]          │
│  ⭐대표   순서변경  순서변경        │
│                                     │
│ [사진4]  [사진5]                   │
│  순서변경  순서변경                 │
└────────────────────────────────────┘
```

**사진 기능**:
- ⭐ **대표 설정**: 목록에서 보일 대표 사진
- **순서 변경**: 드래그하여 순서 조정
- **삭제**: X 버튼으로 삭제

> 💡 **팁**: 
> - 첫 번째 사진이 기본 대표 사진입니다
> - 가로 사진이 보기 좋습니다
> - 밝고 선명한 사진을 사용하세요

### 4.7 6단계: 추가 옵션 (선택사항)

#### 매도인/관리인 연락처

매물과 관련된 연락처를 등록합니다.

```
매도인
┌────────────────────────────────────┐
│ [연락처 검색 또는 추가]             │
│                                     │
│ 선택됨: 홍길동 (010-1234-5678)     │
│         매도인                      │
└────────────────────────────────────┘
```

> 자세한 내용은 [6. 고객 연락처 관리](#6-고객-연락처-관리) 참조

#### 테마 선택 (선택사항)

```
테마: [ ] 급매물
      [ ] 추천매물
      [ ] 신규매물
      [ ] 인기매물
```

#### 라벨 추가 (선택사항)

```
라벨: [ ] 역세권
      [ ] 신축
      [ ] 리모델링
      [ ] 풀옵션
      [추가하기]
```

### 4.8 등록 완료

1. 모든 정보를 확인합니다
2. 하단의 **등록하기** 버튼을 클릭합니다
3. "매물이 등록되었습니다" 메시지 확인

> ⚠️ **주의**: 
> - 필수 항목(*)은 반드시 입력해야 합니다
> - 임시 저장 기능으로 나중에 이어서 작성할 수 있습니다

## 5. 매물 관리하기

### 5.1 매물 목록 보기

1. 좌측 메뉴에서 **매물관리** 클릭
2. 등록된 모든 매물이 표시됩니다

```
매물 관리 (총 150건)
┌────────────────────────────────────────────────────┐
│ [검색] [________] 🔍  [매물종류▼] [거래종류▼]      │
├────────────────────────────────────────────────────┤
│ #123  강남역 5분거리 신축 아파트                   │
│       서울 강남구 | 아파트 | 매매 5억              │
│       조회 156회 | 2025-11-10 등록    [수정] [삭제]│
├────────────────────────────────────────────────────┤
│ #122  역세권 원룸 급매                             │
│       서울 강남구 | 원룸 | 월세 500/50             │
│       조회 89회 | 2025-11-09 등록     [수정] [삭제]│
└────────────────────────────────────────────────────┘
```

### 5.2 매물 검색 및 필터링

#### 키워드 검색
- 제목, 주소로 검색 가능
- 예: "강남", "아파트", "신축"

#### 필터 사용
- **매물 종류**: 아파트, 빌라, 상가 등
- **거래 종류**: 매매, 전세, 월세
- **계약 상태**: 진행중, 계약완료 등
- **등록일**: 최근 1주일, 1개월 등

### 5.3 매물 수정하기

1. 매물 목록에서 수정할 매물 찾기
2. **수정** 버튼 클릭
3. 변경할 내용 수정
4. **저장하기** 버튼 클릭

> 💡 **빠른 수정**: 가격이나 상태만 변경할 때는 목록에서 바로 수정 가능

### 5.4 매물 삭제하기

1. 삭제할 매물의 **삭제** 버튼 클릭
2. 확인 메시지에서 **확인** 클릭

> ⚠️ **주의**: 삭제된 매물은 복구할 수 없습니다. 신중하게 삭제하세요.

### 5.5 매물 상태 변경

계약이 진행되거나 완료되면 상태를 변경합니다.

```
상태 변경
진행중 → 예약 → 계약완료
```

**일괄 상태 변경**:
1. 여러 매물 선택 (체크박스)
2. 상단 **일괄 작업** 선택
3. **상태 변경** 선택
4. 새 상태 선택

### 5.6 엑셀로 내보내기/가져오기

#### 엑셀로 내보내기
1. **내보내기** 버튼 클릭
2. 필터 조건 그대로 엑셀 파일 다운로드
3. 외부에서 매물 정보 활용 가능

#### 엑셀로 가져오기
1. **가져오기** 버튼 클릭
2. 샘플 양식 다운로드
3. 양식에 맞춰 매물 정보 입력
4. 파일 업로드

> 💡 **대량 등록**: 많은 매물을 한 번에 등록할 때 유용합니다.

## 6. 고객 연락처 관리

### 6.1 연락처 관리 개요

매물 등록, 문의, 계약 시 사용하는 고객 연락처를 통합 관리합니다.

### 6.2 연락처 보기

1. 좌측 메뉴에서 **고객관리** > **연락처** 클릭
2. 등록된 모든 연락처 확인

```
연락처 관리 (총 83명)
┌────────────────────────────────────────────────────┐
│ [전체▼] [검색] [________] 🔍          [+ 새 연락처]│
├────────────────────────────────────────────────────┤
│ 홍길동          010-1234-5678                       │
│ 매도인          hong@email.com       [수정] [삭제] │
├────────────────────────────────────────────────────┤
│ 김철수          010-9876-5432                       │
│ 매수인          -                    [수정] [삭제] │
└────────────────────────────────────────────────────┘
```

### 6.3 연락처 타입

| 타입 | 설명 |
|------|------|
| 매도인 | 매물을 내놓은 사람 |
| 매수인 | 매물을 구하는 사람 |
| 관리인 | 건물 관리인, 집주인 대리인 |
| 소유주 | 건물주, 집주인 |
| 세입자 | 현재 거주자 |

### 6.4 새 연락처 추가

1. **+ 새 연락처** 버튼 클릭
2. 정보 입력

```
연락처 추가
┌────────────────────────────────────┐

## 8. 계약 관리

### 8.1 계약 관리 개요

매매, 전세, 월세 등의 계약을 체계적으로 관리합니다.

### 8.2 계약 목록 보기

1. 좌측 메뉴에서 **계약관리** 클릭

```

│ 계약금액: 5억원 | 담당: 박대리                     │
│ 상태: [서명대기]                    [상세보기]    │
└────────────────────────────────────────────────────┘
```

### 8.3 계약 상태 흐름

```
작성중 → 서명대기 → 서명완료 → 보관
           ↓
         해지

| 상태 | 설명 |
|------|------|
| 작성중 | 계약서 작성 진행 중 |
| 서명대기 | 계약서 작성 완료, 서명 필요 |
| 서명완료 | 계약 체결 완료 |
| 해지 | 계약 해지 |
| 보관 | 완료된 계약 보관 |

## 9. 매물 종류 설정

### 9.1 매물 종류란?

매물을 분류하는 카테고리입니다.

```
주거용 (대분류)
  ├─ 아파트 (중분류)
  ├─ 빌라/연립 (중분류)
  └─ 원룸/오피스텔 (중분류)

상업용 (대분류)
  ├─ 상가 (중분류)
  └─ 사무실 (중분류)
```

### 9.2 매물 종류 관리

1. 좌측 메뉴에서 **설정** > **매물 종류** 클릭

```
매물 종류 관리
┌────────────────────────────────────────────────────┐
│                               [+ 새 카테고리 추가]  │
├────────────────────────────────────────────────────┤
│ 주거용                                             │
│   ├─ 아파트 (매물: 45개)          [필드] [수정]   │
│   ├─ 빌라/연립 (매물: 23개)       [필드] [수정]   │
│   └─ 원룸 (매물: 34개)            [필드] [수정]   │
│                                                     │
│ 상업용                                             │
│   ├─ 상가 (매물: 18개)            [필드] [수정]   │
│   └─ 사무실 (매물: 12개)          [필드] [수정]   │
└────────────────────────────────────────────────────┘
```

### 9.3 입력 필드 설정

각 매물 종류마다 다른 입력 항목을 설정할 수 있습니다.

#### 필드 관리 화면

1. **필드** 버튼 클릭
2. 필드 목록 확인

```
아파트 입력 필드 설정
┌────────────────────────────────────────────────────┐
│ [프리셋 적용▼]                      [+ 필드 추가] │
├────────────────────────────────────────────────────┤
│ ↑↓ 세대수 (NUMBER) 필수              [수정] [삭제]│
│ ↑↓ 층수 (NUMBER) 필수                [수정] [삭제]│
│ ↑↓ 방향 (SELECT)                    [수정] [삭제]│
│ ↑↓ 방/욕실 (TEXT)                   [수정] [삭제]│
│ ↑↓ 엘리베이터 (BOOLEAN)             [수정] [삭제]│
└────────────────────────────────────────────────────┘
```

#### 프리셋 활용

자주 사용하는 필드 세트를 빠르게 적용:

```
프리셋 선택
┌────────────────────────────────────┐
│ [ ] 아파트 기본 필드               │
│ [ ] 빌라/연립 기본 필드            │
│ [ ] 원룸 기본 필드                 │
│ [ ] 상가 기본 필드                 │
│ [ ] 토지 기본 필드                 │
│ [ ] 건물 기본 필드                 │
│                                     │
│ 기존 필드:                         │
│ ○ 유지하고 추가                   │
│ ○ 모두 삭제하고 교체              │
│                                     │
│       [취소]      [적용]           │
└────────────────────────────────────┘
```

> 💡 **초보자 팁**: 프리셋을 먼저 적용하고 필요한 것만 추가/삭제하세요.

### 9.4 필드 타입 설명

| 타입 | 설명 | 예시 |
|------|------|------|
| TEXT | 짧은 텍스트 | 동/호수 |
| NUMBER | 숫자 | 세대수, 층수 |
| SELECT | 선택 목록 | 방향(남향, 북향) |
| CHECKBOX | 체크박스 | 옵션(냉장고, 세탁기) |
| DATE | 날짜 | 입주가능일 |
| BOOLEAN | 예/아니오 | 엘리베이터 유무 |
| AREA | 면적 | 전용면적 (㎡↔평) |

## 12. 자주 묻는 질문

### Q1. 비밀번호를 잊어버렸어요

**A**: 로그인 화면에서 "비밀번호를 잊으셨나요?" 링크를 클릭하고 가입한 이메일을 입력하세요. 비밀번호 재설정 링크가 메일로 발송됩니다.

### Q2. 매물 사진이 업로드되지 않아요

**A**: 다음을 확인하세요:
- 이미지 파일 형식 (JPG, PNG, GIF만 가능)
- 파일 크기 (10MB 이하)
- 인터넷 연결 상태

### Q3. 매물을 삭제했는데 복구할 수 있나요?

**A**: 삭제된 매물은 복구할 수 없습니다. 삭제 전에 신중하게 확인하세요. 임시로 숨기고 싶다면 "비활성화" 기능을 사용하세요.

### Q4. 여러 명이 동시에 같은 매물을 수정하면 어떻게 되나요?

**A**: 나중에 저장한 사람의 내용으로 덮어씌워집니다. 동료와 소통하여 동시 수정을 피하세요.

### Q5. 매물을 대량으로 등록하고 싶어요

**A**: 엑셀 가져오기 기능을 사용하세요:
1. 매물관리에서 "가져오기" 클릭
2. 샘플 양식 다운로드
3. 양식에 맞춰 정보 입력
4. 파일 업로드

### Q6. 모바일에서도 사용할 수 있나요?

**A**: 네, 스마트폰과 태블릿에서도 접속 가능합니다. 단, 일부 기능은 PC에서 사용하는 것이 더 편리합니다.

### Q7. 고객에게 매물 정보를 어떻게 공유하나요?

**A**: 매물 상세 페이지에서 "공유하기" 버튼을 클릭하면 링크를 복사하거나 카카오톡으로 바로 공유할 수 있습니다.

### Q8. 계약서는 어떻게 인쇄하나요?

**A**: 계약 상세 화면에서 "인쇄" 버튼을 클릭하면 인쇄용 양식이 나타납니다.

### Q9. 데이터 백업은 어떻게 하나요?

**A**: 관리자가 정기적으로 백업합니다. 개별 데이터는 엑셀 내보내기로 저장할 수 있습니다.

### Q10. 시스템 사용 중 오류가 발생하면 어떻게 하나요?

**A**: 
1. 페이지 새로고침 (F5)
2. 브라우저 캐시 삭제
3. 그래도 해결 안 되면 관리자에게 문의
